サービス利用契約書
{{ c.company_name }}（以下「甲」という）とシャチョツー（以下「乙」という）は、乙が提供するAI業務アシスタントサービス「シャチョツー」（以下「本サービス」という）の利用に関し、以下のとおり契約（以下「本契約」という）を締結する。
第1条（目的）
本契約は、乙が甲に対して本サービスを提供し、甲がこれを利用するにあたっての条件を定めることを目的とする。
第2条（サービス内容）
  1. 乙は甲に対し、以下のサービスを提供する。
    • プラン名: {{ c.plan_name }}
    • 月額利用料: ¥{{ "{:,}".format(c.monthly_amount) }}（税抜）
    • 利用開始日: {{ c.start_date.strftime('%Y年%m月%d日') }}
  2. サービスの詳細な仕様は、別途乙が定めるサービス仕様書に従うものとする。
  3. 乙は、事前に甲に通知の上、サービス内容の変更・追加を行うことができる。
第3条（利用料金および支払方法）
  1. 甲は乙に対し、本サービスの対価として、第2条に定める月額利用料を支払うものとする。
  2. 支払方法は、クレジットカード決済または請求書払い（月末締め翌月末払い）とする。
  3. 従量課金が発生する場合は、利用量に応じて月末締めで計算し、翌月の基本料金と合算して請求する。
  4. 消費税は別途加算する。
  5. 支払いを遅延した場合、甲は年14.6%の割合による遅延損害金を支払うものとする。
第4条（契約期間）
  1. 本契約の初期契約期間は{{ c.initial_term | default('定めない（月単位での解約が可能）') }}。
  2. 甲は、月末の14日前までに乙に書面または電子的方法により通知することで、翌月末をもって本契約を解約することができる。
  3. 解約時、甲のデータは解約日から30日間保持した後、完全に削除する。甲が希望する場合は、削除前にデータのエクスポートを行うことができる。
第5条（データの取り扱い）
  1. 甲が本サービスに入力・蓄積したデータ（以下「利用者データ」という）の所有権は、甲に帰属する。
  2. 乙は、利用者データをAIモデルの学習目的に使用しない。
  3. 乙は、利用者データの暗号化および定期的なバックアップを実施する。
  4. 乙は、個人情報の保護に関する法律その他関連法令を遵守し、利用者データを適切に管理する。
第6条（セキュリティ）
乙は、本サービスの提供にあたり、以下のセキュリティ対策を講じる。
  1. 通信の暗号化（TLS 1.3以上）
  2. 保存データの暗号化（AES-256）
  3. テナント間のデータ分離（行レベルセキュリティ）
  4. 定期的なセキュリティ監査の実施
  5. 不正アクセスの検知および防止措置
第7条（SLA：サービスレベル）
  1. 乙は、本サービスの月間稼働率99.5%以上を保証する。
  2. 月間稼働率が99.5%を下回った場合、乙は翌月の基本料金の10%を甲に返金するものとする。
  3. 計画メンテナンスは48時間前までに甲に通知するものとし、計画メンテナンスの時間は稼働率の計算から除外する。
第8条（知的財産権）
  1. 本サービスに係るソフトウェア、アルゴリズム、UI、ドキュメント等の知的財産権は乙に帰属する。
  2. 利用者データに係る知的財産権は甲に帰属する。
  3. 本サービスの利用により生成されたレポート・分析結果の知的財産権は甲に帰属する。
第9条（禁止事項）
甲は、以下の行為を行ってはならない。
  1. 本サービスのリバースエンジニアリング、逆コンパイル、逆アセンブル
  2. 本サービスの第三者への再販売、サブライセンス
  3. 本サービスへの不正アクセスまたはその試み
  4. 法令に違反する目的での本サービスの利用
  5. 乙または第三者の権利を侵害する行為
第10条（免責事項）
  1. 天災、戦争、テロ、暴動、法令の改廃、通信回線の障害その他の不可抗力により生じた損害について、乙は責任を負わない。
  2. 本サービスにおけるAIの回答は参考情報であり、その正確性、完全性を保証するものではない。甲はAIの回答に基づく意思決定について自己の責任で行うものとする。
  3. 乙が甲に対して負う損害賠償責任の総額は、損害発生日から遡り直近12ヶ月間に甲が乙に支払った利用料の合計額を上限とする。
第11条（反社会的勢力の排除）
  1. 甲および乙は、自らまたはその役員、実質的に経営を支配する者が、暴力団、暴力団関係企業、総会屋、社会運動等標ぼうゴロその他の反社会的勢力に該当しないことを表明し保証する。
  2. 甲または乙が前項に違反した場合、相手方は何らの催告なしに直ちに本契約を解除することができる。
  3. 前項による解除は、解除した当事者の損害賠償請求を妨げない。
第12条（準拠法および管轄裁判所）
  1. 本契約は日本法に準拠し、日本法に従って解釈されるものとする。
  2. 本契約に関する一切の紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とする。
