**目的**
Python スクリプトを実行し、標準出力に "Hello World" という文字列を表示する。これは、開発環境の基本的な動作確認を目的とする。

**条件・手段**
- プログラミング言語として Python を使用する。
- `【仮説】` Python のバージョンは 3.x 系を前提とする。
- `【仮説】` 作成するファイル名は `main.py` とする。
- `【仮説】` 出力先は標準出力（コンソール）とする。

**概要**
Python の標準関数 `print()` を用いて、"Hello World" という文字列をコンソールに出力する、単一の Python スクリプトを作成する。

**機能要件**
- スクリプトを実行した際、標準出力に "Hello World" という文字列が1行表示されること。

**今後の拡張**
- コマンドライン引数を受け取り、表示する文字列を動的に変更する機能。
- 出力内容を標準出力ではなく、指定したファイルに書き出す機能。

**受入条件・テスト観点**
- `python main.py` を実行すると → 標準出力に `Hello World` という文字列が正確に表示されること。
- `python main.py` を実行すると → `Hello World` 以外の余計な文字列（デバッグ情報など）が一切出力されないこと。
