# 目的
Python の実行環境が正しくセットアップされていることを確認するため、標準出力に "Hello World" という文字列を表示する基本的なスクリプトを作成する。

# 条件・手段
- 成果物は単一の Python スクリプトファイルとする。
- スクリプトは Python 3.x 環境で実行可能であること。
- 外部ライブラリへの依存はないこと。
- スクリプトを実行すると、標準出力に "Hello World" と表示されること。

# 概要
`main.py` という名前の Python スクリプトを作成する。このスクリプトは、実行されるとコンソールに "Hello World" という文字列を出力する。

# 機能要件
- スクリプトを実行した際、標準出力に文字列 "Hello World" を表示する機能。

# 非機能要件
- Python の標準的なコーディング規約 (PEP 8) に準拠すること。
- スクリプトはエラーなく正常に終了すること。

# 受入条件・テスト観点
- **主要機能: 文字列の標準出力**
    - [ ] `python main.py` コマンドでスクリプトを実行した際、ターミナル（標準出力）に `Hello World` という文字列が改行付きで表示されること。(手動テスト)
    - [ ] スクリプト実行後、終了コードが 0 (正常終了) であること。(手動テスト)
