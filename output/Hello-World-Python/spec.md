### 目的
基本的な Python 実行環境の動作確認を目的として、標準出力に "Hello World" を表示する Python スクリプトを作成する。

### 条件・手段
- **言語**: Python 3.x
- **ファイル名**: `main.py`
- **実行環境**: 標準的なコマンドラインインターフェース（ターミナル、コマンドプロンプトなど）
- `【仮説】` 出力する文字列は `Hello World` とし、末尾に改行を含む標準的なコンソール出力とする。
- `【仮説】` スクリプトは単一のファイルで完結し、外部ライブラリへの依存はないものとする。

### 概要
本タスクでは、最も基本的なプログラムである "Hello World" をコンソールに出力する Python スクリプトを作成します。
このスクリプトは、開発環境のセットアップが正しく完了しているかを確認するための最初のステップとして利用できます。今回のスコープは、指定された文字列を標準出力に表示するスクリプトの作成に絞ります。

### 機能要件
- `main.py` という名前の Python スクリプトファイルを作成する。
- スクリプトを実行すると、標準出力に `Hello World` という文字列が1行表示されること。

### 今後の拡張
- （スコープ外）コマンドライン引数で表示する文字列を動的に変更する機能。今回は最小要件に絞るため対象外とする。
- （スコープ外）ロギングライブラリを使用して出力する機能。

### 受入条件・テスト観点
- **正常系**:
    - 開発者がターミナルで `python main.py` コマンドを実行すると
      → 標準出力に `Hello World` という文字列が1行だけ表示され、その後改行されること。
- **異常系**:
    - 開発者が `python main.py` コマンドを実行した際に
      → `hello world` (小文字) や `Hello, World!` (カンマ付き) など、仕様と異なる文字列が出力されないこと。
