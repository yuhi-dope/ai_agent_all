# Ship — リリース前一括チェック

以下を順番に実行し、全てクリアしたらデプロイ可能と判定する。

## Step 1: テスト全件実行

```bash
cd shachotwo-app && python -m pytest tests/ -q --tb=short 2>&1 | tail -20
```

- 全テスト pass であること（skip 許容、fail NG）
- 失敗がある場合はここで止めて修正

## Step 2: セキュリティチェック

`/sec-check` を実行して全7件クリアを確認する。

## Step 3: デプロイ前チェック

`/pre-deploy` を実行して全7件クリアを確認する。

## Step 4: 設計書整合チェック

`/sync-design` を実行して、コードと設計書の整合を確認する。

## 結果出力

```
## Ship チェック結果 ({日付})

| Step | チェック | 状態 |
|---|---|---|
| 1 | テスト全件パス | ✅ / ❌ |
| 2 | セキュリティ7件 | ✅ / ❌ |
| 3 | デプロイ前7件 | ✅ / ❌ |
| 4 | 設計書整合 | ✅ / ❌ |

総合判定: 🚀 Ship可能 / ❌ {N}件要対応
```

全てクリアした場合のみ「🚀 Ship可能」と判定する。
