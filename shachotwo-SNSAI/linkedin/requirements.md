> **最終更新**: 2026-03-23

> **目的**: BtoBリード獲得 + 経営者ネットワーク構築 + 採用ブランディング

## 1. プロジェクト概要

LinkedInを活用し、BtoB文脈での信頼構築とリード獲得を行う。
X（認知拡大）・note（SEO・深掘り）と棲み分け、**意思決定者への直接リーチ**を狙う。

**3媒体の役割分担**:
| 媒体 | 役割 | ターゲットの状態 |
|---|---|---|
| X | 認知・リーチ拡大 | 「面白い人がいるな」 |
| note | 信頼構築・SEO | 「この人ちゃんと実績あるな」 |
| **LinkedIn** | **商談化・ネットワーク** | **「この人と仕事したい」** |

## 2. ターゲット定義

### 2.1 LinkedInならではのペルソナ

| 属性 | 詳細 |
|---|---|
| **役職** | CEO / COO / CTO / DX推進責任者 / 経営企画部長 |
| **企業規模** | 従業員30〜1,000名（X より上の中堅企業層もカバー） |
| **業種（優先）** | 建設・製造・医療/介護・士業・不動産 |
| **LinkedInでの行動** | 業界の投稿にいいね・コメント / 経営者コミュニティに参加 / 採用活動 |
| **心理状態** | 「AI/DX導入の相談先を探している」「信頼できるパートナーが欲しい」 |

### 2.2 Xとの違い

| | X | LinkedIn |
|---|---|---|
| 匿名性 | 高い | 低い（実名・社名） |
| 投稿トーン | カジュアル・勢い重視 | プロフェッショナル・丁寧 |
| リード品質 | 玉石混交 | 高い（役職・企業が明確） |
| バイラル性 | 高い | 中程度（コメントで伸びる） |

## 3. コンテンツ戦略

### 3.1 投稿カテゴリ（4本柱）

| # | カテゴリ | 比率 | 目的 |
|---|---|---|---|
| L1 | **事例・実績レポート** | 35% | 信頼構築（「この会社に任せたい」） |
| L2 | **業界インサイト・考察** | 30% | 専門性アピール（「この人詳しいな」） |
| L3 | **ノウハウ共有（教育型）** | 20% | 価値提供（「勉強になる」→フォロー） |
| L4 | **パーソナルストーリー** | 15% | 共感・人間味（「この人と話してみたい」） |

### 3.2 投稿フォーマット

**LinkedIn特有のベストプラクティス**:
- 冒頭3行で惹きつける（「もっと見る」をクリックさせる）
- 改行を多用（1文=1行。密なテキストは読まれない）
- 絵文字は控えめに（Xほどカジュアルにしない）
- CTA は「コメントで教えてください」型が最強（コメント=アルゴリズム優遇）
- ハッシュタグは3〜5個
- 画像/ドキュメント添付で滞在時間UP

### 3.3 テンプレート

#### L1: 事例レポート型

```
{業種}の{課題}をAIで解決した話。

{Before}:
・{課題の具体的な描写}
・{影響: 時間・コスト・品質}

{After}:
・{ソリューション概要}
・{定量的な成果: X時間→Y時間、コストZ%削減}

学び:
{再現性のあるポイント1〜2個}

同じ課題を抱えている方、コメント欄でお聞かせください。

#AI業務改善 #中小企業DX #{業種}DX
```

#### L2: 業界インサイト型

```
{業界}で今起きている変化を3つ共有します。

1. {変化1}
→ {なぜ重要か}

2. {変化2}
→ {経営者が取るべきアクション}

3. {変化3}
→ {具体的な事例}

あなたの業界ではどうですか？コメントで教えてください。

#DX推進 #{業界}DX #経営戦略
```

#### L4: パーソナルストーリー型

```
{告白/学び/失敗}の話をします。

{エピソード: 3〜5行}

ここから学んだこと:
{学び1}
{学び2}

経営者として/エンジニアとして、{メッセージ}。

共感したらぜひコメントください。

#キャリア #経営者 #学び
```

### 3.4 投稿スケジュール

| 頻度 | 内容 |
|---|---|
| **週3〜4投稿** | L1×1, L2×1, L3 or L4×1〜2 |
| **コメント活動** | 毎日15分（ターゲット層の投稿にコメント） |
| **つながり申請** | 週10〜20件（ターゲットペルソナに合致する人） |

## 5. リード獲得戦略

### 5.1 LinkedInならではのリード導線

```
投稿を見る（フィード）
  ↓ いいね / コメント
プロフィール閲覧
  ↓ 「この人に相談したい」
つながり申請 or メッセージ
  ↓
初回ヒアリング（Zoom 30分）
  ↓
提案
```

### 5.2 コメント戦略（最重要）

LinkedInのアルゴリズムは**コメント数**を最も重視する。

| アクション | 頻度 | 対象 |
|---|---|---|
| 自分の投稿へのコメント返信 | 全件・即日 | — |
| ターゲット層の投稿へのコメント | 毎日5〜10件 | 経営者・DX責任者の投稿 |
| インフルエンサーの投稿への早期コメント | 週3〜5件 | 投稿後1時間以内が理想 |

**コメントの品質基準**:
- 「いいですね」NG → 具体的な経験・数字・視点を追加
- 3行以上が目安
- 質問で締めると返信がもらえる

### 5.3 つながり申請テンプレート

```
{相手の名前}さん

{投稿/プロフィール}を拝見し、{具体的に何に興味を持ったか}。

私はAIを活用した中小企業の業務改善を専門にしており、
特に{相手の業界}での{関連する実績1つ}の経験があります。

情報交換させていただければ幸いです。
よろしくお願いいたします。
```

## 6. Xとの連携

### 6.1 コンテンツ変換パターン

| X（元ネタ） | → LinkedIn（変換後） |
|---|---|
| バズった短文投稿 | → L1/L2 として深掘り（丁寧語・プロ感） |
| Mermaid図解 | → ドキュメント投稿（PDF化して添付） |
| 引用RT + コメント | → L2 業界インサイト（元記事を引用しつつ考察） |
| 実績数字ツイート | → L1 事例レポート（背景・プロセスを詳述） |

### 6.2 トーン変換ルール

| X | LinkedIn |
|---|---|
| 「〜した話。」 | 「〜について共有します。」 |
| 「マジで」「ヤバい」 | 使わない |
| 「DMで相談乗ります」 | 「メッセージでご相談ください」 |
| 絵文字多用OK | 控えめ（段落の先頭に1個程度） |
| 「#AI業務改善」 | 「#AI業務改善 #DX推進 #経営戦略」 |

### 7.1 投稿生成フロー


## 10. note記事への誘導戦略

LinkedInはアフィリエイトリンク直貼りに適さない（BtoBプロフェッショナル文脈で広告感はNG）。
代わりに**LinkedIn → note記事 → アフィリエイト**の導線を作る。

### 10.1 誘導パターン

```
LinkedIn投稿（概要+学び）
  ↓ 「詳細はnoteにまとめました」
note記事（詳細手順+ツール紹介+アフィリエイトリンク）
  ↓ 読者がツール登録
アフィリエイト収益
```

### 10.2 投稿内でのnote誘導テンプレート

```
（LinkedIn投稿の末尾）

▼ 詳細な手順・使ったツール・設定方法はnoteにまとめています:
{note記事URL}

#AI業務改善 #中小企業DX
```

**注意**: LinkedInは外部リンクを含む投稿のリーチを下げるアルゴリズムがある。
対策: **リンクは投稿本文ではなくコメント欄の1コメ目に貼る**（ベストプラクティス）。

### 11.1 既存つながり1,000人の棚卸し手順

既存つながりに「繋がっただけ」が多いため、Phase 0で棚卸しを実施:

```
Step 1: LinkedInのつながり一覧をエクスポート（CSV）
Step 2: 以下の軸でフィルタリング
  - 役職: CEO/COO/CTO/部長/経営企画 → ★ターゲット
  - 業種: 建設/製造/医療/介護/士業/不動産 → ★ターゲット
  - それ以外 → 放置（解除はしない）
Step 3: ターゲット層に対して投稿で再活性化
  - L1/L2の投稿を開始し、反応を見る
  - 反応があった人に個別メッセージ
```

> **既存1,000人は資産**。新規獲得より、既存つながりの再活性化の方がROIが高い。
