# SNS AI コマンドリスト

> Claude Code で使えるスキル・コマンド一覧。コピペで実行可能。

---

## 頭のDB管理

```
/head-db <自由記述>
```

| やりたいこと | コマンド例 |
|---|---|
| エントリ追加 | `/head-db 介護施設で記録業務に1日60分かかっている現場を見た。音声入力+AI整形で半分にできると確信した` |
| 複数追加 | `/head-db 今日の商談で学んだこと: 1.歯科は衛生士不足が最大課題 2.院長が全部やるので夜9時まで残る 3.レセプト返戻率が5%超えると赤字` |
| 検索 | `/head-db 建設業に関するエントリを探して` |
| 更新 | `/head-db BRAIN-004の数字を年間4,000万に修正して` |
| 削除 | `/head-db BRAIN-040を削除して` |

## 開発系（shachotwo-app）

| コマンド | 用途 |
|---|---|
| `/implement <パス>` | モジュール実装（サブエージェント自動振り分け） |
| `/new-feature` | 新機能の実装フロー（設計→実装→テスト→レビュー） |
| `/hotfix` | 緊急バグ修正 |
| `/test-module <モジュール>` | テスト実行・修正 |
| `/ship` | リリース前一括チェック |
