## 1. 見積・請求AI（★実装対象）

### 問題の本質

```
- 車両の故障内容→必要部品+作業工賃の見積を作成
- 保険修理の場合、損保会社向けの見積フォーマット（自研センター基準）が必要
- 部品番号の検索、工数の算出が属人化
- 1件30分〜2時間。月50-100件の見積
```

### 支払いロジック

```
■ 見積作成時間50%削減: 月25-50時間削減 × 時給2,000円 = 月5-10万円
■ 見積精度向上→利益率改善: 年50-100万円
■ → BPO月5万円で ROI 2.0〜4.0倍
```

### 製造業見積AIとの差分（ゲノム差分）

| 項目 | 製造業見積AI | 自動車整備見積AI |
|---|---|---|
| 入力 | 図面PDF+仕様 | 車種+年式+故障箇所+写真 |
| 部品DB | 材料費+外注費 | 自動車部品DB（互換品含む） |
| 工数算出 | 加工工程×チャージレート | 整備作業×標準工数（自研センター） |
| 出力 | 見積書 | 見積書+損保向け請求書 |
| 共通パターン | 入力→構造化→工数推定→単価適用→書類生成 |

### 課題テーブル

| 課題 | 影響額（年間） |
|---|---|
| 見積作成に時間がかかる | 人件費120-240万円 |
| 部品番号の検索が属人化 | ベテラン退職リスク |
| 損保向けフォーマット対応 | 形式不備で支払い遅延 |
| 整備士不足（求人倍率4.7倍） | 採用コスト年100-200万円 |

## 2〜7. 拡張ロードマップ

- **② 車検・点検予約**: 車検期限のリコール管理。歯科「リコール」横展開
- **③ 部品在庫・発注**: 部品番号の互換性管理。製造業「在庫最適化」横展開
- **④ 整備記録**: 法定記録の電子化。建設業「工事写真整理」横展開
- **⑤ 保険会社対応**: 事故車の損害査定+保険請求。歯科「レセプト」横展開
- **⑥ 認証・許可管理**: 認証工場・指定工場の維持要件。建設業「許可更新」横展開
- **⑦ 採用支援**: 整備士不足。歯科「採用支援」横展開
