> **ゲノム系統**: 歯科系（オーナー=現場+シフト+原価の同構造）

## 0. エグゼクティブサマリー

### なぜ飲食業BPOか

```
事業所数: 約67万（全業界最大）
市場規模: 約26兆円/年
廃業率: 開業3年以内に70%が閉店
利益率: 平均2-5%。原価管理が命

飲食店オーナーは：
  ・食材原価率が何%か、日次で把握できていない
  ・FL比率（Food+Labor）60%超で赤字なのに気づくのが月末

  ・口コミ対応する暇がない（Google/食べログ/ホットペッパー）
  → 歯科の「院長の時間=売上」と同構造
```

## 1. 原価管理（FLコスト）AI（★実装対象）

### 1.1 問題の本質

```
飲食店の収益構造:
  売上 100%
  − 食材原価（F: Food Cost）: 25-35%（目標30%以下）
  − 人件費（L: Labor Cost）: 25-35%（目標30%以下）
  = FL比率: 55-65%（60%以下が健全経営の目安）
  − 家賃: 8-12%
  − 水道光熱費: 5-8%
  − その他経費: 5-10%
  = 営業利益: 2-10%

  → FL比率が1%上がると利益が10-20%吹き飛ぶ
  → なのに日次でFL比率を把握できている店はほぼない
  → 月末に税理士から試算表が来て「先月赤字でした」と知る
```

### 1.2 支払いロジック

```
■ 直接効果①：食材原価率の改善
  月商300万円 × 食材原価率1%改善 = 月3万円 = 年36万円

■ 直接効果②：人件費率の最適化
  月商300万円 × 人件費率1%改善 = 月3万円 = 年36万円

■ 直接効果③：オーナーの経営判断の速度向上
  赤字月の回避: 年1-2回 × 月損失30万円 = 年30-60万円

■ 合計効果：年102-132万円
  → BPO月3万円（年36万円）で ROI 2.8〜3.7倍
```

### 1.3 業態別のFL目安（ゲノムに詰める内容）

| 業態 | 食材原価率 | 人件費率 | FL比率 | 特徴 |
|---|---|---|---|---|
| 居酒屋 | 28-33% | 28-33% | 58-65% | ドリンク比率で原価率が大きく変動 |
| レストラン | 30-35% | 25-30% | 58-63% | 食材の質が売り。原価率高め |
| ラーメン | 25-30% | 25-30% | 52-58% | 原価率低め。回転率が命 |
| カフェ | 20-28% | 30-35% | 55-60% | ドリンク原価低い。人件費比率高い |
| 焼肉 | 35-42% | 20-25% | 58-65% | 肉の原価が高い。人件費は低め |
| テイクアウト専門 | 30-35% | 15-20% | 48-53% | 人件費が低い。包材コスト注意 |

### 1.4 業務フロー

```
【日次】
  仕入伝票の記録（紙/アプリ/レジ連動）
  → POSレジから売上データ取得
  → 日次FL比率の速報表示
  → 異常値アラート（例: 食材原価率が前週比+3%）

【週次】
  棚卸し（実地 or 簡易）
  → 理論原価vs実際原価の差異分析
  → 廃棄ロスの集計

【月次】
  月次PL（損益計算書）の自動生成
  → メニュー別原価率の算出（ABC分析）
  → 人時売上高の算出
  → オーナーへの改善提案（AI生成）
```

### 1.5 課題テーブル

| 課題 | 具体的な問題 | 影響額（年間） | 発生頻度 |
|---|---|---|---|
| 日次の原価把握不能 | 月末まで赤字に気づかない | 赤字月の損失30-100万円 | 月1-2回 |
| 廃棄ロスの放置 | 発注過多・仕込み過多 | 売上の3-5%（年90-150万円） | 毎日 |
| 食材値上がりへの対応遅れ | 仕入価格上昇を売価に反映できない | 原価率1-3%悪化 | 随時 |
| メニュー分析なし | 低利益メニューの放置 | 利益率の低迷 | 常時 |
| 棚卸しの未実施 | 理論と実際の乖離が見えない | 横領・ロスの放置 | 常時 |

## 2〜7. 拡張ロードマップ

- **② シフト管理**: アルバイト中心。急欠→オーナーがワンオペ。歯科「シフト管理」横展開
- **③ 仕入・発注最適化**: 廃棄vs欠品。天候・曜日で変動。製造業「在庫最適化」横展開
- **④ HACCP記録管理**: 2021年義務化。温度記録・清掃記録。建設業「安全書類」横展開
- **⑤ 口コミ・集客**: Google/食べログ/ホットペッパー。歯科「集患マーケ」横展開
- **⑥ 経理・給与**: 深夜割増・交通費・まかない控除。歯科「経理・給与」横展開
- **⑦ メニュー分析**: ABC分析。製造業「品質管理」横展開
