## 1. 予約管理・リコールAI（★実装対象）

### 問題の本質

```
- ホットペッパービューティーに月3-15万円払っているが「クーポン客」が来るだけでリピート率30-40%
- リピート率を60%に上げれば広告費を半減できる
- しかし「前回来店から○日後にリマインド」すら手作業
- 歯科のリコール管理と完全に同構造
```

### 支払いロジック

```
■ リピート率10%改善 × 月間新規客50人 × 客単価6,000円 × 年来店4回 = 年120万円の増収
■ ホットペッパー掲載費の削減: 年30-60万円
■ → BPO月2万円（年24万円）で ROI 6倍以上
```

### 歯科リコールとの差分（ゲノム差分）

| 項目 | 歯科リコール | 美容リコール |
|---|---|---|
| リコール間隔 | 3-6ヶ月 | 1-3ヶ月（施術による） |
| 連絡手段 | LINE/SMS/はがき | LINE/Instagram DM/SMS |
| パーソナライズ | 治療内容ベース | 施術履歴+好み+季節提案 |
| 予約経路 | 電話/Web | ホットペッパー/自社予約/LINE |
| 共通エンジン | 対象抽出→メッセージ生成→マルチチャネル送信→効果測定 |

### 課題テーブル

| 課題 | 影響額（年間） |
|---|---|
| リピート率が低い（30-40%） | 新規獲得コストの無駄遣い年50-150万円 |
| ホットペッパー依存 | 掲載費年36-180万円 |
| リコール管理が手作業 | スタッフ月5-10時間 |
| 次回予約の取得率が低い | 離脱顧客の逸失LTV |
| 顧客の好み情報が散在 | パーソナライズ提案ができない |

## 2〜6. 拡張ロードマップ

- **② スタッフ管理**: 指名料+歩合+物販インセンティブの給与計算。歯科「経理・給与」横展開
- **③ 口コミ・SNS集客**: Instagram/TikTok投稿の自動生成+Google口コミ返信。歯科「集患マーケ」横展開
- **④ 在庫管理**: カラー剤・シャンプー等の消費量管理。製造業「在庫最適化」横展開
- **⑤ 顧客カルテ**: 施術履歴・アレルギー・好みの電子管理
- **⑥ 採用支援**: 美容師離職率50%（3年以内）。歯科「採用支援」横展開
