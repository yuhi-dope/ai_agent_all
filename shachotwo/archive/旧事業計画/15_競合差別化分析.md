# シャチョツー 競合差別化分析

> **本資料の位置づけ**: 導入検討企業の稟議書添付資料として使用可能。
> 作成: 2026年3月 / 料金体系: 基本料¥30,000/月 + 従量課金
>
> **本資料の設計前提（02_プロダクト設計.md準拠）**:
> - **料金**: 基本料¥30,000/月 + 従量課金（ファイル¥50/件, Q&A¥30/回, BPO¥200/件, ユーザー¥2-5K/月, コネクタ¥5K/個/月）
> - **チャネルファースト**: LINE / メール / ファイルアップロード / 音声 / FAX が主入力。SaaS API は補助
> - **Day 1 Value**: ナレッジゼロでも初日から使える3機能（ファイルチェック・議事録・文書下書き）
> - **Level 0-4 段階的自動化**: リマインダー → データ収集 → 下書き → 承認実行 → 自律実行
> - **統合ファイル処理エンジン**: Excel/PDF/Word/PPT/CSV/画像/音声/CAD を統一処理
> - **3 AIペルソナ**: 参謀（経営層）/ アシスタント（管理職）/ ヘルパー（一般社員）
> - **SIer段階的代替**: 共存（0-2年）→ 浸食（2-4年）→ 代替（4-7年）

---

## 3. 「なぜ既存ツールの組み合わせではダメなのか」

### 3.1 よくある代替案: Notion + ChatGPT + Zapier

多くの企業が「既存ツールの組み合わせで同じことができるのでは？」と考える。
以下に、実際に組み合わせた場合の限界を整理する。

```
┌──────────────────────────────────────────────────────────────────┐
│  「Notion + ChatGPT + Zapier」 で代替した場合                      │
│                                                                  │
│  Notion（ナレッジ管理）                                           │
│    ・社長がドキュメントを「手で書く」必要がある                       │
│    ・業界テンプレートなし → ゼロから構築                             │
│    ・構造化はすべて人間の判断                                       │
│    → 社長の時間: 月20〜40時間（初期は毎日1-2時間）                   │
│                                                                  │
│  ChatGPT Enterprise（Q&A）                                       │
│    ・Notionのデータを参照するには別途RAG構築が必要                    │
│    ・意思決定パターンの学習機能なし                                  │
│    ・能動的な提案機能なし（聞かれたことにしか答えない）                │
│    ・ChatGPTの回答を社内ナレッジに還元する仕組みがない                │
│    → 追加開発費: ¥3M〜10M（RAG構築+連携開発）                       │
│                                                                  │
│  Zapier（自動化）                                                 │
│    ・「何を自動化するか」は人間がすべて設計する                       │
│    ・判断ロジック（社長の暗黙知）は組み込めない                       │
│    ・SaaS間の複雑な条件分岐は対応困難                               │
│    ・日本のSaaS（kintone, freee等）の対応が限定的                    │
│    → 設計+構築工数: 月10〜20時間（社内 or 外注）                     │
│                                                                  │
│  さらに必要なもの                                                  │
│    ・3つのツール間の整合性管理 → 月5〜10時間                        │
│    ・バージョンアップ追従 → 年2〜3回の再設定                         │
│    ・セキュリティ統合管理 → 3ツール別々に設定+監査                    │
│    ・ChatGPT→Zapier→Notion のデータフロー設計+保守                  │
│                                                                  │
│  ★ 致命的に欠けるもの                                              │
│    ① 暗黙知の「引き出し」機能（対話型入力+音声+テンプレ）             │
│    ② 能動的な提案（リスク検出・改善提案・What-if）                   │
│    ③ 業界テンプレート（80%完成スタート）                             │
│    ④ ネットワーク効果（匿名ベンチマーク）                           │
│    ⑤ 意思決定パターンの学習と再現                                   │
└──────────────────────────────────────────────────────────────────┘
```

### 3.2 統合コストと運用負荷の試算

| 項目 | ツール組み合わせ | シャチョツー |
|---|---|---|
| **ツール月額** | Notion ¥150K + ChatGPT Enterprise ¥450K + Zapier ¥15K = **¥615K** | **¥30K+従量（50名企業で実質¥150K〜250K）** |
| **初期構築** | RAG構築+連携開発 **¥5M〜15M** | **¥100K**（オンボーディング支援） |
| **社長の月間工数** | 40〜60時間（ドキュメント手書き+設計+確認） | **5〜10時間**（話すだけ） |
| **運用保守** | 月¥200K〜500K（外注 or 社内エンジニア） | **¥0**（SaaS月額に含む） |
| **セキュリティ管理** | 3ツール個別管理+監査 | **統合管理**（RLS+暗号化+監査ログ一元化） |
| **業界テンプレ** | なし（ゼロから構築） | **80%完成**でスタート |
| **能動提案** | 不可能（仕組み自体が存在しない） | **標準搭載** |
| **ネットワーク効果** | 不可能（自社データのみ） | **匿名ベンチマーク** |

**結論**: ツール組み合わせは月額コストが高く、初期構築費が大きく、社長の時間を大量に消費し、かつシャチョツーの核心機能（暗黙知引き出し・能動提案・業界テンプレ・ネットワーク効果）は再現不可能。
