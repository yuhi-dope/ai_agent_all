## 1. 製造業の基本構造

### 1.1 業種分類

```
シャチョツーのターゲット: 中小製造業（10-300名）約5.2万社

主な業種:
  ① 金属加工（切削・研磨）: 鉄やアルミを削って部品を作る
  ② プレス加工: 金属板を型で押して形を作る
  ③ 板金加工: 金属板を切断・曲げ・溶接
  ④ 射出成形: プラスチック部品を金型で成形
  ⑤ 食品製造: パン、菓子、飲料等の製造ライン
  ⑥ 電子部品: 基板実装、センサー、コネクタ
  ⑦ 化学・薬品: バッチ製造、連続製造

※ 見積BPOの最初のターゲットは①②③（金属加工系）
  → 図面がある + 工程が推定しやすい + チャージレートが標準化しやすい
```

### 1.2 サプライチェーン構造

```
完成品メーカー（トヨタ、日立等）
  │
  ├─ Tier 1（一次下請。大手部品メーカー）
  │    ├─ Tier 2（二次下請。中堅）
  │    │    ├─ Tier 3（三次下請。町工場）← ★ここがターゲット
  │    │    └─ Tier 3
  │    └─ Tier 2
  │
  └─ 商社・代理店（間に入って発注を仲介）

特徴:
  - 下に行くほど立場が弱い（価格交渉力が低い）
  - 下に行くほど「見積もり早く出して」のプレッシャーが強い
  - 町工場は「社長がいないと見積もれない」状態
```

### 1.3 受発注の流れ

```
引合い → 見積 → 受注 → 生産準備 → 生産 → 検査 → 出荷 → 検収 → 請求 → 入金
│         │       │                                              │
│         │       └─ 注文書受領                                   └─ 月末締め翌月払い等
│         └─ ★ここがAI化対象（見積作成→提出→受注判定）
└─ 顧客から図面+仕様が来る
```

### 1.4 工場の組織（10-300名規模）

```
社長 ─┬─ 営業/見積（社長兼任 or 専任1-3名）← ★AI化対象
      ├─ 工場長 ─┬─ 製造部門（旋盤、MC、プレス等の班）
      │          ├─ 品質管理（検査員）
      │          └─ 生産管理（スケジュール管理）
      ├─ 購買（材料・外注の手配）
      └─ 事務・経理
```

## 2. 製造業の見積とは何か

### 2.1 定義

```
見積 = 顧客から受け取った図面に対して「いくらで作れるか」を回答すること

  入力: 図面（PDF/DXF）+ 仕様（材質、数量、表面処理、納期）
  処理: 図面読解 → 工程推定 → 工数算出 → コスト計算
  出力: 見積書（金額、納期、有効期限）
```

### 2.2 なぜ重要か

```
1. 受注機会: 顧客は3社に見積を出す。最速回答に発注しがち
   → 見積作成時間の短縮 = 受注率UP

2. 利益確保: 見積が甘い → 赤字。見積が高い → 失注
   → 適正な見積精度が利益を左右する

3. 属人化リスク: 見積できる人が社長 or ベテラン1人
   → その人が倒れたら見積が止まる = 受注が止まる
```

### 2.3 見積の種類

| 種類 | 精度 | 回答時間 | 用途 |
|---|---|---|---|
| 概算見積 | ±30-50% | 即日-翌日 | 引合い初期の価格感 |
| 詳細見積 | ±5-15% | 2-7日 | 正式受注判断 |
| 試作見積 | ±10-20% | 1-3日 | 少量試作 |
| 量産見積 | ±3-5% | 3-7日 | 大量生産 |

## 3. 見積の完全業務フロー（★最重要セクション）

### 3.0 全体フロー図

```
Step 1: 引合い受領
  │
Step 2: 図面の読み取り ← 最も属人的（経験が物を言う）
  │
Step 3: 加工工程の推定 ← 2番目に属人的
  │
Step 4: 工数の算出
  │
Step 5: コスト計算
  │
Step 6: 見積書の作成・提出
```

### Step 1: 引合い受領

```
■ 顧客から届くもの:
  ① 図面（PDF/DXF/STEP/IGES/紙FAX）
  ② 仕様（材質、数量、表面処理、納期、特記事項）
  ③ 過去図面からの変更点（リピート品の場合）

■ 引合い経路:
  - メール添付（最も多い）
  - FAX（まだ多い。紙をスキャンする手間）
  - 営業が持ち帰り
  - Webフォーム（少ない）

■ この段階でのチェック:
  - 自社で対応可能な加工か
  - 材質は手配可能か
  - 納期は間に合うか
  - 数量は自社の設備キャパに見合うか
  → NO が1つでもあれば即断りor条件交渉

AI化ポイント: 引合い情報の自動構造化（メール→データ抽出）
```

### Step 2: 図面の読み取り

```
■ 図面の基本構成:
  ┌─────────────────────────────────────┐
  │                                     │
  │    正面図      側面図      平面図    │  ← 三面図（三角法）
  │                                     │
  │    詳細図（拡大部分）               │
  │                                     │
  │    ───────────────────────────────  │
  │    表題欄:                          │
  │      品名、図番、材質、スケール、    │
  │      表面処理、作成者、日付          │
  │    ───────────────────────────────  │
  │    注記:                            │
  │      一般公差、表面粗さ、            │
  │      熱処理条件、特記事項            │
  └─────────────────────────────────────┘

■ 読み取るべき情報:

  1. 形状（丸物か角物か板物か）
     → 工程推定の出発点

  2. 寸法（外径、内径、長さ、幅、高さ、穴位置等）
     → 工数算出に必要

  3. 公差（許容される寸法の誤差）
     普通公差（JIS B 0405）: ±0.1〜±0.5mm → 普通の加工でOK
     厳しい公差: ±0.01〜±0.05mm → 研磨が必要 → コストUP
     はめあい公差: H7/g6等 → 用途に応じた精密加工

  4. 表面粗さ
     Ra 6.3（▽1つ）: 粗い → 旋盤のまま
     Ra 1.6（▽2つ）: 普通 → 仕上げ加工
     Ra 0.4（▽3つ）: 鏡面 → 研磨必須 → コスト大幅UP

  5. 材質
     SS400: 一般構造用鋼（安い、加工しやすい）
     S45C: 機械構造用炭素鋼（硬い、焼入れ可能）
     SUS304: ステンレス（錆びない、加工しにくい → コストUP）
     A5052: アルミ合金（軽い、加工しやすい）
     C3604: 真鍮（快削性良い）

  6. 表面処理
     無電解ニッケルメッキ / 硬質クロムメッキ / アルマイト /
     黒染め / 塗装 / めっきなし
     → 外注になることが多い → 外注費を見積もる

AI化ポイント: Document AI + LLM で図面をOCR→形状・寸法・材質を抽出
精度: 初期60-70%（人間の確認必須）
```

### Step 3: 加工工程の推定

```
■ 形状→工程のマッピング（最も重要な暗黙知）

  丸物（シャフト、ピン、ブッシュ等）:
    材料切断 → 旋盤加工（外径・内径・端面）→ MC加工（穴・溝）→ 研磨 → 表面処理

  角物（ブロック、プレート加工品等）:
    材料切断 → MC加工（外形・穴・ポケット）→ 研磨 → 表面処理

  板物（ブラケット、カバー等）:
    レーザー切断 → 曲げ → 溶接 → 表面処理

  複雑形状:
    MC加工 → ワイヤーカット → 放電加工 → 研磨　

■ 標準的な工程順序:
  1. 材料手配（購入 or 在庫）
  2. 材料切断（バンドソー等）
  3. 粗加工（大まかに削る）
  4. 熱処理（必要な場合。外注）
  5. 仕上げ加工（最終寸法に仕上げる）
  6. 研磨（高精度が必要な場合）
  7. 表面処理（メッキ、塗装等。外注）
  8. 検査（寸法測定、外観検査）
  9. 梱包・出荷

■ 熱処理が必要なケース:
  - 図面に「HRC 50-55」等の硬度指定がある
  - S45C等の焼入れ可能材で硬さが必要な用途
  - → 焼入れ→焼戻し→研磨が追加 → コスト+納期増

AI化ポイント: LLMに「形状+材質+公差→工程」の推定をさせる
精度: 初期60-70%。過去実績で改善
```

### Step 4: 工数の算出

```
■ 工数の構成:
  加工時間 = 段取り時間 + (サイクルタイム × 数量)

  段取り時間（だんどりじかん）:
    = 加工を始めるまでの準備時間
    - 治具のセット: 15-60分
    - ワーク（部品）の固定: 5-15分
    - プログラム呼び出し・調整: 10-30分
    - 初品の加工・測定: 15-30分
    - 合計: 30分〜2時間（1回のみ。数量に関わらず固定）

  サイクルタイム:
    = 1個あたりの加工時間
    - 加工プログラムの実行: 数分〜数時間（部品の複雑さによる）
    - ワーク脱着: 1-5分
    - 計測（抜き取り検査）: 数分

■ 工数算出の具体例:
  部品: シャフト φ30×100mm、材質S45C、1工程（CNC旋盤）、数量100個

  段取り時間: 45分（プログラムあり、治具セット、初品確認）
  サイクルタイム: 8分/個（外径・端面・面取り）
  合計加工時間: 45分 + 8分×100個 = 845分 = 14.1時間

  ※ 数量が1個の場合: 45分 + 8分 = 53分
  → 段取りの按分で1個あたりの単価が大きく変わる

AI化ポイント: 形状・寸法からサイクルタイムを推定する
精度: 初期±30-50%。過去実績で改善
```

### Step 5: コスト計算

```
■ 見積金額の構成:

  見積金額 = 材料費 + 加工費 + 表面処理費 + 外注費 + 検査費 + 管理費 + 利益

  1. 材料費
     = 使用重量 × kg単価 + 端材ロス（10-20%）
     or 定尺品単価（丸棒1本、板1枚）

     例: S45C 丸棒 φ35×110mm（加工代含む）
         体積 = π × 0.0175² × 0.11 = 0.0001057 m3
         重量 = 0.0001057 × 7,850kg/m3 = 0.83kg
         材料費 = 0.83kg × ¥150/kg × 1.15（端材ロス）= ¥143

  2. 加工費
     = 合計加工時間 / 60 × チャージレート

     例（上記シャフト100個の場合）:
         CNC旋盤 14.1h × ¥6,000/h = ¥84,600
         1個あたり: ¥846

  3. 表面処理費（外注）
     無電解ニッケルメッキ: ¥200-1,000/個（サイズ・膜厚による）
     アルマイト: ¥100-500/個
     黒染め: ¥50-200/個

  4. 検査費
     全数検査: サイクルタイムに含めることが多い
     三次元測定: ¥2,000-5,000/回（特殊計測）

  5. 管理費
     = (材料費 + 加工費 + 外注費) × 管理費率（10-20%）
     梱包、運搬、事務処理、間接人件費等

  6. 利益
     = (原価合計 + 管理費) × 利益率（10-30%）
     新規: 高め（20-30%）、リピート: 低め（10-15%）
     大口: 低め、小口: 高め

■ 見積書の例:

  品名: シャフト A-001
  材質: S45C
  数量: 100個
  ──────────────────────
  材料費:      ¥14,300（¥143×100）
  加工費:      ¥84,600（CNC旋盤 14.1h）
  表面処理費:  ¥30,000（黒染め ¥300×100）
  検査費:      ¥5,000
  ──────────────────────
  原価小計:    ¥133,900
  管理費(15%): ¥20,085
  ──────────────────────
  原価合計:    ¥153,985
  利益(15%):   ¥23,098
  ──────────────────────
  見積金額:    ¥177,083 → ¥177,000（端数調整）
  単価:        ¥1,770/個
```

### Step 6: 見積書の作成・提出

```
■ 見積書の形式:
  ① 明細あり: 工程別に金額を記載（丁寧だが手の内を見せる）
  ② 明細なし: 合計金額のみ（一般的）
  ③ 一式見積: 「加工一式 ¥XXX」（詳細不要の場合）

■ 記載項目:
  - 宛先（会社名）
  - 見積番号（自動採番）
  - 品名、図番
  - 数量、単価、金額
  - 材質、表面処理
  - 納期
  - 有効期限（通常30日）
  - 支払条件（月末締め翌月末払い等）
  - 備考（前提条件、除外事項）

■ 提出後:
  受注 → 注文書受領 → 生産
  失注 → 理由確認（価格? 納期? 品質?）→ 次回に活かす
  保留 → フォローアップ
```

## 3+. 見積構造型別の詳細フロー

> 上記Section 3は金属切削（Type A）の見積フローを詳述した。
> ここでは4つの見積構造型それぞれの特徴と計算例を示す。
> 詳細な定義は `e_02a_製造業セグメントモデル.md` のL3セクションを参照。

### Type A: チャージレート型（金属切削・プラスチック射出成型・ゴム成型）

```
■ 基本式: 加工費 = (段取り時間 + サイクルタイム × 数量) ÷ 60 × チャージレート

■ 金属切削: 上記Section 3のStep 1-6を参照

■ 3業種の構造比較:
  │              │ 金属切削         │ プラスチック射出  │ ゴム成型         │
  ├──────────────┼──────────────────┼──────────────────┼──────────────────┤
  │ 金型         │ 不要（治具のみ） │ ★必須（高額）   │ ★必須（中額）   │
  │ サイクルタイム│ 数分〜数時間     │ 数十秒〜数分     │ 数分〜数十分     │
  │ 二次加工     │ 表面処理・研磨   │ ゲートカット     │ バリ取り（大）   │
  │ 見積構造     │ ピース単価のみ   │ 金型費+ピース単価 │ 金型費+ピース単価 │
  │ 典型ロット   │ 1〜500個         │ 500〜100,000個   │ 100〜10,000個    │
```

#### プラスチック射出成型の見積

```
■ 見積 = 金型費（初回のみ）+ ピース単価 × 数量

  金型費を決める要素:
    型構造: 2プレート(¥50-200万) / 3プレート(¥150-500万) / ホットランナー(¥300-1,000万)
    金型材質: S50C(試作,~10万shot) / NAK80(中量産,~30万shot) / SKD61(量産,~100万shot)
    キャビティ数: 1個取り→4個取り→16個取り（取り数で生産効率UP、金型費UP）
    スライド: アンダーカット1箇所あたり+¥30-100万

  ピース単価の構成:
    材料費 = ショット重量(g) × 樹脂kg単価 ÷ 1000
    成型費 = サイクルタイム(秒) ÷ 3600 × チャージレート(¥/h) ÷ キャビティ数
    + 二次加工費（ゲートカット/印刷/組立/検査）

■ 主要樹脂の単価:
  PP: ¥150-250/kg  ABS: ¥250-400/kg  PC: ¥500-750/kg
  POM: ¥400-550/kg  PA66: ¥500-800/kg  PEEK: ¥15,000-22,000/kg

■ 成型機チャージレート:
  50t: ¥3,000-4,500/h  100t: ¥4,000-6,000/h  200t: ¥6,000-9,000/h
  350t: ¥8,000-12,000/h  850t: ¥15,000-25,000/h

■ 計算例: 自動車スイッチパネル（ABS難燃, 45g, 月5,000個）
  金型費: ¥250万（2プレート+スライド2箇所, NAK80, 2個取り）
  ピース単価: ¥170（材料¥20 + 成型¥45 + 二次加工¥65 + 管理費・利益）
  金型償却込み実質単価: ¥178（¥250万÷30万個=¥8.3）
```

#### ゴム成型の見積

```
■ 金型費+ピース単価（射出成型と同じ2段構造）
■ 特徴: 加硫時間が長い（3-30分）、バリ取りがピース単価の20-40%

  金型費目安:
    Oリング金型(1-4個取り): ¥10-30万
    パッキン金型: ¥20-60万
    異形品・複合品: ¥50-250万

  主要ゴムの単価:
    NBR: ¥500-900/kg  EPDM: ¥500-800/kg  シリコーン: ¥2,000-4,000/kg
    フッ素ゴム: ¥5,000-12,000/kg

  成型機チャージレート:
    小型プレス(~50t): ¥3,000-5,000/h
    中型プレス(50-150t): ¥4,500-7,000/h
    大型プレス(150-300t): ¥6,000-10,000/h

■ 計算例: Oリング（NBR70°, P-25, 3,000個）
  金型費: ¥40万（圧縮成型, S50C, 16個取り）
  ピース単価: ¥50（材料¥1 + 成型¥22 + バリ取り¥8 + 検査¥6 + 管理費・利益）
  → 成型費43%、バリ取り16%で原価の約60%を占める
```

### Type B: 面積/回数/長さ単価型（板金加工・印刷）

```
■ 基本式: 加工費 = 単位量（面積/回数/長さ）× 単価
■ Type Aとの違い: 「設備×時間」ではなく「どれだけ切ったか/曲げたか」

■ 板金の3大工程:
  ①レーザー切断: 切断長(mm) × mm単価 + ピアス数 × ピアス単価
    SPCC t1.6: ¥3-5/mm, SUS304 t2.0: ¥7-10/mm, A5052 t2.0: ¥4-6/mm
  ②曲げ(ベンダー): 曲げ回数 × 単価(¥150-600/回) + 段取り
  ③溶接: TIG ¥10-20/mm, MIG ¥5-10/mm, スポット ¥30-50/点

■ 計算例: SUS304ブラケット（t2.0, 200×150mm, L曲げ1回, スポット4点, 10個）
  材料費: ¥3,050  切断費: ¥68,028  曲げ費: ¥5,000  溶接費: ¥3,100
  → 見積金額: ¥106,000（¥10,600/個）

■ 印刷の見積構成:
  版代: CTP版 ¥1,500-3,000/版 × 色数（4色両面=8版）
  紙代: 紙種×連量×枚数 + 予備紙10-15%
  印刷費: 基本料金(色合わせ) + 通し数 × 通し単価
  加工費: 断裁/折り/製本/型抜き/箔押し等

■ 計算例: A4チラシ（4色両面, コート90kg, 10,000部）
  版代¥16,000 + 紙代¥33,600 + 印刷費¥94,400 + 加工費¥1,500
  → 見積金額: ¥200,000（¥20/部）
  → 50,000部なら¥10-12/部（固定費按分が効く）
```

### Type C: BOM組立型（機械メーカー・家具）

```
■ 基本式: 原価 = Σ(部品費) + 設計費 + 組立費 + 試運転費 + 輸送費
■ Type Aとの違い: 「1部品の加工」ではなく「製品全体の構成」
■ 部品1点ごとにType A/Bの見積が入れ子になる

  部品の3区分:
    購入品: カタログ価格×掛率(0.6-0.8)×数量
    内製品: Type A/Bの見積が入れ子
    外注品: 外注先見積 or 概算

  設計者レート: ¥30,000-50,000/日
  組立者レート: ¥20,000-35,000/日
  試運転: ¥30,000-45,000/日

■ 計算例: 小型コンベア装置（BOM 30部品, 設計2日, 組立3日, 試運転1日）
  部品費¥205,500 + 設計¥80,000 + 組立¥75,000 + 試運転¥35,000 + 梱包輸送¥65,000
  → 見積金額: ¥635,000
```

### Type D: バッチ配合型（食品OEM・化学受託）

```
■ 基本式: 原価 = Σ(原料×配合比×kg単価) × バッチ数 + 工程費 + 包装資材費 + 検査費
■ Type Aとの違い: 「設備×時間」ではなく「原料配合×バッチサイクル」

  配合展開: レシピの各原料% × バッチサイズ(kg) = 原料使用量
  工程費: 計量→混合→加熱殺菌→冷却→充填→ラベル→箱詰め × ライン費用
  ロット切替費: ライン洗浄（アレルゲン管理で1-4時間）

■ 計算例: ドレッシングOEM（500ml × 5,000本, 6バッチ）
  原料費¥772,650 + 工程費¥741,000 + 包装資材¥277,860 + 検査¥55,000 + 洗浄¥20,000
  → 見積金額: ¥2,400,000（¥480/本）
```

## 4. 判断分岐・例外パターン（★AI化に必須）

### 4.1 図面が不完全な場合

```
よくあるケース:
  - 寸法が一部抜けている → 顧客に確認 or 推測で見積（注記付き）
  - 公差の指定がない → 一般公差（JIS B 0405 中級）で見積
  - 材質未指定 → SS400（最も一般的）で仮見積
  - 表面処理未指定 → なしで見積（注記付き）

AI化: 不完全な項目を自動検出→「確認事項リスト」を自動生成
```

### 4.2 特急品

```
通常納期: 2-4週間
特急: 1週間以内 → 割増 20-50%
超特急: 2-3日 → 割増 50-100%（残業・休日出勤）

判断: 現在の設備稼働状況と照らし合わせて可否判断
```

### 4.3 小ロット vs 量産

```
1個:   段取り按分が大きい → 単価が高い（段取り÷1個）
10個:  段取り按分が減る → 単価下がる
100個: さらに下がる
1000個: 治具・プログラム最適化の余地 → さらに下がる
10000個: 専用設備の検討 → 大幅に下がる

数量ブレイク（数量割引の段階）:
  1-9個: 基本単価
  10-49個: 5-10%引き
  50-99個: 10-15%引き
  100個以上: 15-25%引き
```

### 4.4 材料費の変動

```
鋼材相場は3-6ヶ月で10-20%変動することがある
  → 見積有効期限（通常30日）内に材料を発注できるか
  → 長期契約の場合は「材料費変動条項」を入れる
```

### 4.5 断るべき案件の判断

```
断る基準:
  - 自社に対応設備がない（5軸MCが必要だが持っていない等）
  - 公差が自社の加工精度限界を超える
  - 材質が特殊すぎて手配不可
  - 納期が物理的に不可能
  - 利益が出ない（価格が合わない）
  - 品質リスクが高い（クレーム歴のある顧客）

→ AI化: 「受注可否チェック」を自動判定
```

### 4.6 材料の最小購入単位

```
丸棒は4m or 5.5mバーで購入。50mmの部品1個でも4m分の材料費が発生。
  → 端材の在庫管理が重要。前の案件の残りを使えば材料費が劇的に下がる
AI化: 端材在庫DBを構築→「φ50 S45C 残り1.2mあり」→ 材料費¥0で見積
```

### 4.7 端材の活用判断

```
端材を使う条件:
  - 材質が一致（ミルシートが必要な場合は同一ロットか確認）
  - サイズが足りる（削り代含めて）
  - 保管状態が良好（錆・変形なし）
AI化: 端材DBと照合→自動で「端材利用可」フラグ→材料費を自動調整
```

### 4.8 外注先の価格差

```
同じメッキでも外注先で2-3倍違う:
  A社: 無電解Ni ¥300/個（バレルメッキ。大量向き）
  B社: 無電解Ni ¥800/個（ラックメッキ。品質重視）
  → 品質要求と数量で最適な外注先を選ぶ
AI化: 外注先マスタに品質レベル・最小ロット・納期を登録→自動選定
```

### 4.9 数量変更への対応

```
「100個って言ったけど50個で」→ 段取り按分が2倍 → 単価UP
「10個の追加」→ 同じ段取りで追加可能なら安い。金型外したら段取り再発生
AI化: 数量変更時に自動再計算→「50個だと単価¥X→¥Yになります」を即回答
```

### 4.10 値引き要求

```
「もう少し安くならない?」→ 削れるポイント:
  ① 利益率を下げる（最終手段。15%→10%）
  ② 表面処理のグレードダウン（硬質Cr→無電解Ni）
  ③ 公差の緩和提案（±0.01→±0.02で研磨工程を省ける）
  ④ ロットまとめ（月1回×50個→3ヶ月分150個で段取り按分削減）
AI化: 「¥X安くしたい」→ 自動で①〜④の選択肢と影響額を提示
```

### 4.11 加工性の問題（DFM）

```
図面通りだと加工できない or 非常に高くなるケース:
  - 深い穴（L/D比が10超。ドリルが届かない or 折れる）
  - 薄肉部（0.5mm以下の壁。加工中にビビリ・変形）
  - 鋭角コーナー（MC加工ではRが残る。ワイヤーカットが必要→高額）
  → DFM提案:「ここをR3にすればMCだけで加工可能。¥X安くなります」
AI化: 形状解析→DFMチェック→代替形状と価格差を自動提案（Phase 2+）
```

### 4.12 熱処理後の追加工程

```
焼入れ(HRC50以上)後は通常の切削不可→研磨 or 放電加工が必要
  見積でよくある見落とし:
    焼入れ前の粗加工 → 焼入れ → 研磨で仕上げ（3工程）
    を「切削+焼入れ」の2工程で見積もってしまう→赤字
AI化: 材質+硬度指定を検出→「研磨工程が必要」を自動警告
```

### 4.13 複合加工品の工程設計

```
旋盤→MC→研磨→メッキ→仕上げMCの5工程品:
  - 各工程間で寸法検査が必要（中間検査費）
  - メッキ厚を考慮した寸法設計（メッキ前はφ29.98→メッキ後φ30.00）
  - 工程順序を間違えると精度が出ない
AI化: 工程数が4以上→「複合加工品チェックリスト」を自動表示
```

### 4.14 現物からの加工（リバースエンジニアリング）

```
「図面ないけどこれと同じの作って」→ 現物の寸法測定が必要
  追加費用: 測定 ¥5,000-30,000（部品の複雑さによる）+ 図面作成 ¥10,000-50,000
  3Dスキャンの場合: ¥20,000-100,000（外注）
AI化: 現物写真→概算サイズ推定→概算見積を即答。精密測定は別途
```

### 4.15 試作 vs 量産の価格差

```
同じ部品でも:
  試作1個: ¥15,000/個（段取り+プログラム費が全部1個に乗る）
  量産100個: ¥2,000/個（段取り按分+最適化効果）
  量産1,000個: ¥1,200/個（治具製作で更に効率化）
  → 試作は量産の5-15倍が普通。顧客に「量産時はこの単価」と併記する
AI化: 試作見積時に「量産時の参考単価」を自動併記
```

### 4.16 材料調達リードタイム

```
即納材: SS400丸棒、SUS304丸棒（在庫豊富。翌日入手可能）
1-2週間: SUS316L、SCM435、A7075（注文品。流通在庫が少ない）
3-4週間: チタン合金、インコネル、ハステロイ（特殊材。都度手配）
  → 納期回答の前に材料納期を確認する必要がある
AI化: 材料マスタに「標準リードタイム」列を追加→納期自動計算
```

### 4.17 外注ボトルネック

```
全体納期 = MAX(各工程の所要日数の累計)
  内製加工3日+メッキ外注10日+仕上げ2日 = 15日
  → メッキがボトルネック。内製をいくら急いでも15日は切れない
  → 特急メッキ: 割増50-100%で5日に短縮可能
AI化: 工程表の中で最長パスを自動検出→ボトルネック工程を表示
```

### 4.18 設備空き状況

```
MCが3週間先まで埋まっている:
  → 通常品: 「納期4週間」で回答
  → 特急: 夜間・休日に割り込み → 割増30-50%
  → 外注: 協力工場に出す → マージン10-15%乗せる
AI化: Phase 1では手動。Phase 2で設備稼働カレンダー連携
```

### 4.19 検査書類の費用

```
検査成績表の工数:
  簡易版（主要寸法のみ）: ¥500-1,000/ロット
  標準版（全寸法記録）: ¥1,000-3,000/ロット
  詳細版（3次元測定+写真+Cpk）: ¥5,000-30,000/ロット
  → 見積段階で「検査成績表要否」を確認しないと後で揉める
AI化: 顧客の品質レベル(Q1-Q4)から自動で検査費を加算
```

### 4.20 材料証明の手配

```
ミルシート（材料メーカーの成分証明書）:
  通常: 材料商社経由で無料〜¥3,000/件
  特殊材: ¥5,000-10,000/件
  指定メーカー指定ロット: ¥10,000-30,000/件（在庫から探す手間）
  → 航空宇宙・医療向けは必須。一般品は不要
AI化: 顧客の産業(IND-D/E)を検出→「ミルシート要」を自動フラグ
```

### 4.21 得意先別の暗黙ルール

```
A社: 利益率15%。見積有効期限60日。運賃込み。FAX回答。
B社: 利益率20%。見積有効期限30日。運賃別途。メール回答。PDF添付。

  → これが社長の頭にしかない。ベテランが退職すると失われる
```

### 4.22 消費税・運賃の扱い

```
見積に含む/別途は得意先によって違う:
  パターンA: 税抜表示、運賃別途（最も一般的）
```

### 4.23 支払い条件の影響

```
支払い条件で実質コストが変わる:

  手形60日: 90日の資金拘束。年利3-4%相当
```

### 4.24 相見積もり対策

```
「3社で見積取ってます」と言われたら:
  ① 価格勝負: 利益率を下げる（消耗戦。避けたい）
  ② 納期勝負: 「うちなら1週間早く出せます」（AI即答で差別化）
  ③ 品質勝負: 「うちはCpk管理できます」（品質レベルUP提案）
  ④ 提案勝負: DFM提案で「この形状変更で¥X安くなります」
AI化: 過去の受注/失注データから「この得意先はどこで決める？」を分析
```

## 4+. 原価計算の「罠」集（初心者がやりがちなミス）

```
■ 材料費の罠:
  1. 削り代を忘れる → φ50部品に必要な素材はφ55（片側2.5mmの削り代）
  2. 端材を無視 → 4mバーから50mm切り出し。残り3.95mは端材（or在庫）
  3. 材料相場が古い → SUS304は半年で10-20%変動。3ヶ月前の単価→赤字
  4. 購入単位を無視 → 真鍮C3604は1本(2m)単位。50mm部品1個でも2m分

■ 加工費の罠:
  5. 段取り忘れ → 1個の見積でサイクルタイムだけ→段取り30分の¥3,000-5,000が丸々抜ける
  6. 被削性無視 → SUS304はSS400の1.7倍の加工時間。同じ形状でも材質で全然違う
  7. 搬送忘れ → MC→研磨→メッキ(外注)→仕上げ。移動回数×ハンドリング費が抜ける
  8. 小ロット過小評価 → 5個: 段取り按分¥600/個。100個: ¥30/個。20倍の差

■ 外注費の罠:
  9. ミニマムチャージ → メッキ1個でも最低¥5,000-10,000。10個なら¥500-1,000/個
  10. 熱処理後の仕上げ忘れ → 焼入れで歪む→研磨追加。見積に入れ忘れがち
  11. 外注納期未考慮 → メッキ2週間→全体4週間。特急なら割増

■ 管理費・利益の罠:
  12. 利益率の基準不統一 → 原価×20%=¥12,000。売値に対しては16.7%。どっち?
  13. 得意先ルール忘れ → A社15%、B社20%。間違えると「前回と違う」と言われる
  14. 消費税・運賃の扱い → 含む/別途が得意先で違う。入れ忘れると丸損

→ AI化ポイント: これら全てを見積生成時に自動チェック→警告を出す仕組みにする
```
