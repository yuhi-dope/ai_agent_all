## 支払いロジックの大前提

### 院長の時間単価 = 12,000〜25,000円/時間

院長は**治療して初めて収入が発生する**。なのに月20〜40時間を事務作業に費やしている。この時間を診療に振り替えるだけで**月24〜100万円の増収**。「月5〜10万円のBPO」は院長の事務時間のごく一部を解放するだけで元が取れる。

## 1. レセプト点検・請求最適化

### 支払いロジック（月2〜5万円）

```
■ 直接効果①：査定損失の回避
  月間請求額600万円 × 査定率0.3%改善 = 月1.8万円 = 年21.6万円
  月間請求額1,000万円の医院なら = 月3万円 = 年36万円

■ 直接効果②：返戻対応の事務工数削減
  返戻の再請求処理 = スタッフ月5〜10時間 = 月8〜15万円の人件費

■ 間接効果：院長のレセプトチェック時間削減
  月3〜5時間 × 時間単価15,000円 = 月4.5〜7.5万円 = 年54〜90万円相当

■ 合計効果：年84〜141万円
  → BPO月3万円（年36万円）で ROI 2.3〜3.9倍
```

**なぜ払うか:** レセプトは「毎月10日」の法定期限があり逃げられない。査定=直接の収入減。しかも属人化が深刻で「ベテラン事務員1人」に依存している医院が大半。この1人が辞めると業務が崩壊する恐怖が支払い動機になる。

### 基礎データ

| 医院規模 | ユニット数 | 1日患者数 | 月間レセプト枚数 | 月間保険収入 |
|---|---|---|---|---|
| 小規模（開業1年目） | 2〜3台 | 10〜15人 | 150〜250枚 | 200〜400万円 |
| 標準（平均的医院） | 3〜4台 | 20〜25人 | 300〜450枚 | 400〜600万円 |
| 中規模 | 5〜6台 | 30〜40人 | 500〜600枚 | 700〜900万円 |
| 大規模（年商1億円） | 7台以上 | 50〜60人 | 600〜800枚 | 800万〜1,200万円 |

- レセプト平均単価: 開業初期は約2,000点、安定期は約1,288〜1,400点
- ユニット1台あたりの目安: レセプト120枚/月
- 1枚あたりの点検時間: 2〜5分
- 500枚規模の医院で延べ15〜25時間/月の点検工数

### 返戻・査定が発生する原因トップ5

1. **患者資格情報の誤り**（保険証の記号番号、有効期限切れ、資格喪失後受診）
2. **病名と診療内容の不整合**（病名漏れ、部位の不一致）
3. **算定ルール違反**（重複算定、回数超過、算定不可の組み合わせ）
4. **記載漏れ・記載間違い**（生年月日誤記、負担割合ミス、摘要欄未記入）
5. **書類不備**（指導計画書・指導記録の添付漏れ）

### 業務フロー

```
【日次（毎日）】
  患者来院 → 保険証確認・受付登録 → カルテ入力
  → 診療内容のレセコン入力 → 会計処理
                    ↓
【月初（1〜5日）】
  前月分レセプト一括出力 → 点検開始
  ├─ 病名と処置の整合性チェック（根管治療なのに歯髄炎の病名がない等）
  ├─ 算定ルール違反チェック（同月に算定不可の組み合わせ、回数超過）
  ├─ 部位の一致確認（治療部位と病名の部位）
  ├─ 摘要欄の記載確認（7番残存時の咬合状態等）
  └─ 患者資格情報の有効性確認（保険証の期限切れ等）
                    ↓
【月初（5〜8日）】
  院長による最終確認（3〜5時間かけて目視チェック）
                    ↓
【毎月10日（法定期限）】
  オンライン請求（社保・国保の審査支払機関へ送信）
                    ↓
【翌月〜翌々月】
  返戻・査定レセプトの確認 → 原因調査
  → 再請求 or 再審査請求の書類作成・再送信
```

## 2. 採用業務（特に歯科衛生士）

### 支払いロジック（月3〜5万円 or 成功報酬）

```
■ 歯科衛生士の有効求人倍率 = 22.6倍（1人を23医院で取り合い）
■ 歯科衛生士1人の年間売上貢献 = 約1,440万円（月120万円）
■ 転職経験あり = 約76.4%（日本歯科衛生士会2020年調査）
■ 新卒3年以内離職率 = 約30〜35%

■ 採用単価（現状）：
  ・人材紹介会社（成功報酬） = 70〜80万円/人（年収の20%）
  ・グッピー（クリック課金） = 10〜15万円/人
  ・ジョブメドレー（成約課金） = 20〜25万円/人
  ・掲載型求人広告 = 40〜100万円/人

■ 早期離職（3ヶ月以内）の損失：
  採用コスト70〜100万 + 3ヶ月分給与90万 + 教育コスト30〜50万
  + 再採用コスト70〜100万 = 合計260〜340万円

■ AI採用支援の効果：
  ・求人票最適化→応募数1.5〜2倍
  ・スカウト文面自動生成→院長の週2時間削減
  ・面接評価テンプレート→ミスマッチ採用防止
  → 採用単価30%削減（100万→70万円）= 1人あたり30万円節約
  → 早期離職率を20%削減→年1回の離職回避で260万円節約

■ 月3万円（年36万円）で採用1人あたり30万円節約 → ROI 8倍以上
```

**なぜ払うか:** 歯科衛生士は超売り手市場（求人倍率22.6倍）で、1人辞めると「残ったスタッフの負荷増→連鎖離職」が起きる。衛生士1人=年間売上1,440万円の喪失。採用は院長にとって「経営存続に関わる最重要課題」だが、診療の合間にやるしかなく、応募に24時間以内に返信できず他院に流れるケースが頻発。

### 主な離職理由

1. 結婚 (29.3%)
2. 院長との人間関係 (29.0%)
3. 出産・育児 (28.7%)
4. 給与・待遇への不満 (22.3%)
5. 仕事内容への不満 (22.0%)
6. 筋骨格系障害（腰痛・肩こり・手首の痛み）

## 3. 経理・給与計算

### 支払いロジック（月2〜3万円）

```
■ 院長が経理に費やす時間 = 月5〜10時間
  → 5時間 × 15,000円 = 月7.5万円の機会損失

■ 給与計算の複雑さ（歯科特有）：
  ・職種別基本給（歯科医師/衛生士/助手/受付で全員異なる）
  ・歩合制の計算（自費売上の○%、ホワイトニング1件+1,500円等）
  ・パート時給スタッフとの混在

■ 税理士顧問料 = 月3〜6万円を既に支払い中
  → AI経理支援で税理士への「丸投げ範囲」を削減し月1〜2万円節約
  + 院長時間5時間削減 = 月7.5万円

■ 月2万円のBPOで月9.5万円の効果 → ROI 4.7倍
```

**なぜ払うか:** 院長の多くが「丸投げ」で税理士に月3〜6万円+決算報酬を払っている。つまり経理を外に出す習慣は既にある。問題は税理士が「月次の数字」を翌月にしか出さないので、経営判断が常に1ヶ月遅れること。リアルタイムで自費率・保険点数・経費を可視化するAIツールに月2〜3万円は「税理士への追加費用」として許容範囲。

## 4. リコール管理

### 支払いロジック（月1〜3万円）

```
■ リコール率：全国平均60%、40%以下の医院も多い
  優良医院は80〜95%（LINE自動連絡・動画説明を導入）

■ リコール対象200名の医院で率10%改善した場合：
  月20名増 × 10,000円（定期検診10割） = 月20万円 = 年240万円の増収

■ さらに：
  ・リコール患者は自費治療（ホワイトニング等）にもつながる→LTV 1.6倍
  ・新患獲得コスト（5,000円/人）の1/5〜1/7で既存患者を維持できる
  ・患者1人あたりLTV = 約30万円（年3万円 × 10年通院）

■ 月2万円（年24万円）のBPOで年240万円の増収 → ROI 10倍
```

**なぜ払うか:** リコールは「最も確実に売上を増やせる施策」。新患獲得より5〜7倍コスト効率が良い。しかも定期検診は予約が読めるため、チェア稼働率の安定化にも貢献。多くの院長が「リコールが大事なのは分かっている。でも手が回らない」状態。月2万円で年240万円増収のROIは即決ライン。

| か強診加算 | 120点 | 1,200円 |

患者の窓口負担は3割 = 1,500〜3,000円程度

## 6. 労務管理・行政手続き

### 支払いロジック（月1〜2万円）

```

  ・三師届・業務従事者届（2年に1回、偶数年12月末）
  ・施設基準届出（か強診・外来環等）
  ・レセプト請求（毎月10日）
  ・X線装置設置届
  ・保険医療機関指定申請

  ・届出期限の自動アラート（2年に1回の三師届等を忘れない）

```

**なぜ払うか:** 歯科医院は常時10人以上のスタッフがいるケースが多く、就業規則の届出義務がある。しかし「事務長」がいない医院がほとんどで、院長が労務も兼務。有給5日取得義務違反は罰金30万円/人。届出漏れは保険医取消のリスクもある。「知らなかった」では済まない法的リスクの回避として月1〜2万円は安い。
