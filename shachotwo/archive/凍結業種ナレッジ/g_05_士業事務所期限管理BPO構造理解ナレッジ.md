## 1. 士業事務所の基本構造

### 1.1 3士業の違い

| 項目 | 税理士 | 社労士 | 弁護士 |
|---|---|---|---|
| 正式名称 | 税理士 | 社会保険労務士 | 弁護士 |
| 管轄 | 国税庁・税務署 | 厚労省・年金事務所 | 裁判所・弁護士会 |
| 主な業務 | 税務申告、記帳代行、決算、税務相談 | 社保手続き、給与計算、就業規則、助成金 | 訴訟代理、契約書、法律相談、債権回収 |
| 事務所数 | 約26,000事務所 | 約15,000事務所 | 約18,000事務所 |
| 顧問料相場 | 月2-5万/法人 | 月2-5万/法人 | 月3-10万/法人 |
| 決算料 | 年1回20-30万 | なし | なし |
| 繁忙期 | 12-3月（年末調整+確定申告） | 6-7月（算定基礎届+年度更新） | 通年（期日次第） |
| 懲戒リスク | 期限ミス→業務停止 | 届出漏れ→業務停止 | 期日徒過→懲戒+損害賠償 |

### 1.2 事務所の組織（3-30名）

```
所長（有資格者）
  ├─ 有資格者（勤務税理士/社労士/弁護士）: 0-5名
  ├─ 補助者/パラリーガル: 1-10名
  │    → 税理士事務所: 記帳、申告書下書き
  │    → 社労士事務所: 届出書類の作成
  │    → 弁護士事務所: 書面調査、書類作成
  └─ 事務員: 1-5名
       → 受付、電話対応、ファイリング、★期限管理

特徴:
  - 所長 = 経営者 = メインの実務者 = 全ての最終責任者
  - 期限管理は所長の頭の中 or 事務員のExcelカレンダー
  - 担当者が辞めると期限管理ごと崩壊するリスク
```

  月額顧問料: 80社 × 平均¥30,000 = 月240万円
  スポット: 助成金（成功報酬）、就業規則等 = 月50-100万
  年商: 約3,500-4,000万円

  月額顧問料: 20社 × 平均¥50,000 = 月100万円
  個別案件: タイムチャージ or 着手金+成功報酬 = 月200-500万

## 2. 期限管理とは何か

### 2.1 定義

```
期限管理 = 顧問先ごとの法定期限・届出期限を漏れなく管理し、
           期限内に申告・届出・手続きを完了させる業務

  入力: 顧問先情報（決算月、届出状況等）+ 法定期限のカレンダー
  処理: 期限のリスト化 → 作業スケジュール化 → 進捗管理 → 完了確認
  出力: 期限管理台帳、作業指示、完了報告
```

### 2.2 なぜ重要か

```
期限ミスの結果:
  ① 顧問先への損害 → 損害賠償請求
     - 延滞税（遅延利息）の発生
     - 青色申告の取消し（期限後申告が2回で取消し）
     - 助成金の受給機会喪失
     - 裁判の権利喪失（控訴期限の徒過 = 確定判決）

  ② 士業本人の懲戒
     - 戒告（注意）
     - 業務停止（数ヶ月〜2年）
     - 登録抹消（最も重い）

  ③ 信頼喪失
     - 顧問先の離脱 = 直接の売上減
     - 口コミでの評判悪化

具体例:
  法人税の申告期限を1日遅れた → 延滞税+無申告加算税 = 数十万〜数百万円
  控訴期限（14日）を1日でも過ぎた → 判決確定。取り返しがつかない
  算定基礎届を出し忘れた → 社員の社会保険料が正しく計算されない
```

### 2.3 現状の管理方法

```
① Excelカレンダー: 最も多い。属人化しやすい
② 手帳・ホワイトボード: 小規模事務所
③ MyKomon（マイコモン）: 税理士事務所向けSaaS
④ Chatwork/Slack: タスク管理として利用
⑤ 税務ソフトの期限管理機能: 達人シリーズ、JDL等
⑥ 所長の頭の中: 最もリスクが高い

→ どれも「漏れるリスク」がある。100社以上になると人力では限界
→ ★AI化で「漏れない仕組み」を作る
```

## 3. 期限管理の完全業務フロー（★最重要セクション）

### 3.1 税理士事務所の期限一覧

#### 月次の期限

| 期限日 | 届出・手続き | 対象 | 遅延リスク |
|---|---|---|---|
| 毎月10日 | 源泉所得税の納付 | 給与・報酬から源泉徴収した税金 | 不納付加算税10%+延滞税 |
| 毎月10日 | 住民税特別徴収の納付 | 従業員の住民税 | 延滞金 |
| 毎月末日 | 社会保険料の納付 | 会社負担+本人負担 | 延滞金+差押え |
| 7/10, 1/20 | 源泉所得税の納期特例 | 従業員10名以下の会社 | 同上 |

```
■ 源泉所得税の詳細:
  会社が従業員の給与から天引きした所得税を国に納付する
  → 翌月10日が期限（例: 3月分給与の源泉税 → 4月10日）
  → 10名以下の会社は半年に1回（7/10と1/20）にまとめて納付可能

  AI化ポイント: 顧問先ごとに「毎月」or「半年」を管理し、期限前にアラート
```

#### 決算・申告の期限

```
■ 法人税・消費税・住民税・事業税の申告

  原則: 事業年度終了日の翌日から2ヶ月以内

  例（3月決算法人）:
    事業年度: 4/1 - 3/31
    申告期限: 5/31
    ※ 5/31が土日の場合 → 翌月曜日

  例（12月決算法人）:
    事業年度: 1/1 - 12/31
    申告期限: 2/28（or 2/29）

  延長の特例:
    定款で「定時株主総会を3ヶ月以内に開催」→ 申告期限を1ヶ月延長可
    → 3月決算の場合: 5/31 → 6/30
    ※ ただし延滞税は5/31から発生する（利子税として）
    ※ 届出が必要（「申告期限の延長の特例の申請書」）

■ 決算から申告までのワークフロー:

  決算日
    ↓（1-2週間）
  決算整理（減価償却、引当金、棚卸等）
    ↓（1-2週間）
  試算表（決算書ドラフト）作成
    ↓（1-2週間）
  申告書作成
    ↓（数日）
  所長チェック
    ↓
  電子申告（e-Tax / eLTAX）
    ↓
  納付（ダイレクト納付 / 振替納税 / 窓口）

  → 100社抱えると、毎月8-10社の決算が回ってくる
  → スケジュール管理が命
```

#### 年次の期限

| 期限 | 届出・手続き | 内容 |
|---|---|---|
| 1/31 | 法定調書合計表 | 給与・報酬の支払調書を税務署に提出 |
| 1/31 | 給与支払報告書 | 従業員の住所地の市区町村に提出 |
| 1/31 | 償却資産申告書 | 固定資産税の申告 |
| 3/15 | 所得税確定申告 | 個人事業主・不動産所得等 |
| 3/15 | 消費税確定申告（個人） | 個人事業主の消費税 |
| 12月末 | 年末調整 | 従業員の所得税の精算 |

```
■ 確定申告シーズンの実態（2/16-3/15）:
  税理士事務所にとって最大の繁忙期
  - 個人の確定申告: 顧問先の個人事業主+不動産オーナー
  - 同時に法人の決算も止まらない
  - 全職員が残業。土日出勤も普通
  - 所長がチェックのボトルネック

  AI化で最も価値があるのは:
  → 期限の可視化（何件が残っているか一目でわかる）
  → 作業の優先順位付け（期限が近い順にソート）
  → 進捗管理（誰がどこまでやっているか）
```

#### スポット業務の期限

```
■ 会社設立時:
  法人設立届出書: 設立日から2ヶ月以内（税務署）
  青色申告承認申請書: 設立日から3ヶ月以内 or 最初の事業年度末
  給与支払事務所等の開設届出書: 開設日から1ヶ月以内
  源泉所得税の納期特例の申請書: 随時（効果は翌月から）

  ※ 青色申告の申請を忘れると、初年度から白色申告になる
  → 欠損金の繰越控除（10年）が使えない = 大きな不利益

■ 届出書の主要なもの:
  消費税の課税事業者届出書: 基準期間の課税売上高が1,000万円超
  簡易課税の届出書: 適用を受けたい事業年度開始日の前日まで
  役員報酬の変更: 定時株主総会から1ヶ月以内（損金算入の要件）
```

### 3.2 社労士事務所の期限一覧

#### 月次の期限

| 期限 | 届出・手続き | 内容 |
|---|---|---|
| 入社日から5日以内 | 資格取得届 | 社会保険の加入届 |
| 退社日から5日以内 | 資格喪失届 | 社会保険の脱退届 |
| 入社翌月10日 | 雇用保険 資格取得届 | ハローワークに届出 |
| 退社翌日から10日以内 | 雇用保険 資格喪失届 | 離職票の発行 |
| 毎月の給与計算 | 給与計算・明細発行 | 締日→計算→支給日 |
| 変更から5日以内 | 氏名・住所変更届 | マイナンバーで省略可の場合あり |

```
■ 入退社手続きの詳細:
  入社時に必要な届出（社保加入の場合）:
    ① 健康保険・厚生年金 資格取得届（年金事務所/健保組合）
    ② 雇用保険 資格取得届（ハローワーク）
    ③ 所得税 扶養控除等申告書（社内保管）
    ④ 住民税 特別徴収への切替届（市区町村）

  退社時:
    ① 健康保険・厚生年金 資格喪失届
    ② 雇用保険 資格喪失届+離職証明書
    ③ 住民税 普通徴収への切替届
    ④ 源泉徴収票の発行

  → 毎月数件〜数十件発生（顧問先の規模による）
  → 期限が「5日以内」「10日以内」と短い → 漏れやすい
```

#### 年次の期限

| 期限 | 届出・手続き | 内容 |
|---|---|---|
| 7/1-7/10 | 算定基礎届 | 4-6月の給与を基に社保料を決定 |
| 6/1-7/10 | 年度更新 | 労働保険料の申告・納付 |
| 賞与支払日+5日 | 賞与支払届 | 賞与の社保料計算用 |
| 月額変更後速やかに | 月額変更届 | 固定給が大幅に変わった場合 |
| 36協定の期間満了前 | 36協定の更新届出 | 時間外労働の上限を定める |

```
■ 算定基礎届（★社労士の最繁忙期）:
  毎年7月1日〜10日に提出
  4・5・6月の給与を基に、9月以降の社会保険料を決定

  作業フロー:
    4-6月の給与データ収集（全顧問先）
    → 報酬月額の算出
    → 算定基礎届の作成（電子申請 or 紙）
    → 提出（7/10期限）

  → 顧問先80社の全従業員分 = 数千件の届出
  → 期限が7/10で固定。延長不可
  → 年度更新（労働保険）も同時期 → ★最繁忙

■ 年度更新:
  前年度の確定保険料の申告 + 今年度の概算保険料の申告
  → 6/1〜7/10の期間に提出・納付
```

### 3.3 弁護士事務所の期限一覧

#### 訴訟関連の期限

| 期限 | 内容 | ミスの影響 |
|---|---|---|
| 裁判所指定日 | 口頭弁論期日 | 欠席→不利な判決 |
| 裁判所指定日 | 準備書面の提出期限 | 未提出→主張できない |
| 判決から14日 | 控訴期限 | 徒過→判決確定（取返し不可） |
| 判決から14日 | 上告期限 | 同上 |
| 送達から30日 | 支払督促への異議申立 | 徒過→仮執行宣言 |

```
■ 控訴期限の重要性:
  判決言渡日の翌日から14日以内に控訴状を提出
  → 1日でも遅れたら判決確定。何億円の判決でも覆せない
  → 弁護士の期限管理で最もリスクが高い

■ 消滅時効の管理:
  債権の消滅時効:
    一般債権: 5年（知った時から）or 10年（権利行使可能時から）
    不法行為: 3年（知った時から）or 20年（行為時から）
    商事債権: 5年

  時効の完成猶予:
    裁判上の請求（訴訟提起）→ 時効停止
    催告（内容証明送付）→ 6ヶ月間停止（その間に訴訟提起が必要）
    協議の合意 → 1年間停止（書面による合意が必要）

  → 時効が近い債権を管理し、適切なタイミングで時効中断措置を取る
```

#### 契約・登記関連

```
不動産登記: 売買完了後速やかに（期限はないが放置はリスク）
商業登記: 変更から2週間以内（役員変更、本店移転等）
  → 登記懈怠は過料の対象（最大100万円）
契約更新: 契約書に定めた更新日前に通知
```

## 4. 判断分岐・例外パターン（★AI化に必須）

### 4.1 期限が土日祝に当たる場合

```
原則: 期限日が土日祝 → 翌営業日（翌月曜日）に繰り延べ
  ※ 国税通則法第10条

例: 法人税申告期限5/31（土曜）→ 6/2（月曜）が期限

例外:
  裁判期日は裁判所が決めるので土日にはならない
  控訴期限の14日は暦日計算（土日も含む）だが、最終日が土日祝なら翌営業日
```

### 4.2 災害等による期限延長

```
大規模災害: 国税庁長官が期限延長を告示
  → 対象地域の納税者は申告・納付期限が延長される

個別の事情: 病気、事故等 → 税務署長に申請して延長

コロナ特例: 2020-2021年に広く延長が認められた
```

### 4.3 新設法人の初年度の特殊な期限

```
設立日から:
  2ヶ月以内: 法人設立届出書、給与支払事務所等の開設届出書
  3ヶ月以内 or 初年度末のいずれか早い日: 青色申告承認申請書
  1ヶ月以内: 源泉所得税の納期特例の申請書

初年度の決算:
  設立日〜初年度末（決算月末日）→ 翌日から2ヶ月以内に申告
  ※ 設立日が期中の場合、初年度は1年未満になる
```

### 4.4 顧問先の決算期変更

```
顧問先が決算期を変更した場合:
  旧決算期末日までの期間で一旦決算・申告
  → 新決算期の開始
  → 期限管理台帳の全面更新が必要

例: 3月決算 → 12月決算に変更
  4/1-12/31の9ヶ月で一旦決算・申告（2/28期限）
  以降は1/1-12/31の12ヶ月決算
```

### 4.5 複数の届出が連動するケース

```
役員報酬の変更:
  ① 株主総会議事録の作成
  ② 役員報酬の変更（定時株主総会後1ヶ月以内）
  ③ 社会保険の月額変更届（報酬月額が変更後）
  ④ 源泉所得税の変更

→ 1つの変更が複数の届出に波及する
→ AI化: 「変更イベント」をトリガーに関連する届出を自動リスト化
```

### 4.6 期限ミスが発生した場合のリカバリー

```
■ 税務申告の期限後申告:
  期限後でも申告は可能（無申告加算税15-20%+延滞税が発生）
  → 1ヶ月以内の自主的な期限後申告は無申告加算税なし（初回のみ）
  → 速やかに申告+納付するのが最善

■ 社保届出の遅延:
  遡って届出可能（ただし遅延理由の説明が必要）
  → 保険料の遡及徴収が発生する場合あり

■ 裁判期日の徒過:
  基本的に取り返しがつかない
  → 控訴期限の徒過 = 判決確定
  → 期日の欠席 = 不利な認定
  → 弁護士賠償保険で対応するケース
```

## 6. AI化マッピング

| 業務ステップ | AI化可否 | 自動化度 | 人間の確認 |
|---|---|---|---|
| 顧問先の決算月・届出状況の管理 | ◎ | 90% | 初期設定のみ |
| 法定期限の自動カレンダー生成 | ◎ | 95% | 不要 |
| 期限前アラート（30日前/14日前/7日前/3日前） | ◎ | 95% | 不要 |
| 作業の割り振り（担当者×期限×優先度） | ○ | 70% | 判断必要 |
| 進捗管理（未着手/作業中/チェック待ち/完了） | ◎ | 85% | 更新のみ |
| 届出書類の下書き生成 | ○ | 60% | 確認必須 |
| 電子申告の実行 | △ | 30% | 資格者の確認必須 |
| 変更イベントからの連動届出リスト生成 | ◎ | 85% | 確認 |
| 繁忙期の負荷予測 | ○ | 70% | 参考情報 |

### AI化の優先順位

```
Phase 1（即座に価値）:
  ① 顧問先×期限の自動カレンダー生成
  ② 段階的アラート（30日前/14日前/7日前/3日前/当日）
  ③ 進捗ダッシュボード（何件が残っているか一目瞭然）
  → 期限ミスの90%を防止

Phase 2（3ヶ月後）:
  ④ 変更イベント→連動届出の自動リスト化
  ⑤ 繁忙期の負荷予測+作業割り振り提案
  ⑥ 届出書類のテンプレート自動生成

Phase 3（6ヶ月後）:
  ⑦ 過去の申告データから次年度の作業量予測
  ⑧ 顧問先への自動リマインド（資料依頼等）
```
