## 支払いロジックの大前提

### 不動産管理会社の収益構造

管理会社の収益は**管理手数料（家賃の3〜5%）が安定収入の柱**。管理戸数の積み上げがそのまま経営基盤になる。逆に言えば、管理戸数が減れば直ちに経営が傾く「ストック型ビジネス」。

| 管理戸数 | 月間管理手数料収入 | 年間売上（管理手数料のみ） | 仲介・その他含む年商 |
|---|---|---|---|
| 500戸 | 150〜250万円 | 1,800〜3,000万円 | 3,000〜5,000万円 |
| 1,000戸 | 300〜500万円 | 3,600〜6,000万円 | 6,000〜1億円 |
| 3,000戸 | 900〜1,500万円 | 1〜1.8億円 | 1.5〜3億円 |
| 5,000戸 | 1,500〜2,500万円 | 1.8〜3億円 | 3〜5億円 |

※ 平均家賃6〜10万円、管理手数料率3〜5%で算出

### 社長・管理者の時間単価 = 8,000〜15,000円/時間

不動産管理会社の社長は**「営業（オーナー獲得）」と「トラブル対応」に自ら走る**ケースが大半。管理戸数1,000戸クラスで事務スタッフ3〜5名、営業2〜3名の体制が典型。社長が事務作業に月40〜80時間取られている=オーナー営業・管理受託拡大に充てられない。

## 1. 家賃管理・督促

### 支払いロジック（月3〜7万円）

```
■ 直接効果①：滞納損失の回避
  管理戸数1,000戸 × 平均家賃7万円 × 滞納率5% = 月350万円が滞延
  AI督促で回収率5%改善 = 月17.5万円の回収増 = 年210万円
  管理手数料分（5%）= 年10.5万円の自社売上回復

■ 直接効果②：入金消込の自動化
  1,000戸の消込 = 月40〜60時間（事務スタッフ2名分の半月）
  → 自動消込で80%削減 = 月32〜48時間削減
  → 時給1,500円 × 40時間 = 月6万円の人件費削減

■ 直接効果③：長期滞納の早期発見・対処
  3ヶ月以上の長期滞納 = 回収困難率急上昇（回収率50%以下）
  1件あたりの平均損失 = 家賃3〜6ヶ月分 + 法的費用 = 35〜60万円
  年3件の長期滞納を早期対処で回避 = 年105〜180万円

■ 間接効果：社長の督促電話からの解放
  社長が自ら督促に月5〜10時間 × 時給12,000円 = 月6〜12万円相当

■ 合計効果：年240〜420万円
  → BPO月5万円（年60万円）で ROI 4.0〜7.0倍
  ※ 控えめに見てもROI 4.0倍
```

**なぜ払うか:** 管理会社の社長にとって滞納督促は「最もやりたくない仕事」。精神的に消耗するうえ、対応が遅れると損失が雪だるま式に膨らむ。しかも入金消込という「何の創造性もない作業」に毎月事務スタッフが丸2日費やしている。月5万円で消込が自動化され、督促が仕組み化されるなら、精神的な解放だけでも十分に払う価値がある。「入居者に嫌われる電話」をAIが代わってくれるのは、金額以上の価値。

### 滞納が発生する主な原因トップ5

1. **収入減少・失業**（最多。特にコロナ以降増加傾向）
2. **口座残高不足（うっかり忘れ）**（口座振替日の管理不足）
3. **家賃支払い優先度の低下**（他の支払いを優先）
4. **入居者との連絡が取れない**（電話に出ない、引っ越し済み）
5. **家賃の振込先・金額の誤り**（名義違い、端数ミス）

### 業務フロー

```
【月初（1〜3日）】入金消込
  口座データ（CSV/API）取得 → 管理台帳の請求データと照合
  ├─ 口座名義のゆらぎ対応（姓名逆、旧姓、カタカナ/漢字違い等）
  ├─ 金額不一致の処理（端数、日割、管理費込み/別等）
  └─ 未入金リスト作成
                    ↓
【月初（3〜5日）】初回督促
  未入金者へSMS/LINE送信（支払い忘れのリマインド）
  → 3日以内に反応なし → 電話督促（1件5〜15分）
                    ↓
【月中（10〜15日）】二次督促
  電話未応答者 → 書面（督促状）郵送
  → 保証会社利用物件 → 保証会社に代位弁済請求
  → 保証会社未利用物件 → 内容証明検討
                    ↓
【月末】滞納状況集計
  ├─ 1ヶ月滞納: 電話+書面の継続督促
  ├─ 2ヶ月滞納: 連帯保証人への連絡
  ├─ 3ヶ月以上: 法的措置検討（弁護士連携）
  └─ オーナーへの滞納状況報告
                    ↓
【随時】保証会社との連携
  代位弁済請求 → 入金確認 → 求償権行使状況の追跡
  ※保証会社経由の入金も消込対象
```

## 2. 入居者対応・クレーム管理

### 支払いロジック（月3〜7万円）

```
■ 直接効果①：退去防止
  クレーム未対応/対応遅延による退去 = 年間の最大損失要因
  退去1件のコスト:
    空室期間2ヶ月 × 家賃7万円 = 14万円
    + 原状回復費用（オーナー負担分） = 5〜15万円
    + 仲介手数料（新規募集） = 家賃1ヶ月分 = 7万円
    + 広告料（AD） = 家賃1〜2ヶ月分 = 7〜14万円
    + 管理手数料の逸失（空室期間）= 0.7万円
  → 退去1件あたり = 34〜51万円の損失
  → AI対応で年3件の不満退去を防止 = 年102〜153万円

■ 直接効果②：対応時間の削減
  1,000戸で月間クレーム・問合せ = 50〜100件
  1件あたり対応時間 = 30分〜2時間（平均45分）
  月間合計 = 37〜75時間
  → AIで一次対応を自動化し30%削減 = 月11〜22時間削減 = 月1.7〜3.3万円

■ 直接効果③：夜間・休日対応の外部化
  緊急対応（水漏れ・鍵紛失等） = 月5〜10件
  夜間・休日の社長への電話 = 精神的負荷（プライスレス）
  夜間対応代行会社 = 月3〜5万円が相場
  → AIチャットボットで一次受付+緊急度判定 = 代行会社コスト削減

■ 間接効果：入居者満足度向上 → 口コミ改善 → 空室率低下
  Googleマップの口コミ評価 = 入居検討者の判断材料
  対応品質の向上 → 入居者の紹介増加

■ 合計効果：年210〜360万円
  → BPO月5万円（年60万円）で ROI 3.5〜6.0倍
  ※ 控えめに見てもROI 3.5倍
```

**なぜ払うか:** 不動産管理会社の社長が最も疲弊するのが「入居者からの電話」。夜中の水漏れ、隣人の騒音苦情、ゴミ出しトラブル——24時間365日の対応が暗黙の了解になっている。しかも対応が遅れれば退去、退去すればオーナーから「なぜ防げなかった」と詰められる。月5万円で「夜中に電話が鳴らなくなる」なら、社長の健康と家庭を守る投資として即決される。

### クレーム・問合せの内訳（管理戸数1,000戸の場合）

| カテゴリ | 月間件数 | 緊急度 | 対応時間/件 |
|---|---|---|---|

■ 直接効果②：募集資料作成の効率化
  マイソク（募集図面）作成 = 1件30分〜1時間

  消防設備点検（年2回義務）、防火管理者届出

## セキュリティ・コンプライアンス（不動産管理業固有）

### 個人情報の取り扱い

不動産管理で扱う個人情報は多岐にわたり、漏洩時のリスクが極めて高い。

| 情報カテゴリ | 具体的な情報 | リスクレベル |
|---|---|---|
| 入居者情報 | 氏名、住所、電話番号、勤務先、年収、口座情報 | 高 |
| オーナー情報 | 氏名、住所、口座情報、物件情報、収支情報 | 高 |
| 連帯保証人情報 | 氏名、住所、勤務先、年収 | 高 |
| 家賃滞納情報 | 滞納履歴、督促記録 | 極高（信用情報） |
| 入居審査情報 | 審査結果、否決理由 | 極高 |
| 鍵情報 | 暗証番号、キーボックスの場所 | 極高（物理セキュリティ） |

### 宅建業法上の義務

- **従業者名簿の備付け**（宅建業法48条3項）: 従業者の情報を記載し事務所に備え置く
- **帳簿の備付け**（宅建業法49条）: 取引記録を5年間保存（業者間は10年）
- **守秘義務**（宅建業法75条の3）: 業務上知り得た秘密を正当な理由なく漏らしてはならない
- **標識の掲示**（宅建業法50条）: 事務所に報酬額表・免許証・標識を掲示

### 反社会的勢力チェック

- 入居審査時に反社チェックが求められるケースが増加（暴力団排除条例）
- 賃貸借契約書に暴力団排除条項を入れることが標準化

### シャチョツーでの対応

- `tenants`（入居者）テーブル、`owners`（オーナー）テーブルは暗号化対象
- 鍵情報（暗証番号等）は特に厳格な暗号化 + アクセス制限
- 全アクセスを `audit_logs` に記録
- 入居者情報のエクスポートには管理者承認が必須
- 家賃滞納情報の閲覧ログは特別に記録（信用情報保護）
- 同意取得フロー（`consent_records` テーブル連携）
- 追加オプション: 不動産セキュリティパック ¥30,000/月（宅建業法完全準拠 + 鍵情報暗号化 + 反社チェック連携）

### 個人情報保護法対応

- 利用目的の特定・通知（入居申込時に明示）
- 第三者提供の制限（保証会社・仲介会社への情報共有は同意必須）
- 開示・訂正・削除請求への対応体制
- 漏洩時の報告義務（個人情報保護委員会への報告 + 本人通知）
- 2022年改正対応: 漏洩報告の義務化、個人の権利拡充、越境移転規制

### シャチョツーの差別化ポイント

1. **ツールではなくエージェント**: 既存サービスは「管理ソフト」。シャチョツーは「AI管理マネージャー」として判断・提案・実行まで行う
2. **業務横断の一気通貫**: 家賃管理→督促→修繕→オーナー報告が一つのシステムで連携。データの二重入力不要
3. **能動提案**: 「給湯器が10年超の物件が15件あります。冬前に点検を推奨」「空室3ヶ月の物件、賃料を3,000円下げれば成約率が2倍になります」
4. **業界ナレッジの蓄積**: 使うほど精度が上がる「育つDB」。滞納予測、空室期間予測、修繕費予測がデータ蓄積で向上
5. **価格帯**: PM代行の1/3〜1/5。管理ソフトの2〜5倍だが、カバー範囲とAI判断で差別化
