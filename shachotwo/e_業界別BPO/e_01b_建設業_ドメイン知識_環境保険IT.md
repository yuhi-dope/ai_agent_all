## セクション13: 環境・廃棄物管理

### 13.1 建設リサイクル法

#### 制度概要

建設工事に係る資材の再資源化等に関する法律（建設リサイクル法、2000年制定・2002年完全施行）。特定建設資材（コンクリート、アスファルト・コンクリート、木材）を対象に、分別解体と再資源化を義務付ける。

#### 対象工事の規模基準（法第9条）

| 工事種類 | 規模基準 |
|---|---|
| 建築物の解体工事 | 床面積 **80m2以上** |
| 建築物の新築・増築工事 | 床面積 **500m2以上** |
| 建築物の修繕・模様替（リフォーム等） | 請負代金 **1億円以上** |
| 建築物以外の工作物（土木工事等） | 請負代金 **500万円以上** |

#### 届出義務

- 対象建設工事の発注者は、工事着手の **7日前まで** に都道府県知事に届出
- 届出内容: 分別解体等の計画、工事概要、元請業者名等
- **届出を怠った場合**: 20万円以下の罰金

#### リサイクル率の目標と実績

| 品目 | 2018年度実績 | 2024年度目標 |
|---|---|---|
| コンクリート塊 | **99.3%** | 99%以上維持 |
| アスファルト・コンクリート塊 | **99.5%** | 99%以上維持 |
| 建設発生木材 | **96.2%** | 97%以上 |
| 建設汚泥 | **94.6%** | 95%以上 |
| 建設混合廃棄物 | **63.2%** | 60%以上 |
| **全体（再資源化・縮減率）** | **97.2%** | — |

> **出典**: 国土交通省「建設副産物実態調査」（2018年度）。2024年度調査は実施中、結果未公表（2026年4月時点）。

#### 建設リサイクル推進計画2020の重点施策

1. 建設混合廃棄物の排出量削減（2024年度目標: 200万トン以下）
2. 建設発生土の有効利用率向上
3. 建設廃棄物の最終処分量を1995年度比で**96%削減**

#### 中小建設会社特有の課題

- **分別の人手不足**: 現場作業員が分別を面倒がり、混合廃棄物として排出しがち → 処分費が割高（混合廃棄物は分別品の2-3倍の単価）
- **元請の圧力**: 元請から「処分費を下げろ」と言われるが、分別には手間がかかる
- **不法投棄リスク**: 処分費を削るために無許可業者に出す零細業者が後を絶たない → 排出者責任で元請・下請とも処罰対象
- **マニフェスト管理**: 紙マニフェストの保管（5年間）が事務負担

### 13.2 産業廃棄物処理（マニフェスト制度）

#### マニフェスト制度の基本

産業廃棄物管理票（マニフェスト）は、排出事業者が産業廃棄物の処理を委託する際に、廃棄物の種類・数量・運搬業者・処分業者等を記載して交付する管理票。廃棄物処理法（廃掃法）第12条の3に基づく。

#### 紙マニフェストの7枚複写

| 伝票 | 保管者 | 返送タイミング |
|---|---|---|
| A票 | 排出事業者 | 交付時に控え |
| B1票 | 収集運搬業者 | 運搬終了後 |
| B2票 | 排出事業者へ返送 | 運搬終了後10日以内 |
| C1票 | 処分業者 | 処分終了後 |
| C2票 | 収集運搬業者へ返送 | 処分終了後10日以内 |
| D票 | 排出事業者へ返送 | 処分終了後10日以内 |
| E票 | 排出事業者へ返送 | 最終処分終了後10日以内 |

- **保管義務**: 5年間
- **報告義務**: 毎年6月30日までに前年度の産業廃棄物管理票に関する報告書を都道府県知事に提出

#### 電子マニフェスト（JWNET）

- **運営**: 公益財団法人 日本産業廃棄物処理振興センター
- **義務化対象**（2020年4月~）: 前々年度の特別管理産業廃棄物の発生量が **年間50トン以上** の事業場（PCB廃棄物を除く）
- **料金**: 排出事業者 基本料 26,400円/年 + 登録料 1件あたり22円（税込参考）
- **メリット**: 紙の保管不要、返送確認の自動化、報告書作成の省力化

#### 2025-2027年の制度改正

- **2026年1月施行**: 廃棄物処理委託契約書の法定記載事項追加（処分方法の具体化等）
- **2027年4月施行**: 電子マニフェストの処分完了報告時の報告項目追加

#### 収集運搬許可・処分業許可

| 許可種類 | 概要 | 中小建設会社との関係 |
|---|---|---|
| 収集運搬許可 | 産廃を運搬する業者が取得 | 自社施工で出た廃材を自分で運ぶ場合は不要（自社運搬）。ただし他社の廃棄物を運搬する場合は必要 |
| 処分業許可 | 産廃を中間処理・最終処分する業者が取得 | 建設会社は通常不要だが、自社で破砕プラント等を持つ場合は必要 |

#### 中小建設会社の実務フロー

```
1. 委託契約書の締結（収集運搬業者・処分業者と書面契約、5年保管）
2. マニフェスト交付（廃棄物引渡し時に7枚複写を交付 or 電子マニフェスト登録）
3. 返送票の確認（B2票: 運搬完了、D票: 処分完了、E票: 最終処分完了）
4. 90日ルール（交付日から90日以内にD票が返送されない場合、都道府県に報告）
5. 180日ルール（交付日から180日以内にE票が返送されない場合、都道府県に報告）
6. 年次報告（毎年6月30日まで）
```

#### 中小企業特有の課題

- **契約書の更新漏れ**: 5年ごとの更新を忘れがち → 無契約での委託は廃掃法違反
- **マニフェストの交付漏れ**: 少量の廃棄物（端材等）で交付を省略しがち
- **処分場の料金交渉力がない**: 大手と比べて処分量が少なく、単価が割高
- **混合廃棄物の安易な排出**: 分別の手間を惜しんで混合廃棄物として出す → 処分単価3,000-8,000円/m3が混合だと10,000-20,000円/m3以上

### 13.3 廃棄物の種類と処分単価（建設系）

| 廃棄物種類 | 品目例 | 処分単価目安（関東圏） | リサイクル方法 |
|---|---|---|---|
| コンクリート塊 | 解体コンクリート、ブロック | 1,000-3,000円/t | 破砕→再生砕石（路盤材） |
| アスファルト塊 | 道路舗装撤去材 | 1,000-2,500円/t | 加熱再生→再生アスファルト合材 |
| 建設汚泥 | シールド掘削土、杭掘削泥水 | 3,000-8,000円/t | 脱水→改良土、セメント固化 |
| 木くず | 型枠材、伐採木、内装木材 | 2,000-5,000円/t | チップ化→燃料、堆肥 |
| 廃プラスチック | 塩ビ管、防水シート、養生材 | 5,000-15,000円/t | マテリアルリサイクル or 焼却 |
| 混合廃棄物 | 上記が混合したもの | **10,000-25,000円/t** | 手選別→各品目に分類後処理 |
| 石膏ボード | 内装解体材 | 8,000-15,000円/t | リサイクルボード原料 |
| ガラス・陶磁器くず | 窓ガラス、タイル | 3,000-8,000円/t | 路盤材、タイル原料 |
| 金属くず | 鉄筋、配管、アルミ | **有価物（売却益あり）** | スクラップ業者が買取 |
| 繊維くず | 畳、カーペット | 5,000-10,000円/t | 焼却 |

> **ポイント**: 分別を徹底すれば処分費を40-60%削減可能。特に金属くずは有価物として売却益が出る。

### 13.4 アスベスト対策

#### 大気汚染防止法の段階的改正

| 施行日 | 改正内容 |
|---|---|
| **2021年4月** | レベル3建材（成形板等）も規制対象に追加。全石綿含有建材が規制対象に |
| **2022年4月** | 事前調査結果の報告義務化（解体80m2以上 or リフォーム100万円以上） |
| **2023年10月** | 事前調査の有資格者化。建築物石綿含有建材調査者による調査が必須 |

#### アスベスト含有建材のレベル分類

| レベル | 建材種類 | 飛散性 | 除去費用目安（m2あたり） | 主な場所 |
|---|---|---|---|---|
| **レベル1** | 吹付け石綿、石綿含有吹付けロックウール | **著しく高い** | 15,000-85,000円/m2 | 機械室天井、鉄骨耐火被覆、ボイラー室 |
| **レベル2** | 石綿含有保温材、断熱材、耐火被覆材 | **高い** | 10,000-50,000円/m2 | 配管保温材、煙突断熱材 |
| **レベル3** | 石綿含有成形板（スレート、Pタイル等） | **比較的低い** | 3,000-10,000円/m2 | 屋根材、壁材、床タイル、天井材 |

#### 事前調査の義務化（2022年4月~）

- **報告対象工事**: 解体部分の延床面積80m2以上 or 請負金額100万円以上（税込）
- **報告先**: 労働基準監督署 + 都道府県等（大気汚染防止法）
- **報告方法**: 石綿事前調査結果報告システム（電子申請）
- **2023年10月~**: 調査実施者は以下の有資格者に限定
  - 一般建築物石綿含有建材調査者
  - 特定建築物石綿含有建材調査者
  - 一戸建て等石綿含有建材調査者（戸建住宅のみ）

#### アスベスト除去の手順（レベル1の場合）

```
1. 事前調査（有資格者による目視・設計図書確認 → 必要に応じ分析調査）
2. 事前調査結果の報告（電子申請）
3. 作業計画の策定（石綿障害予防規則第4条）
4. 労基署への計画届出（作業開始14日前まで）
5. 都道府県等への届出（作業開始14日前まで）
6. 養生・隔離（負圧除塵装置の設置、セキュリティゾーンの設置）
7. 湿潤化・除去作業（作業者は全面型呼吸用保護具+使い捨て保護衣）
8. 粉塵濃度測定（作業中・完了後）
9. 除去後の清掃・確認（目視確認）
10. 廃石綿の搬出（特別管理産業廃棄物として処理、二重梱包+表示）
11. 記録の保存（40年間）
```

#### 中小建設会社の実務的影響

- **解体工事では必ず事前調査が必要**: 2006年以前の建物は高確率で石綿含有建材が使用されている
- **調査費用**: 1棟あたり10-50万円（規模・調査箇所数による）
- **調査者不足**: 有資格者の需要急増で、調査の予約が1-2ヶ月待ちのケースも
- **費用の転嫁**: 施主・元請に調査費用を見積りに入れる必要があるが、理解を得にくい
- **みなし判定**: 分析調査を行わず「石綿含有あり」とみなして除去工事を行う選択肢もある（分析費用vs除去費用のバランス）

### 13.5 騒音・振動規制

#### 特定建設作業（騒音規制法・振動規制法）

**騒音規制法の特定建設作業（8種類）**

1. くい打機（もんけんを除く）、くい抜機、くい打くい抜機の使用
2. びょう打機の使用
3. さく岩機の使用
4. 空気圧縮機（電動機以外の原動機を用い、定格出力15kW以上）の使用
5. コンクリートプラント（混練容量0.45m3以上）又はアスファルトプラント（混練重量200kg以上）の設置
6. バックホウ（定格出力80kW以上）の使用
7. トラクターショベル（定格出力70kW以上）の使用
8. ブルドーザー（定格出力40kW以上）の使用

> **注意**: 6-8は低騒音型建設機械として指定を受けた機械を使用する場合は特定建設作業に該当しない。

**振動規制法の特定建設作業（4種類）**

1. くい打機（もんけん・圧入式を除く）、くい抜機、くい打くい抜機の使用
2. 鋼球の使用
3. 舗装版破砕機の使用
4. ブレーカー（手持ち式を除く）の使用

#### 規制基準

| 規制項目 | 騒音 | 振動 |
|---|---|---|
| **基準値** | 敷地境界で **85dB以下** | 敷地境界で **75dB以下** |
| **作業禁止時間（第1号地域）** | 19:00-翌7:00 | 19:00-翌7:00 |
| **作業禁止時間（第2号地域）** | 22:00-翌6:00 | 22:00-翌6:00 |
| **1日の作業時間上限（第1号）** | 10時間以内 | 10時間以内 |
| **1日の作業時間上限（第2号）** | 14時間以内 | 14時間以内 |
| **連続作業日数** | 6日以内 | 6日以内 |
| **日曜・祝日** | 作業禁止 | 作業禁止 |

#### 届出

- 特定建設作業を行う場合、**作業開始7日前まで**に市区町村に届出
- **届出を怠った場合**: 10万円以下の罰金

#### 中小建設会社の実務ポイント

- **低騒音型建設機械の使用推奨**: 住宅密集地では事実上必須。リース機でも型式を確認すること
- **近隣への事前挨拶**: 法的義務ではないが、苦情防止のためほぼ必須。工程表の掲示も重要
- **苦情対応**: 規制基準内でも苦情は来る。測定記録（騒音計データ）を残しておくことが重要
- **条例による上乗せ規制**: 東京都、大阪府等では騒音規制法より厳しい基準を条例で設定

### 13.6 建設発生土の処理

#### 建設発生土と産業廃棄物の区分

- **建設発生土**（残土）: 廃棄物処理法上の「廃棄物」ではない。土砂、砂利等
- **建設汚泥**: 産業廃棄物（無機性汚泥）。コーン指数200kN/m2以下の泥状のもの

> **判定基準**: 標準仕様ダンプトラックに山積みできず、その上を人が歩けないような流動性を呈する状態のものが「汚泥」。そうでなければ「建設発生土」。

#### 土量配分計画

公共工事では「土量配分計画」の作成が求められる。発生土の搬出先を事前に特定し、有効利用を最大化する計画。

```
【計画の要素】
1. 発生土量の算定（切土量・掘削量）
2. 発生土の品質分類（第1種~第4種、泥土）
3. 利用可能な受入先の特定（近隣工事、ストックヤード、残土処分場）
4. 運搬計画（距離・コスト・環境負荷）
5. 残土処分場の確保（自治体の残土条例に基づく許可施設）
```

#### 発生土の品質区分

| 区分 | コーン指数 | 用途 |
|---|---|---|
| 第1種建設発生土 | 800kN/m2以上 | 道路路体、河川堤防 |
| 第2種建設発生土 | 400kN/m2以上 | 道路路体、宅地造成 |
| 第3種建設発生土 | 200kN/m2以上 | 水面埋立、土地造成 |
| 第4種建設発生土 | 200kN/m2未満 | 水面埋立（条件付き） |
| 泥土 | 流動性あり | 処理（脱水・固化）後利用 |

#### 中小建設会社の課題

- **残土処分場の不足**: 首都圏では残土処分場が減少し、処分費が高騰（1m3あたり3,000-10,000円、地域差大）
- **残土条例**: 自治体ごとに異なる残土条例があり、手続きが煩雑
- **不法投棄**: 処分費削減のため山間部等に不法に捨てる事例が社会問題化（2021年 熱海市土石流災害の一因）
- **盛土規制法**（2023年5月施行）: 熱海市土石流災害を契機に制定。盛土等の安全管理を強化。罰則も大幅に強化（最大3年以下の懲役又は1,000万円以下の罰金、法人は3億円以下）

### 13.7 環境配慮型施工

#### 低騒音型建設機械指定制度

- **根拠**: 国土交通省「低騒音型・低振動型建設機械の指定に関する規程」（1997年制定）
- **指定機種**: バックホウ、ブルドーザー、トラクターショベル、発動発電機、空気圧縮機等
- **効果**: 指定機械を使用する場合、騒音規制法の特定建設作業（6-8号）に該当しない
- **公共工事での使用推奨**: 国交省直轄工事では低騒音型・排出ガス対策型の使用が原則

#### 排出ガス対策型建設機械

- 国交省が指定する排出ガス基準を満たした建設機械
- 2006年以降、公共工事では排出ガス対策型の使用が原則
- **第3次基準値（Tier3）以上**が現在の標準

#### グリーン調達

- 国等による環境物品等の調達の推進等に関する法律（グリーン購入法）に基づく
- 公共工事では、環境に配慮した資材・建設機械の使用が推奨
- **対象品目例**: 再生砕石、再生アスファルト、高炉セメント、低VOC塗料、低騒音型建設機械

### 13.8 フロン回収義務

#### フロン排出抑制法と建設業の関係

建物の解体工事において、業務用エアコン・冷凍冷蔵機器（第一種特定製品）のフロン類を回収する義務がある。

#### 特定解体工事元請業者の義務

1. **事前確認**: 解体工事前に第一種特定製品の有無を確認
2. **書面説明**: 発注者に対し、フロン類の回収が必要な機器の有無を書面で説明
3. **書面の保存**: 3年間
4. **フロン回収の手配**: 第一種フロン類充填回収業者（都道府県登録業者）に依頼

#### 罰則

- 機器のフロン類を回収せずに廃棄した場合: **1年以下の懲役又は50万円以下の罰金**
- 行程管理票の未記載・虚偽記載: 30万円以下の罰金

#### 中小建設会社の実務ポイント

- 解体工事を受注した場合、まずエアコン室外機等の有無を確認
- 家庭用エアコンは家電リサイクル法の対象（フロン排出抑制法ではない）
- 業務用エアコン（天井カセット型、ビル用マルチ等）がフロン排出抑制法の対象
- 回収費用: 機器1台あたり3,000-30,000円（冷媒量・機器種類による）

---

### 13.9 土壌汚染対策法

#### 建設業との関係

- 一定規模以上の土地の形質変更（掘削工事等）を行う際に届出が必要
- 調査の結果、汚染が判明した場合は区域指定と対策措置が必要

#### 届出基準

| 区分 | 届出基準 |
|---|---|
| 一般の土地 | **3,000m2以上**の形質変更 |
| 有害物質使用特定施設の設置地 | **900m2以上**の形質変更 |
| 届出時期 | 形質変更着手の**30日前まで** |

#### 区域指定

| 区域 | 内容 | 制限 |
|---|---|---|
| 要措置区域 | 健康被害のおそれがある | 汚染除去等の措置が必要。形質変更は原則禁止 |
| 形質変更時要届出区域 | 健康被害のおそれはないが汚染あり | 形質変更時に届出が必要 |

#### 2025年改正の検討方向（中央環境審議会）

- **自然由来汚染土壌の規制緩和**: 従来は自然由来でも区域指定されていたが、搬出時にのみ基準適合性を確認する仕組みへの変更を検討
- **建設発生土の扱い**: 自然由来汚染土壌を含む建設発生土の合理的な管理方法の検討

#### 中小建設会社の実務ポイント

- 工場跡地やガソリンスタンド跡地での工事では要注意
- 掘削中に異臭・変色土壌を発見した場合は直ちに作業中止し、発注者・監理者に報告
- 汚染土壌の処理費用は極めて高額（通常の残土処分費の10-50倍）→ 施工前に土壌調査を推奨

## セクション14: 保険・保証・リスク管理

### 14.1 建設工事保険

#### 保険の種類と概要

| 保険種類 | 対象工事 | 補償内容 |
|---|---|---|
| **建設工事保険** | 建築工事（ビル、住宅、工場等） | 工事目的物・資材の損害 |
| **土木工事保険** | 土木工事（道路、橋梁、トンネル等） | 工事目的物・資材の損害 |
| **組立保険** | 機械・設備の据付・組立工事 | 組立対象物・資材の損害 |

#### 補償される損害

- **火災、落雷、爆発**
- **台風、暴風雨、洪水**（基本補償に含まれる）
- **盗難**
- **施工ミス、作業ミス**
- **第三者の行為**
- **地盤沈下、崖崩れ**

#### 補償されない損害（基本契約）

- **地震、噴火、津波** → 地震危険担保特約で対応可能
- **故意、重大な過失**
- **自然消耗、経年劣化**
- **設計の欠陥そのもの**（ただし設計欠陥に起因する物理的損害は担保される場合あり）
- **戦争、内乱**

#### 保険料の目安（建設工事保険）

| 工事種類 | 保険料率（対請負金額） | 保険料例（1億円工事） |
|---|---|---|
| 木造住宅新築 | 0.15-0.25% | 15-25万円 |
| RC造ビル新築 | 0.08-0.15% | 8-15万円 |
| 鉄骨造工場新築 | 0.10-0.20% | 10-20万円 |
| リフォーム工事 | 0.20-0.40% | 20-40万円 |
| 解体工事 | 0.30-0.50% | 30-50万円 |

> **注意**: 地震危険担保特約を付帯した場合、保険料は基本料の1.5-3倍程度増加。縮小割合80%、免責100万円が一般的。

---

### 14.2 賠償責任保険

#### 請負業者賠償責任保険（施工中のリスク）

- **対象**: 工事施工中に第三者の身体・財物に与えた損害
- **事例**: 
  - 足場の落下物が通行人に当たってケガ → 治療費・慰謝料を補償
  - 掘削工事で隣家の塀を損壊 → 修復費用を補償
  - 工事車両が通行人に接触 → 自動車保険の範囲外の損害を補償
- **保険料**: 年間完工高に対して0.03-0.10%程度
- **支払限度額**: 1事故あたり1億-5億円が一般的

#### 生産物賠償責任保険（PL保険、引渡し後のリスク）

- **対象**: 引渡し後の工事目的物の欠陥に起因する第三者損害
- **事例**:
  - 施工した外壁タイルが剥落して通行人を負傷させた
  - 施工した給排水管からの漏水で下階テナントの商品が損害
- **保険料**: 年間完工高に対して0.02-0.08%程度
- **注意**: 工事目的物そのものの損害（瑕疵の補修費用等）は対象外

#### 中小建設会社の実務ポイント

- **元請から保険加入を求められるケースが増加**: 下請けであっても加入を契約条件にする元請が多い
- **包括契約**: 年間の全工事をまとめて1つの保険契約にすると割安
- **免責金額**: 10-50万円の免責が一般的。小額損害は自己負担

---

### 14.3 労災保険（建設業の特殊性）

#### 有期事業の一括

建設業の労災保険は、一般の事業（継続事業）と異なり、工事ごとに保険関係が成立する「有期事業」。ただし小規模工事は年間で一括して申告する「一括有期事業」の仕組みがある。

| 区分 | 適用要件 | 申告方法 |
|---|---|---|
| **一括有期事業** | 請負金額1.8億円未満 かつ 概算保険料160万円未満 | 年度末に一括して確定申告（毎年7月10日まで） |
| **単独有期事業** | 上記以外の大規模工事 | 工事ごとに保険関係成立届 → 概算保険料申告 → 確定申告 |

#### 建設業の労災保険料率（2025年度）

| 事業の種類 | 保険料率 |
|---|---|
| 水力発電施設・ずい道新設事業 | 62/1000 |
| 道路新設事業 | 11/1000 |
| 舗装工事業 | 9/1000 |
| 鉄道又は軌道新設事業 | 9.5/1000 |
| 建築事業（既設建築物設備工事含む） | 9.5/1000 |
| 機械装置の組立て又は据付け事業 | 6.5/1000 |
| その他の建設事業 | 15/1000 |

> **注意**: 労災保険料は請負金額に労務費率を乗じた額を賃金総額とみなし、これに保険料率を乗じて算出。建設業の労務費率は事業種類ごとに21-24%。

#### メリット制

- 一括有期事業で**年間確定保険料100万円以上**（または3年間の概算保険料が年40万円以上）の場合に適用
- 収支率（保険料に対する保険給付額の割合）に応じて、保険料率が **±40%** の範囲で増減
- **結果**: 無災害を続ければ保険料が最大40%安くなる。労災が多ければ最大40%高くなる

#### 労災隠し問題

- **労災隠し**: 労災事故を労基署に報告しない（労働安全衛生法第100条違反、50万円以下の罰金）
- **背景**: メリット制により保険料が上がることを恐れる / 元請からの指名停止を恐れる / 工事の中断を避けたい
- **厚労省の対策**: 労災かくしは犯罪です（2002年~通達強化）、医療機関からの通報制度
- **中小建設会社への影響**: 下請業者で発生した労災も元請の労災保険で処理 → 元請のメリット制に影響するため、元請から「労災を使うな」という圧力が発生しやすい

#### 一人親方の特別加入

- 建設業では一人親方（個人事業主として現場作業に従事する者）が多い
- 通常の労災保険の対象外だが、特別加入制度で任意加入可能
- 給付基礎日額: 3,500-25,000円の範囲で選択
- 保険料: 給付基礎日額 × 365 × 保険料率

---

### 14.4 完成保証制度

#### 住宅完成保証制度

- **根拠**: 住宅の品質確保の促進等に関する法律（品確法）関連
- **目的**: 施工中に建設業者が倒産した場合に、住宅の完成を保証
- **運営**: 住宅保証機構（現: 住宅あんしん保証）等
- **保証方式**: 代替履行（他の建設業者が引き継いで完成）、または金銭保証（前払金の返還等）
- **保証料**: 工事請負金額の0.5-1.0%程度

#### 履行保証（公共工事）

| 保証種類 | 内容 | 保証率 |
|---|---|---|
| 入札ボンド | 落札者が正当な理由なく契約を辞退した場合の損害を保証 | 入札金額の5%程度 |
| 契約保証金 | 契約不履行時の損害を保証 | 契約金額の10% |
| 履行ボンド（金銭保証） | 受注者の債務不履行時に損害額を金銭で補償 | 契約金額の10-30% |
| 履行ボンド（役務保証） | 代替業者が残工事を完成させる | 契約金額の10-30% |

### 14.5 住宅瑕疵担保責任保険

#### 住宅瑕疵担保履行法（2009年10月施行）

- **対象**: 新築住宅を引き渡す建設業者・宅建業者
- **義務**: 「保険加入」または「保証金の供託」のいずれかの資力確保措置
- **対象部位**: 構造耐力上主要な部分 + 雨水の浸入を防止する部分
- **保証期間**: **10年間**

#### 保険と供託の比較

| 項目 | 保険（住宅瑕疵担保責任保険） | 供託（保証金の供託） |
|---|---|---|
| 保険料/供託金 | 1戸あたり5-10万円程度 | 1戸: 2,000万円（戸数に応じて増加） |
| 手続き | 保険法人に申込み+検査 | 法務局に供託 |
| 適するケース | 中小建設会社（ほとんどの場合こちら） | 大手ハウスメーカー |
| 業者が倒産した場合 | 住宅取得者が保険法人に直接請求可能 | 住宅取得者が供託金から還付 |

#### 指定保険法人（6法人）

1. 住宅あんしん保証
2. 住宅保証機構
3. 日本住宅保証検査機構（JIO）
4. ハウスプラス住宅保証
5. ハウスジーメン
6. あすかJHS

#### 年次届出

- **基準日**: 毎年3月31日
- **届出期限**: 基準日から3週間以内（4月21日まで）
- **届出先**: 許可行政庁（国土交通大臣 or 都道府県知事）
- **届出を怠った場合**: 新築住宅の請負契約ができなくなる（業務停止）

### 14.6 建設業退職金共済（建退共）

#### 制度概要

- **根拠法**: 中小企業退職金共済法
- **運営**: 独立行政法人 勤労者退職金共済機構
- **対象**: 建設業で働く現場労働者（会社を転々としても通算して退職金が貯まる）
- **掛金**: 1日あたり**320円**（共済証紙代）
- **支払方法**: 共済証紙を共済手帳に貼付 or 電子申請方式

#### 退職金の目安

| 掛金納付年数 | 証紙枚数（21日/月換算） | 退職金額 |
|---|---|---|
| 3年 | 756枚 | 約24万円 |
| 5年 | 1,260枚 | 約43万円 |
| 10年 | 2,520枚 | 約95万円 |
| 20年 | 5,040枚 | 約226万円 |
| 30年 | 7,560枚 | 約396万円 |
| 40年 | 10,080枚 | 約612万円 |

#### 事業主のメリット

- 掛金は法人税法上の**損金**（全額経費算入）
- 新規加入時に国が50日分（16,000円相当）を助成
- **経営事項審査（経審）で加点**: 建退共への加入・掛金納付実績は経審の「その他の審査項目（W）」で15点の加点
- 公共工事の入札参加資格審査で有利

#### 建退共とCCUSの連携

- 建設キャリアアップシステム（CCUS）と連携し、就業履歴から自動的に証紙貼付相当の掛金計上が可能に
- 「電子申請方式」として、証紙の物理的な購入・貼付が不要になりつつある

### 14.7 中小企業退職金共済（中退共）

#### 建退共との使い分け

| 項目 | 建退共 | 中退共 |
|---|---|---|
| 対象者 | 現場労働者（日雇・期間工含む） | 常用雇用の従業員（事務職・営業職含む） |
| 掛金 | 日額320円 | 月額5,000-30,000円（16段階） |
| 掛金負担 | 事業主（全額） | 事業主（全額） |
| 退職金の受取 | 建設業を辞めたとき | その企業を退職したとき |
| ポータビリティ | 建設業内で転職しても通算 | 企業間で引継ぎ可能（条件あり） |

- 事務員・経理担当者 → 中退共
- 現場作業員（直用） → 建退共 + 中退共の併用も可能
- 現場作業員（日雇） → 建退共のみ

### 14.8 ボンド制度（公共工事）

#### 入札ボンド

- **目的**: 落札者が正当な理由なく契約締結を拒否した場合に備える
- **発行機関**: 建設業保証会社（東日本建設業保証、西日本建設業保証、北海道建設業信用保証）、損害保険会社
- **保証額**: 入札金額の5%程度
- **手数料**: 保証額の0.1-0.3%程度
- **審査**: 金融機関等による建設業者の信用審査が行われる → 不良不適格業者の排除機能

#### 履行ボンド

- **金銭的保証**: 受注者が工事を完成できなかった場合の損害額を金銭で補填
- **役務的保証**: 保証人が代替業者を選定し、残工事を完成させる
- **保証額**: 契約金額の10-30%
- **適用**: 国交省直轄工事では2001年から導入

#### 前払金保証

- 公共工事では請負代金の40%を上限に前払いを受けることができる
- 前払金保証会社が保証することで発注者のリスクを軽減
- **保証料率**: 概ね0.05-0.15%程度

---

### 14.9 信用調査と倒産リスク

#### 建設業者の信用調査

| 調査機関 | 特徴 | 建設業向けサービス |
|---|---|---|
| 帝国データバンク（TDB） | 国内最大の企業信用調査会社。147万社のデータベース | 評点（100点満点）、業界別分析レポート |
| 東京商工リサーチ（TSR） | TDBと並ぶ大手。詳細な財務分析 | 企業情報レポート、倒産予測スコア |

#### 建設業の倒産動向（2025年上半期）

- 建設業の倒産は**4年連続増加**、2025年上半期は過去10年最多ペース
- **主な要因**: 人手不足（求人倍率8倍超）、資材価格高騰（鉄鋼・木材）、高齢化
- 人手不足を直接原因とする倒産: 建設業が全業種中最多（約3割）

#### 元請倒産時の下請の対策

1. **工事代金の早期回収**: 出来高に応じた中間払いを契約に入れる
2. **信用調査の実施**: 新規元請との取引開始前にTDB/TSRの企業情報を確認
3. **取引信用保険**: 元請の倒産による売掛金回収不能をカバーする保険
4. **動産譲渡担保**: 工事で使用する資材・機材に担保権を設定
5. **建設業法の保護規定**: 特定建設業者（元請）は下請代金を**50日以内**に支払う義務（建設業法第24条の6）
6. **2025年12月施行 改正建設業法**: 原価割れ契約の禁止、リスク情報の通知義務化、工期ダンピングの禁止

#### 天災リスクと保険カバー

| 災害 | 建設工事保険（基本） | 特約 |
|---|---|---|
| 台風・暴風雨 | **補償あり** | — |
| 洪水・高潮 | **補償あり** | — |
| 豪雪 | **補償あり** | — |
| 落雷 | **補償あり** | — |
| 地震 | **補償なし** | 地震危険担保特約（保険料1.5-3倍増、縮小割合80%、免責100万円） |
| 津波 | **補償なし** | 地震危険担保特約で対応 |
| 噴火 | **補償なし** | 地震危険担保特約で対応 |

> **不可抗力条項（民間工事）**: 民間工事（民間連合約款）では、天災等の不可抗力による損害は原則として「受注者が善良な管理者の注意をしたにもかかわらず生じた場合」は発注者負担。ただし交渉ベース。公共工事では請負金額の1%を超える部分について発注者が負担。

#### BIM/CIM義務化の動向

- **2023年度~**: 国交省直轄の全公共工事でBIM/CIM原則適用（小規模工事除く）
- **2026年春**: BIM図面審査の開始（確認申請でBIMモデルを提出する仕組み）

#### 中小建設会社のBIM導入の現実

- **導入率**: 中小建設会社のBIM導入率は依然として**10-20%程度**（推定）
- **障壁**: 
  1. ソフトウェア費用（年間30-50万円/ライセンス）
  2. 高性能PC（30-60万円）
  3. 人材育成コスト（研修+OJTで6ヶ月-1年）
  4. 既存の2D CAD（Jw_cad、AutoCAD）との併用負担
- **国交省の支援**: 「中小事業者によるBIM導入・活用に向けたステップ」を公開し、段階的な導入を推奨

#### IT導入の主な課題（中小建設会社特有）

1. **高齢化**: 現場所長・職長の平均年齢が50代後半。「スマホは使えるがアプリは使えない」層が多い
2. **ITリテラシー格差**: 事務所のPC操作は問題ないが、現場でタブレットを使いこなせない
3. **費用対効果の不透明さ**: 「便利そうだが、うちの規模で元が取れるのか」
4. **現場の反発**: 「今までのやり方で問題ない」「紙の方が早い」
5. **経営者のIT投資への消極性**: 設備投資（重機・車両）には使うが、IT投資は「コスト」と見なしがち
6. **人材不足**: IT担当者がいない。「誰が設定・管理するのか」

### 15.6 SaaS疲れ問題

#### 中小建設会社が直面するSaaS疲れの実態

**典型的なツール乱立パターン（従業員50名規模の建設会社）**

```
1. グリーンサイト（安全書類）     ← 元請A社から要求
2. Buildee（労務安全）            ← 元請B社から要求
3. ANDPAD（施工管理）             ← 元請C社から要求
4. CCUS（キャリアアップ）         ← 国交省の要請
5. 建退共電子申請                  ← 労務管理
6. JWNET（電子マニフェスト）       ← 廃棄物管理
7. freee（会計）                  ← 自社経理
8. Jw_cad（図面）                 ← 自社設計
9. LINE WORKS or Chatwork         ← 社内コミュニケーション
10. Excel（見積・工程・台帳）      ← 何にでも使う
```

#### 主な問題

| 問題 | 具体的な状況 | 影響 |
|---|---|---|
| **ID管理の煩雑さ** | 10個以上のサービスのID/パスワードを管理。パスワード忘れで業務停止 | 月に2-3回は「ログインできない」騒ぎ |
| **二重入力** | 作業員情報をグリーンサイト・Buildee・CCUSに別々に入力 | 1人あたり15-30分の重複作業。50名規模で月12-25時間のロス |
| **データの分断** | 写真はANDPAD、図面はDropbox、書類はグリーンサイト。横断検索不可 | 必要な情報を探すのに平均15分/回 |
| **元請ごとのツール強制** | 元請が違えば使うシステムも違う。A社はANDPAD、B社はSPIDERPLUS | 複数ツールの操作を覚える負担 |
| **更新費用の累積** | 各SaaSの月額料金が積み重なり、年間50-100万円に | 「便利だが高い」という不満 |
| **操作教育の負担** | 新しいツールが入るたびに研修が必要。高齢職人は覚えられない | 結局、若手社員が全部代行 |

#### API連携・データ連携の現状

- **CCUS × グリーンサイト**: 連携あり（2021年~）。技能者情報の自動連携
- **CCUS × 建退共**: 連携あり。就業履歴から掛金計上の自動化
- **ANDPAD × グリーンサイト**: 一部連携あり
- **Buildee × CCUS**: 連携あり（入退場管理と就業履歴の連携）
- **全体的には**: まだまだ「個別連携」の段階で、統合プラットフォームには程遠い

#### SaaS疲れ問題が示すシャチョツーの参入機会

中小建設会社は「もうこれ以上ツールを増やしたくない」と感じている。一方で、元請からのデジタル化要求は強まる一方。この矛盾を解決するのが、「1つのプラットフォームで全てのバックオフィス業務を統合する」シャチョツーのコンセプトに合致する。

**特に大きな課題**:
- 安全書類の作成（グリーンサイト/Buildeeへの入力代行 or API連携）
- 作業員情報の一元管理（CCUS・グリーンサイト・建退共への同期）
- 経理業務（出来高管理・原価管理・請求書発行の自動化）
- マニフェスト管理（電子マニフェストの自動登録・期限管理）
