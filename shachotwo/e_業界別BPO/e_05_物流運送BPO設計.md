## 1. 配車計画AI（★実装対象）

### 1.1 問題の本質

```
配車計画の現状:
  1. 荷主から配送依頼を受ける（電話/FAX/メール）
  2. 配車マンがドライバーの稼働状況を確認
  3. 車両の空き・積載量・配送エリアを考慮
  4. ルートを組む（経験と勘）
  5. ドライバーに指示（電話/LINE/点呼時に口頭）
  6. 当日の変更対応（キャンセル・追加・渋滞）

  → ベテラン配車マン1人に全依存。平均年齢50代
  → 配車マンの退職=会社の存続危機
  → 最適化されていない配車=燃料費・人件費の無駄
  → 空車率（荷物を積んでいない走行の割合）が全国平均40%
```

### 1.2 支払いロジック

```
■ 直接効果①：燃料費の削減
  車両50台 × 月間燃料費20万円/台 = 月1,000万円
  ルート最適化で10%削減 = 月100万円 = 年1,200万円

■ 直接効果②：空車率の改善
  空車率40%→35%に改善
  → 実車率向上分の売上増 = 月50-100万円

■ 直接効果③：配車マンの工数削減
  配車計画: 毎朝2-3時間 × 配車マン2名 = 月120-180時間
  → AI支援で50%削減 = 月60-90時間削減
  → 時給2,000円 × 75時間 = 月15万円

■ 直接効果④：2024年問題対応
  残業上限960h/年のシミュレーション
  → 違反=車両停止命令。事業継続不能リスクの回避

■ 合計効果：年1,400-2,000万円（50台規模）
  → BPO月15万円（年180万円）で ROI 7.8〜11.1倍
```

### 1.3 建設業積算AIとの構造比較

| 項目 | 建設業積算AI | 配車計画AI |
|---|---|---|
| 属人化の対象 | 熟練積算士 | ベテラン配車マン |
| 入力 | 図面PDF | 配送依頼（荷主×届先×荷量×日時） |
| 処理 | 数量拾い出し→単価選定→内訳書 | ドライバー×車両×ルート→最適配車 |
| 出力 | 内訳書（Excel/PDF） | 配車表（ドライバー別の運行指示） |
| 学習データ | 過去の積算実績 | 過去の配車実績+走行データ |
| 共通パターン | 「経験と勘」の形式知化→AI推定→人間が確認・修正 |

### 1.4 配車計画の主な制約条件（ゲノムに詰める内容）

**ドライバー制約**
- 運転時間: 1日最大9時間（改善基準告示）
- 拘束時間: 1日最大13時間（例外15時間、週2回まで）
- 休息期間: 継続11時間以上を基本（最低9時間）
- 連続運転: 4時間以内に30分以上の休憩
- 年間残業上限: 960時間（2024年4月〜）

**車両制約**
- 積載量（2t/4t/10t/トレーラー）
- 車種（平ボディ/ウイング/冷凍/タンク）
- 車両の整備予定・車検日

**配送制約**
- 荷主の指定時間帯（時間指定配送）
- 荷卸し時間の見積もり
- 有料道路の使用可否（コスト制約）
- 特殊条件（危険物、重量物、温度管理）

### 1.5 業務フロー

```
【前日〜当日朝】
  荷主から配送依頼受付（電話/FAX/メール/EDI）
  → 配送依頼を一覧化（荷主×届先×荷量×時間指定×特記事項）
  → AIが最適配車案を生成
    - ドライバー×車両の割当
    - ルート（配送順序）の最適化
    - 改善基準告示の制約チェック
  → 配車マンが確認・修正
  → ドライバーへ運行指示（配車表/アプリ）

【当日】
  出庫前点呼（対面 or IT点呼）
  → 運行開始
  → リアルタイムの進捗管理（GPS連携）
  → 変更対応（キャンセル・追加→再配車）
  → 帰庫・帰庫後点呼

【月次】
  実績分析
  → ドライバー別の稼働率・残業時間・燃料効率
  → 荷主別の収益性分析
  → 改善基準告示の遵守状況レポート
```

### 1.6 課題テーブル

| 課題 | 具体的な問題 | 影響額（年間） | 発生頻度 |
|---|---|---|---|
| 配車マンへの一極集中 | 1人が休むと配車が回らない | 事業停止リスク | 常時 |
| ルートの非最適化 | 経験と勘。迂回・空車走行が多い | 燃料費の無駄（年数百万円） | 毎日 |
| 2024年問題への対応不足 | 残業時間の把握が手作業。超過に気づかない | 車両停止命令リスク | 常時 |
| 当日の変更対応 | キャンセル・追加で配車を組み直す時間がない | ドライバーの遊休・非効率配送 | 毎日 |
| 荷主対応の遅さ | 「この荷物、明日行ける？」に即答できない | 受注機会の逸失 | 毎日 |
| 配車マンの高齢化 | 平均年齢50代。後継者がいない | 5-10年以内に業務崩壊 | 常時 |

## 2〜8. 拡張ロードマップ

- 荷主ごとにバラバラの運賃体系。燃料サーチャージの自動計算

### ⑤ 車両管理（Tier2 / ¥2-3万）
- 車検・点検・保険の期限管理。整備記録一元化
- 製造業の「設備保全」パターンの横展開

- 入出庫管理。ロケーション最適化

- 建設業の「安全書類」パターンの横展開

### ⑧ 許可更新・届出（Tier3 / ¥1-2万）
