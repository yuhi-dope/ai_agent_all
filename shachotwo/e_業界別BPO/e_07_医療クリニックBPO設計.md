## 1. レセプト点検AI（★実装対象）

### 1.1 問題の本質

```
医科レセプトの現状:
  ・診療報酬点数表は約5,000項目
  ・診療科ごとに算定ルールが大きく異なる（内科vs整形vs眼科...）
  ・薬剤×病名の適応チェックが膨大
  ・検査×病名の適応、回数制限の管理
  ・DPC（包括払い）対象病院と出来高払いの違い

  → 歯科より算定ルールが桁違いに複雑
  → だからこそAI化の価値が高い
```

### 1.2 支払いロジック

```
■ 直接効果①：査定損失の回避
  月間請求額1,000万円 × 査定率0.3%改善 = 月3万円 = 年36万円
  月間請求額2,000万円の医院なら = 月6万円 = 年72万円

■ 直接効果②：算定漏れの防止
  特定疾患療養管理料、外来管理加算、在宅時医学総合管理料等
  → 算定漏れ月3-10万円 = 年36-120万円

■ 直接効果③：院長のレセプトチェック時間削減
  月5-10時間 × 時間単価20,000円 = 月10-20万円 = 年120-240万円

■ 合計効果：年192-432万円
  → BPO月8万円（年96万円）で ROI 2.0〜4.5倍
```

### 1.3 歯科レセプトとの差分（ゲノム差分）

| 項目 | 歯科 | 医科（クリニック） |
|---|---|---|
| 点数表の項目数 | 約1,500 | 約5,000 |
| 診療科の分岐 | なし（歯科のみ） | 内科/整形/皮膚/眼科/耳鼻科等で大きく異なる |
| 薬剤チェック | 限定的 | 膨大（薬剤×適応病名×禁忌×用量） |
| DPC | 対象外 | 有床19床以下は出来高（対象外が多い） |
| 主な加算 | か強診、SPT等 | 特定疾患管理料、在宅系加算、生活習慣病管理料等 |
| 共通エンジン | ルールベースチェック→AI予測→レポート生成（同一） |

→ エンジンは歯科と同一。ゲノム（算定ルールDB）を診療科別に追加する設計。

### 1.4 診療科別の主な算定ポイント（ゲノムに詰める内容）

**内科**
- 特定疾患療養管理料（月2回まで、対象疾患の確認）
- 生活習慣病管理料（糖尿病・高血圧・脂質異常症）
- 在宅時医学総合管理料（訪問診療を行う場合）
- 薬剤の適応病名チェック（処方薬×病名の対応）

**整形外科**
- リハビリテーション料（疾患別リハの算定上限日数）
- 運動器リハ/脳血管リハの区分管理
- 注射料（関節注射、トリガーポイント注射）

**皮膚科**
- 特定疾患処方管理加算
- 皮膚科特定疾患指導管理料
- 光線療法の算定

**眼科**
- 精密眼底検査、精密眼圧測定
- 視力検査の回数制限
- コンタクトレンズ検査料

### 1.5 業務フロー

```
（歯科のレセプト点検と同一フロー。差分は算定ルールDBのみ）

【日次】 診療→カルテ入力→レセコン入力→会計
【月初】 前月分レセプト一括出力→点検→院長最終チェック
【毎月10日】 オンライン請求（支払基金・国保連）
【翌月〜翌々月】 返戻・査定の確認→再請求
```

### 1.6 課題テーブル

| 課題 | 具体的な問題 | 影響額（年間） | 発生頻度 |
|---|---|---|---|
| 院長がチェックに張り付き | 月末〜月初の5-10時間 | 機会損失120-240万円 | 毎月 |
| 病名漏れ | 処方薬に対応する病名がない→査定 | 36-72万円の減収 | 月数件〜数十件 |
| 算定漏れ（取れるのに取っていない） | 管理料・加算の算定漏れ | 36-120万円の逸失 | 毎月 |
| 薬剤の適応外処方 | 適応病名のない薬剤処方→査定 | 同上 | 月数件 |
| 事務員の属人化 | レセプトチェックできるのは1人 | 事業リスク | 常時 |
| 診療報酬改定への追従 | 2年ごとの改定。新ルールの把握漏れ | 誤請求リスク | 2年ごと |

## 2〜8. 拡張ロードマップ

> 以下は#1モジュール（レセプト点検AI）のPMF確認後に実装。

### ② 患者予約・リコール（Tier1 / ¥3-5万）
- 慢性疾患（糖尿病・高血圧）のフォローアップ管理
- 歯科の「リコール管理」パターンの横展開
- 予約枠最適化（診療内容別の所要時間推定）

### ③ 紹介状・診断書作成支援（Tier1 / ¥2-3万）
- 医師が診療後に書類作成で月20-40h
- LLMで下書き生成、医師は確認・修正のみ
- 士業の「書類作成支援」パターンの横展開

### ④ 経理・給与（Tier2 / ¥2-3万）
- 保険診療+自費+健診の収益管理
- 歯科の「経理・給与」パターンの横展開

### ⑤ 院内感染・医療安全（Tier2 / ¥1-2万）
- チェックリスト+報告書+保健所対応
- 歯科の「院内感染管理」パターンの横展開

### ⑥ 採用支援・看護師（Tier2 / ¥2-3万）
- 看護師求人倍率2.4倍。紹介料60-100万/人
- 歯科の「採用支援（衛生士）」パターンの横展開

### ⑦ 施設基準管理（Tier3 / ¥1-2万）
- 在宅療養支援診療所等の基準充足チェック
- 歯科の「届出・許認可管理」パターンの横展開

### ⑧ 届出・許認可（Tier3 / ¥1-2万）
- 保険医療機関指定更新
- 歯科の「届出・許認可管理」パターンの横展開
