# 製造業 × シャチョツー — 全貌

> **このドキュメントの目的**: 製造業の業界人が読んで「完全に理解している」と感じるレベルで、
> シャチョツーが製造業に何をもたらすかを1つのドキュメントにまとめる。
> **対象読者**: 製造業経営者、工場長、見積担当、コンサルタント、パートナー候補
> **最終更新**: 2026-03-28

## 1. 製造業の現実（業界人なら「分かる」話）

### 1.1 中小製造業の現場

従業員10-300名の中小製造業、約5.2万社（経産省 工業統計調査）。
金属加工、プレス、板金、射出成形、食品、電子部品。業種は違っても共通する現実がある。

**社長の1日（Before）**

```
AM 7:00  出社。工場長と今日の生産予定を確認（ホワイトボード）
AM 7:30  急ぎの見積依頼がメールで3件。図面をプリントして見積担当に渡す
         → 見積担当は今日も残業確定。1件に2時間はかかる
AM 8:00  昨日の夜勤で設備が止まったらしい。修理業者を呼ぶ
         → いつも突然壊れる。交換時期がわからない
AM 9:00  得意先から電話「例の部品、来週に前倒しできない？」
         → 工場長「うーん、MCが来週いっぱいなんだよな…」
         → この判断が工場長の頭の中にしかない
AM 10:00 品質クレームの報告。先月納品した部品の寸法がズレていた
         → 検査記録は紙。原因を遡るのに半日かかる
PM 1:00  購買担当「すみません、材料の発注忘れてました。ラインが明日止まります」
         → 何がいつ足りなくなるか、誰も把握していない
PM 2:00  新しく入ったベトナム人実習生に作業を教える
         → 手順書が5年前のまま。日本語しかない
PM 3:00  来月のISO審査の準備。書類が全然揃っていない
         → 品質管理責任者が1人で泣きながら準備中
PM 5:00  やっと見積が1件できた。あと2件は明日。
         → 顧客は3社に見積を出している。遅い会社は候補から落ちる
PM 7:00  退社。今日も何も「経営」できなかった。
```

### 1.2 数字で見る課題

| 指標 | 数字 | 出所 |
|---|---|---|
| 見積工数 | 1件30分-3時間 × 月100件 = 月50-300時間 | 自社ヒアリング |
| 回答遅延 | 業界平均3-5日。2日以内回答で受注率30%UP | 中小機構調査 |
| 赤字受注率 | 中小の15-20%が原価割れ見積 | 日刊工業新聞 |
| 熟練依存 | 見積できる人の平均年齢57歳。10年後に半減 | 経産省 |
| 生産計画 | 80%が属人的（工場長の頭の中） | ものづくり白書 |
| 品質コスト | 不良対応コスト 年間売上の3-5% | 品質管理学会 |
| 有効求人倍率 | 製造業1.8倍（全業種平均1.3倍） | 厚労省 |

### 1.3 2025年問題と業界トレンド

**団塊世代の完全退職（2025-2027）**

見積・生産計画・品質判断のノウハウが消える。中小製造業の約30%が後継者未定（中小企業庁）。
ベテラン見積担当が1人しかいない工場で、その人が退職したら受注が止まる。

**DX補助金の追い風**

| 補助金 | 上限 | 補助率 |
|---|---|---|
| IT導入補助金 | 最大450万円 | 2/3 |
| ものづくり補助金 | 最大1,250万円 | 1/2-2/3 |
| 事業再構築補助金 | 最大1億円 | 1/2-2/3 |

シャチョツーの年間費用（¥156-480万）は、IT導入補助金で2/3カバー可能。

**受注構造の不可逆な変化**

- 多品種少量化（ロットサイズ 10年で50%縮小）
- 短納期化（リードタイム 10年で30%短縮）
- 見積件数は増えるが1件あたりの売上は減る

結論: **見積工数の効率化が死活問題**。件数をさばけない会社は、受注機会を逃して縮小していく。

## 3. 見積の全貌（これが分かってないと話にならない）

### 3.1 見積の基本フロー

```
Step 1: 引合い受領（図面PDF/DXF + 仕様がメールで届く）
  │
Step 2: 図面読解（形状・寸法・公差・材質・表面粗さを読む）← 最も属人的
  │
Step 3: 工程推定（丸物→旋盤、角物→MC、板物→レーザー+曲げ）← 2番目に属人的
  │
Step 4: 工数算出（段取り時間 + サイクルタイム × 数量）
  │
Step 5: コスト計算（チャージレート × 時間 + 材料費 + 外注費 + 管理費 + 利益）
  │
Step 6: 見積書作成・提出
```

### 3.2 チャージレートの仕組み

設備ごとの時間単価。会社ごとに全く違う。これが見積の根幹。

```
チャージレート = (設備償却費 + 労務費 + 間接費) / 年間稼働時間
```

| 設備 | チャージレート相場 |
|---|---|
| 汎用旋盤 | ¥3,500-5,000/h |
| CNC旋盤 | ¥5,000-7,000/h |
| マシニングセンタ（MC） | ¥6,000-9,000/h |
| 5軸MC | ¥10,000-15,000/h |
| 研磨盤 | ¥5,000-8,000/h |
| ワイヤーカット | ¥8,000-12,000/h |
| レーザー加工機 | ¥6,000-10,000/h |

同じCNC旋盤でも、償却が終わった設備なら¥4,000/h、最新機なら¥7,000/h。
**会社固有のチャージレートデータが見積AIの精度を決める。**

### 3.3 見積の4類型

**Type A: チャージレート型（金属切削・成型）**

```
加工費 = (段取り時間 + サイクルタイム × 数量) ÷ 60 × チャージレート

例: S45Cシャフト φ30×100mm、100個、CNC旋盤
  段取り45分 + 8分/個 × 100個 = 845分 = 14.1時間
  加工費 = 14.1h × ¥6,000/h = ¥84,600
  材料費 = 0.83kg × ¥150/kg × 1.15（ロス率）× 100個 = ¥14,300
  表面処理（黒染め）= ¥300 × 100個 = ¥30,000
  検査費 = ¥5,000
  管理費(15%) = ¥20,085
  利益(15%) = ¥23,098
  → 見積金額: ¥177,000（単価¥1,770/個）
```

**Type B: 面積/回数/長さ型（板金）**

```
レーザー切断: 切断長(mm) × mm単価 + ピアス数 × ピアス単価
曲げ: 曲げ回数 × ¥150-600/回
溶接: TIG ¥10-20/mm, MIG ¥5-10/mm, スポット ¥30-50/点
```

**Type C: BOM組立型（機械メーカー）**

```
原価 = Σ(部品費) + 設計費 + 組立費 + 試運転費 + 輸送費
部品1点ごとにType A/Bの見積が入れ子になる
```

**Type D: バッチ配合型（食品OEM・化学受託）**

```
原価 = Σ(原料×配合比×kg単価) × バッチ数 + 工程費 + 包装資材費 + 検査費
配合展開: レシピの各原料% × バッチサイズ(kg) = 原料使用量
```

### 3.4 見積の「例外」（ここがAI化の難所）

| パターン | 内容 | AI対応 |
|---|---|---|
| **特急料金** | 1週間以内→+20-50%、2-3日→+50-100% | 設備稼働状況から自動可否判定 |
| **試作 vs 量産** | 1個の単価と1万個の単価は10倍違う | 数量ブレイクの自動計算 |
| **リピート品** | 前回見積をベースに微修正。値引き圧力あり | 過去見積DB自動参照 |
| **得意先別掛率** | A社には15%利益、B社には10%利益 | 顧客別ルールDB |
| **材料支給** | 顧客が材料を持ってくる→材料費ゼロ | 自動控除 |
| **図面不完全** | 寸法抜け・公差未指定・材質未記載 | 確認事項リスト自動生成 |
| **端材活用** | 前の案件の残りを使えば材料費が劇的に下がる | 端材在庫DBと照合 |
| **外注先の価格差** | 同じメッキでも外注先で2-3倍違う | QCD評価DBから自動選定 |
| **数量変更** | 「100個→50個」で段取り按分が2倍→単価UP | 即時再計算 |
| **値引き要求** | 利益率低下/公差緩和/ロットまとめ等の代替案提示 | 削減ポイントの自動提案 |

### 4.2 見積AI — 3層アーキテクチャ（キラーフィーチャー）

```
Layer 0: LLMデフォルト
  何も設定がなくても概算が出る。全24業種Day 1対応。
  精度: ±30-50%（概算レベル）
  Confidence: 0.3-0.5
  作業: ゼロ

Layer 1: YAML設定
  業種データ（材料単価・設備・工程ルーティング）を定義。
  精度: ±15-25%
  Confidence: 0.6-0.8
  作業: YAMLファイル1つ追加（数時間）

Layer 2: Pythonプラグイン
  計算の公式自体が違う業種に対応。
  精度: ±5-15%
  Confidence: 0.7-0.9
  作業: プラグイン1ファイル実装（数日）
  必要な業種: 樹脂成型（金型償却）、食品/化学（配合展開）、電子部品（BOM展開）の3グループのみ

+ 顧客の過去実績データ → Confidence: 0.9-1.0（精度±3-10%）
```

**解決順序（各データ項目ごと）:**

```
チャージレート: 顧客DB → YAML → LLM推定
材料単価:       顧客DB → YAML → LLM推定
工程リスト:     Plugin → YAMLルール → LLM推定
サイクルタイム: 過去実績 → YAML → LLM推定
```

上のレイヤーがなければ下にフォールバック。顧客データが溜まるほど精度が上がる。

## 5. 競合比較

| | シャチョツー | CADDi | meviy | Terminal | ものレボ |
|---|---|---|---|---|---|
| **ポジション** | BPO全域（経営OS） | 受発注プラットフォーム | 自動見積サービス | 生産管理SaaS | 生産管理SaaS |
| **見積機能** | AI見積（3層エンジン） | 調達側が価格提示 | 3D CAD→自動見積 | なし | なし |
| **対象** | 中小全般（加工屋側） | 調達企業+加工屋 | 板金・切削限定 | 中小製造 | 中小製造 |
| **強み** | 8モジュール統合+会社固有学習 | 規模・取引データ量 | 3D CAD連携精度 | UI/UXの完成度 | 低価格 |
| **弱み** | 新参。実績がまだ少ない | 加工屋は受身。自社の見積に使えない | 対応工程が限定的。2D図面非対応 | 見積機能なし | 見積機能なし |
| **月額** | ¥3-40万 | 手数料型（取引の数%） | 無料（発注時課金） | ¥5-15万 | ¥1-5万 |
| **カバー範囲** | 見積→生産→品質→在庫→ISO | 調達効率化 | 見積のみ | 生産管理 | 生産管理 |

**シャチョツーの独自ポジション:**

1. **競合は「1つの課題」を解くツール。シャチョツーは「8つの課題」をまとめて解くOS。** 見積AIで入って、生産計画→品質→在庫→ISOと横展開する。
2. **加工屋側の見積に特化。** meviyは発注者向け。CADDiは調達プラットフォーム。自社の見積担当の代替になるのはシャチョツーだけ。
3. **使うほど精度が上がる。** 会社固有のチャージレート・工数実績が蓄積され、3ヶ月で精度が±30%→±15%→±10%に収束する。
4. **ゲノム駆動。** 金属加工も食品もプラスチックも、YAML+プラグインの追加だけで1つのエンジンが対応。

**正直な弱み:**
- 新参。導入実績がまだ少ない。パイロット企業での実証フェーズ
- 3D CAD対応はPhase 2+。現時点は2D図面（PDF/画像）のみ
- Layer 0（LLMのみ）の精度は±30-50%。顧客データの蓄積が精度の鍵

### 9.4 主要産業集積地の沿革と土着文化

⑪立地戦略 + 営業戦略の地域別カスタマイズに活用。

**燕三条（新潟県）— 金属加工の聖地**
- 沿革: 江戸時代の和釘生産 → 明治の洋食器 → 昭和のステンレス加工 → 令和のiPhoneケース/アウトドアギア
- 土着文化: 「職人が職人を育てる」師弟文化。新参企業は3-5年かけて信頼を構築する必要あり。品質への要求水準が極めて高い。「うちの研磨は日本一」という自負がある。協力関係が構築できれば、技術相談・緊急対応で地域ネットワークが活きる。
- BPOへの示唆: 見積AIの精度が「うちの感覚と違う」と言われるリスク。初期は人間との並走必須。

**東大阪（大阪府）— 町工場の密集地帯**
- 沿革: 大阪の「モノづくり」の中心。ねじ・バネから宇宙部品まで。町工場6,000社が半径5km内に密集。
- 土着文化: 「なんぼ（いくら）？」の値段交渉文化。見積回答速度が競争力。1日以内に回答が標準。横のつながりが強い。自社でできない加工は隣の工場に流す。
- BPOへの示唆: 見積速度の改善が最大の訴求ポイント。「3時間→30分」は刺さる。

**浜松（静岡県）— 自動車・楽器の街**
- 沿革: ヤマハ/スズキ/カワイの企業城下町。自動車部品+楽器部品の二本柱。
- 土着文化: Tier1-2のサプライチェーンに組み込まれている企業が多い。品質基準が厳しい（自動車PPAP対応）。EV化の影響が最も大きい地域の一つ。エンジン部品→モーター部品への転換が課題。
- BPOへの示唆: 品質管理(③)と技術戦略(⑬)のニーズが特に高い。

**諏訪（長野県）— 精密機器の里**
- 沿革: セイコーエプソンを中心とした精密加工クラスター。時計→プリンタ→半導体装置の進化。
- 土着文化: μm（ミクロン）単位の精度が当たり前。「0.01mmの公差で文句を言うのは素人」。冬の厳しさと職人気質が重なる。寡黙だが仕事は確実。
- BPOへの示唆: 品質データの管理(Cpk/SPC)への関心が高い。

**大田区（東京都）— 試作のメッカ**
- 沿革: 「図面を持っていけば1日で試作品ができる」と言われた時代。3,500社→2,000社に減少。
- 土着文化: 多品種少量の極み。1個から受ける。技術の幅が広い（何でもやる）。地代が高く、利益率が低い。後継者不在率が高い。
- BPOへの示唆: 見積の多品種対応と事業承継(⑯)のニーズ。

**北九州（福岡県）— 鉄鋼・重工業**
- 沿革: 八幡製鐵所から100年。鉄鋼→自動車→半導体（TSMC熊本の波及効果）。
- 土着文化: 大企業の下請けが多い。Tier2-3の中小企業が技術力を持つ。九州全域の物流拠点としての立地優位性。
- BPOへの示唆: 半導体関連の設備投資(⑩)と立地戦略(⑪)のニーズが急上昇中。

**群馬/栃木 — 自動車部品（北関東ベルト）**
- 沿革: SUBARU(群馬)/日産(栃木)の企業城下町。東北自動車道沿いに部品工場が並ぶ。
- 土着文化: 堅実で保守的。新しいものには慎重。実績と信頼で取引が成立する。「隣の工場がやっている」が最大の導入理由になる。
- BPOへの示唆: パイロット企業1社の成功事例が地域展開のカギ。

**熊本/広島 — 半導体新興（2024-2026）**
- 沿革: TSMC熊本工場(2024年稼働)、マイクロン広島工場の影響で急激に産業構造が変化中。
- 土着文化: 農業県から半導体県への転換。技術人材の争奪戦。地価急騰。
- BPOへの示唆: 新規参入企業向けの設備投資(⑩)+人材(⑮)のセット提案。

| 表面処理(環境規制) | 全般 | →±5% | RoHS/REACH規制強化 | 六価クロムフリー化 |
| 3Dプリンタ(金属AM) | 試作/航空 | ▲40-60% | 造形速度向上+材料多様化 | 後工程(仕上げ加工)との融合 |
| 溶接(建設/インフラ) | 建設 | →±5% | 国土強靭化計画 | 溶接ロボット導入 |

**中国リスク:**
- 中国の過剰生産能力 → 鋼材/アルミ材の国際価格下落 → 日本の材料費は下がるが受注単価も下がる
- 中国EVメーカーの台頭 → 日系自動車部品サプライチェーンの縮小
- 米中デカップリング → 半導体関連は国内回帰で恩恵

**為替の影響:**
- 円安(¥150+): 輸出企業に有利。ただし材料輸入コスト増で中小は相殺
- 円高(¥120-): 輸入材料が安くなるが、海外からの受注が減少
- 中小製造業の多くは内需型 → 為替の直接影響は限定的。ただし顧客(自動車メーカー等)経由で間接影響

### 9.6 加工技術のシフトと進化（2025-2035年）

⑩設備投資 + ⑬技術戦略の「次に何が来るか」を示すテクノロジーロードマップ。

**切削加工 → 積層造形（3Dプリンタ）**
- 現状: 試作・小ロット（月産1-50個）で3Dプリンタが切削を置換しつつある
- 転換点: 金属AM（SLM/EBM）の造形速度が現在の3-5倍になると、月産100個まで経済合理性
- 切削が残る領域: 量産（月産1,000個以上）、高精度（IT5以下）、大型（500mm以上）
- 社長への示唆: 試作売上比率30%以上なら、3Dプリンタ導入の検討タイミング

**放電加工 → レーザー加工**
- 現状: 薄板・微細加工（板厚3mm以下）でファイバーレーザーが放電を置換
- 転換点: 2025-2028年にレーザー出力が上がり、板厚10mmまで対応可能に
- 放電が残る領域: 厚物（板厚20mm以上）、超硬合金、複雑形状の金型加工
- 社長への示唆: 薄板中心なら放電→レーザーへの移行で生産性3倍

**手仕上げ → ロボット研磨**
- 現状: 単純形状（平面、外径）の研磨はロボット化進行中
- 転換点: AIビジョン+力覚センサの進化で、複雑形状も対応可能に（2027-2030年）
- 手仕上げが残る領域: 金型の最終仕上げ、特殊素材、超鏡面
- 社長への示唆: 研磨工程の人件費が年間¥500万以上なら、ロボット研磨のROIが出る

**単品加工 → 複合加工機**
- 現状: 旋盤+MCの2工程を1台で完結する複合加工機（MillTurn等）
- メリット: 段取り替え時間を50-70%削減、仕掛品ゼロ、精度向上（チャッキング回数減）
- デメリット: 1台¥5,000万-1.5億、プログラミングが複雑、故障時の影響が大きい
- 社長への示唆: 月産100種以上の多品種生産なら、複合化のROIが高い

**2D CAD → 3D CAD/CAM**
- 現状: 中小製造業の3D CAD導入率は約30%（2024年）
- 転換点: 5軸加工には3D CAM必須。2D CADでは5軸のツールパスが出せない
- 社長への示唆: 5軸MC導入を検討するなら、先にCAD/CAM投資（¥200-500万）が必要

**人手検査 → AI画像検査**
- 現状: 外観検査（キズ/打痕/汚れ）からAI画像検査の導入が進む
- 転換点: 寸法検査はまだ三次元測定機が主流。AI化は2028-2030年
- 社長への示唆: 外観検査の工数が月40時間以上なら、AI画像検査（¥300-1,000万）のROIが出る
