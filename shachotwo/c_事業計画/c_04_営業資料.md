### 9.1 マーケティング表現の修正ガイドライン

#### 承認済み表現 vs 禁止表現

| シーン | 禁止表現 | 承認済み表現 | 理由 |
|---|---|---|---|

### 10.1 ROIフレームワーク詳細

```
ROI(%) = (年間効果合計 − 年間投資額) / 年間投資額 × 100

年間効果合計 = A + B + C + D + E
  A: 社長の時間価値回復
  B: 意思決定の質向上（ミス削減 + 判断速度向上）
  C: 属人化リスク削減（社長不在リスク + キーマン退職リスク）
  D: 業務自動化効果（BPO）
  E: ネットワーク効果（ベンチマーク = コンサル代替価値）
```

## 13. Sales Objection Handling（想定反論と切り返し）

### 反論パターン10選

```
■ 反論1: 「AIに会社の情報を入れて大丈夫なの？」
  切り返し:
  → 「ごもっともなご心配です。3つの安全策を取っています:
    ①全データはAES-256暗号化で保存
    ②AIの学習にお客様のデータは一切使いません（契約で保証）
    ③いつでも全データをエクスポートして削除可能です
    大手銀行と同じレベルの暗号化で、メガバンクのセキュリティ審査にも対応できる設計です」

■ 反論2: 「うちは特殊だから汎用AIじゃ無理でしょ」
  切り返し:
  → 「実はそこが一番の強みです。シャチョツーは"業種別テンプレート"を持っていて、
    御社の業種の業務パターンを最初から80%理解した状態でスタートします。
    残り20%は社長との対話で2週間程度で完成します。
    汎用AIではなく、御社専用のAIに育てていくイメージです」

■ 反論3: 「コンサルの方が安心じゃない？」
  切り返し:
  → 「コンサルは月100万で3ヶ月のプロジェクト、シャチョツーは月3万で毎日動きます。
    しかもコンサルの知見はプロジェクト終了後に散逸しますが、
    シャチョツーは蓄積し続けます。年間コスト比較:
    コンサル: 300万〜1,200万 / シャチョツー: 36万〜96万
    10倍以上のコスト差で、365日稼働です」

■ 反論4: 「社員が使いこなせるか不安」
  切り返し:
  → 「LINEで質問するだけです。社長がLINEを使えれば使えます。
    初期設定も7日間のオンボーディングで、こちらが全て対応します。
    ITに詳しくない建設業の社長さんが一番活用してくださっています」

■ 反論5: 「他のAIツール（ChatGPTとか）で十分じゃない？」
  切り返し:
  → 「ChatGPTは"一般的な回答"をします。シャチョツーは"御社の回答"をします。
    『うちの見積もり基準は？』と聞いた時、ChatGPTは一般論を答えますが、
    シャチョツーは『利益率15%以上、ただし既存顧客は10%でもGO、
    社長承認は100万超から』と答えます。この差が全てです」

■ 反論6: 「月3万は高い」
  切り返し:
  → 「月3万で、社長の時間が月10時間以上空きます。
    社長の時給を仮に1万円とすると、月10万円分の価値です。
    加えてSaaS統合で月数万円の削減、BPOで人件費の削減。
    実質的なROIは初月から3倍以上です」

■ 反論7: 「データが取られるんじゃないの？（競合に漏れないか）」
  切り返し:
  → 「全てのデータは会社単位で完全に分離されています（行レベルセキュリティ）。
    他社のデータは見えませんし、当社の社員もお客様のデータにアクセスできません。
    ベンチマーク機能では匿名化・統計処理した情報のみを使用し、
    個社を特定できる情報は一切共有されません」

■ 反論8: 「今は忙しいから、落ち着いてから」
  切り返し:
  → 「実は"忙しい会社"ほど効果が大きいです。
    社長が忙しくて現場に指示を出せない → シャチョツーが代わりに回答する。
    初期設定は30分の音声インタビューだけ。
    むしろ忙しい今こそ、社長の分身を作るタイミングです」

■ 反論9: 「うちみたいな小さい会社に必要？」
  切り返し:
  → 「10人の会社だからこそ、社長一人に知識が集中するリスクが高い。
    社長が倒れたら会社が回らない、これが中小企業最大のリスクです。
    シャチョツーは"社長のバックアップ"です。
    保険と同じで、何もない時にこそ備えるものです」

■ 反論10: 「導入事例はあるの？」
  切り返し:
  → 「建設業（30名）で月8時間の工数削減、製造業（50名）でQ&A応答時間を
    "翌日回答"から"即時回答"に短縮、歯科（15名）で新人育成期間を
    3ヶ月→1.5ヶ月に短縮した事例があります。
    ご興味あれば、同業種の事例を詳しくご説明します」
```
