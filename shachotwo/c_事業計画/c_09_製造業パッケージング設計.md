## 1. 3ティア詳細設計

### 1.1 Starter（月額 ¥50,000）

**コンセプト**: 社長が「お、これ便利」と思うまでのハードル最低

```
含まれるもの:
  ・ブレイン全機能（Q&A / ナレッジ / デジタルツイン）
  ・⑨ PLインテリジェンス簡易版（月次PL自動取得 + 加工種別分析）
  ・Day 1 Value 3機能（ファイルチェック/議事録/文書下書き）
  ・ユーザー5名（社長1名 + ヘルパー4名）
  ・LINE / メール / Web Chat チャネル

含まれないもの:
  ・①-⑧ BPO業務自動化モジュール
  ・⑨ PLインテリジェンスのフル機能（業界ベンチマーク比較、四半期トレンド）
  ・⑩-⑬ 経営インテリジェンス
```

**なぜ⑨PLの簡易版を入れるか**:

```
理由1: 社長が毎月見るものを作る
  → 見積AIは担当者の仕事。社長が直接触らない可能性がある
  → PLレポートは社長が毎月100%見る。「あの数字どうなってる？」の答えが即座に出る
  → 利用頻度が高い = 解約しにくい

理由2: BPO導入の動機を作る
  → 「CNC旋盤の粗利が15%しかない」と分かる
  → 「じゃあ見積AIでチャージレートを見直そう」= BPOコアへの自然な導線

理由3: データが溜まる
  → 月次PLデータの蓄積 = ⑨フル版・⑩設備投資シミュレーターの前提条件
  → Starterの間にデータが溜まり、Professionalへの移行時に即座に精度が出る
```

**Starter → Professional 移行トリガー**:

| トリガー | 検知方法 | アップセルアクション |
|---|---|---|
| 見積に関するQ&Aが月5件以上 | Q&Aログ分析 | 「見積AIで自動化しませんか？」提案 |
| PLレポートで赤字加工種別を発見 | ⑨の出力分析 | 「見積AIで適正価格に修正できます」 |
| ファイルチェックで図面PDFを3回以上処理 | 利用ログ | 「見積AIなら図面から自動見積できます」 |
| 利用開始から30日経過 | 日付 | 定期フォローコール |

**価格の根拠**:

```
ブレイン単体: ¥30,000/月（c_02定義）
⑨PL簡易版: ¥20,000/月の付加価値
合計: ¥50,000/月

年額: ¥600,000
IT導入補助金（2/3補助）適用後: ¥200,000/年 = 月¥16,667
→ 「月1.7万円でAI経営参謀」という訴求が可能
```

### 1.2 Professional（月額 ¥250,000）

**コンセプト**: BPO業務自動化の本丸。月次の工数削減効果を実感 → 定着

```
含まれるもの:
  ・Starterの全機能
  ・① 見積AI（3層エンジン × 4類型）         ★キラー
  ・② 生産計画AI（山積み/山崩し/ガント）      ★キラー
  ・③ 品質管理（SPC + Cp/Cpk + 管理図）
  ・④ 在庫最適化（ABC分析 + 安全在庫 + EOQ）
  ・⑤ SOP管理（手順書自動生成 + 多言語翻訳）
  ・⑥ 設備保全（MTBF/MTTR + 保全カレンダー）
  ・⑦ 仕入管理（BOM → MRP → 発注書自動）
  ・⑧ ISO文書管理（9001/14001 + 監査チェックリスト）
  ・⑨ PLインテリジェンス フル版
  ・ユーザー10名
  ・SaaS連携（kintone / freee / 生産管理SaaS CSV）

含まれないもの:
  ・⑩-⑬ 経営インテリジェンス
  ・バックオフィス（経理・給与・勤怠）
```

**c_02 との差分**:

```
c_02定義: BPOコア¥250,000（①-④のみ） + 追加モジュール¥100,000/個（⑤-⑧）
  → ①-⑧全部入れると ¥250,000 + ¥400,000 = ¥650,000

本設計: Professional¥250,000（①-⑨全部入り）
  → 理由: ⑤-⑧を個別課金にすると「どれを選べばいいか分からない」で離脱する
  → 製造業は8モジュールが有機的に連携（見積→生産計画→品質→在庫→SOP→保全→仕入→ISO）
  → バラ売りより「全部入り」で横展開させた方がLTV最大化

価格設計の思想:
  旧: ¥250,000 + ¥100,000 × 4 = ¥650,000（全部入れると高い）
  新: ¥250,000（全部入り。安い）
  → 「¥250,000で8モジュール全部使える」= 競合比で圧倒的な価格競争力
  → 個別アドオンの複雑さを排除 → 営業が説明しやすい → 受注率向上
  → ⑤-⑧の限界費用はほぼゼロ（①-④のデータを使い回すだけ）
```

**Professional の月額価値（ROI）**:

```
ケース: 金属加工 年商5億 従業員30名

① 見積AI:    工数削減120h/月 × ¥3,000 + 受注増¥240万/年 = 年¥672万
② 生産計画:  納期遅延削減 + 外注費最適化 = 年¥200-400万
③ 品質管理:  不良率半減 × 年商5億 × 不良率2.5% = 年¥625万（の半分 = ¥312万）
④ 在庫最適化: 在庫回転率改善 → 運転資金圧縮 年¥100-200万
⑤ SOP管理:  教育工数削減 年¥50-100万
⑥ 設備保全:  突発停止回避 年¥100-300万
⑦ 仕入管理:  発注漏れゼロ + 仕入単価交渉材料 年¥50-100万
⑧ ISO文書:  審査準備工数 年¥30-60万
⑨ PL分析:   赤字加工特定 → 価格改定 年¥100-300万

保守的合計: 年¥1,300-2,800万
費用: ¥250,000 × 12 = 年¥300万
ROI: 4.3-9.3倍
```

### 1.3 Enterprise（月額 ¥450,000）

**コンセプト**: 経営インテリジェンスで「社長の右腕」を完成。LTV最大化

```
含まれるもの:
  ・Professionalの全機能（①-⑨）
  ・⑩ 設備投資シミュレーター（投資回収計算 + 感度分析）
  ・⑪ 立地戦略AI（工場移転/新設の工業団地スコアリング）
  ・⑫ 市場インテリジェンス（為替/材料相場/需要変動 → 自社インパクト）
  ・⑬ 技術戦略AI（技術ロードマップ生成 + 競争力評価）
  ・ユーザー20名
  ・優先サポート（24h以内レスポンス）
  ・四半期経営レビューレポート（自動生成）

追加オプション:
  ・バックオフィス: +¥200,000/月（経理・給与・勤怠。Phase 2+）
```

**Enterprise 追加分の月額価値**:

```
⑩ 設備投資シミュレーター: ¥3-8万/月の価値
  「MC更新に5,000万。5年で回収できるのか？」→ 即答

⑪ 立地戦略AI: ¥2-5万/月の価値
  「第2工場どこに建てる？」→ 補助金込みでスコアリング

⑫ 市場インテリジェンス: ¥3-5万/月の価値
  「円安が進んだら材料費どうなる？」→ 自社PLへのインパクト試算

⑬ 技術戦略AI: ¥3-5万/月の価値
  「うちの5軸MC技術、あと何年持つ？」→ ロードマップ生成

追加分合計: ¥11-23万/月の価値
追加費用: ¥200,000/月（= Enterprise ¥450,000 - Professional ¥250,000）
→ ROI: 0.55-1.15倍（単独では微妙。①-⑧の蓄積データがあって初めて価値が出る）
→ だからこそ Professional → Enterprise の「育てるアップセル」が正解
```

| ⑫ 市場インテリジェンス | ¥100,000 | 為替/材料相場の急変時 |
| ⑬ 技術戦略AI | ¥100,000 | 中期経営計画の策定時 |

```

  アドオン2個: ¥250,000 + ¥200,000 = ¥450,000 = Enterprise と同額

→ 営業トーク:
  「2つ以上追加するなら、Enterpriseにした方がお得ですよ」
  → 自然にEnterprise移行が起きる
```

### 3.3 セグメント別 最適アップセル順序

セグメントによって「刺さるモジュール」の順番が異なる。

**S1: 金属加工（CNC旋盤/MC）— 民生・雑多**

```
最大の痛み: 見積が遅い。件数をさばけない
攻め順: ①見積AI → ④在庫 → ②生産計画 → ③品質 → ⑨PL分析 → ⑥保全
理由: 見積件数が多い（月100件超）。即効性が最も高い
```

**S4: 自動車サプライヤー（Tier 2-3）**

```
最大の痛み: IATF要求が重い。品質記録と赤字受注
攻め順: ③品質管理 → ①見積AI → ⑧ISO/IATF → ②生産計画 → ⑤SOP → ⑨PL分析
理由: Cpk必須。品質管理の自動化で監査対応が劇的に楽になる
```

**S2: 半導体装置向け（精密加工）**

```
最大の痛み: 高精度品の見積。工程推定が難しい
攻め順: ①見積AI → ⑤SOP → ③品質 → ⑥保全 → ②生産計画 → ⑬技術戦略
理由: 1件の見積が高額（数百万〜）。精度が命。SOP管理でクリーン度維持
```

**S9: 食品OEM/化学受託**

```
最大の痛み: 配合展開が複雑。原価計算の精度が低い
攻め順: ①見積AI(Type D) → ④在庫 → ⑦仕入管理 → ③品質(HACCP) → ⑨PL分析
理由: 配合展開の自動化が最大のインパクト。原材料ロスの削減直結
```

**S5: 板金加工**

```
最大の痛み: 切断長/曲げ回数の計算が面倒
攻め順: ①見積AI(Type B) → ②生産計画 → ④在庫 → ⑥保全 → ⑧ISO
理由: Type B見積エンジンで即効性。レーザー加工機の保全需要も高い
```

## 4. フリーミアム/トライアル戦略

### 4.1 14日間無料トライアル（Starter + 見積AI体験）

```
■ なぜ14日か（30日ではなく）
  ・製造業の社長は忙しい。30日あると「来月やろう」で忘れる
  ・14日間の緊迫感で「今週中に試す」行動を促す
  ・競合比較: ANDPAD 14日、Terminal 30日、ものレボ 14日
  → 14日が業界標準。30日は長すぎて離脱率が上がる

■ トライアルで使える機能（Starter + α）
  ・ブレイン全機能（Q&A / ナレッジ / デジタルツイン）
  ・⑨ PL簡易版
  ・① 見積AI（3件まで）★ デモ効果最大
  ・Day 1 Value 3機能

■ なぜ「見積AI 3件まで」を入れるか
  ・見積AIは製造業の最大の痛み = デモ効果が最大
  ・「実際の図面で見積が出る」体験は、Q&Aやナレッジの10倍インパクトがある
  ・3件の制限は「もっと使いたい」を作るため（freeeのトライアルと同じ手法）
  ・3件の間にAIの精度が分かる → 「過去データを入れればもっと精度上がりますよ」
    → Professional への自然な導線

■ トライアル中にやること（カスタマーサクセス）
  Day 1:  Welcome メール + オンボーディングガイド（5分で完了する初回設定）
  Day 3:  「最初の見積は試しましたか？」フォローメール
  Day 7:  中間レビューコール（15分）。使用状況を確認
  Day 10: 「あと4日でトライアル終了です」+ ROI試算レポートを自動送付
  Day 13: 「明日で終了。Starterなら月5万。補助金で月1.7万です」
```
