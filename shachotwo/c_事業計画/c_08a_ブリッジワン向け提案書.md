# AI BPO事業に関する業務提携のご提案

**提案者**: 湊合同会社 代表社員 杉本雄飛
**提案先**: 株式会社ブリッジワン 御中
**日付**: 2026年○月○日

---

## 1. ご提案の概要

湊合同会社が開発するAI BPOプラットフォーム「シャチョツー」を、ブリッジワン様の顧客向けサービスの一環としてご活用いただく業務提携をご提案いたします。

ブリッジワン様のコンサルティング事業に、AIによる業務自動化ツールを組み合わせることで、顧客企業への提供価値を大幅に向上させることが可能です。

---

## 2. 提携スキーム

### 全体構造

```
湊合同会社（AI BPOプロダクトの開発・提供）
  │
  │  業務提携契約
  │
  ▼
ブリッジワン様（コンサルティング + AIツールを組み合わせて顧客に提案）
  │
  ▼
ブリッジワン様の顧客（製造業・建設業・不動産・医療等）
```

### 役割分担

| | ブリッジワン様 | 湊合同会社 |
|---|---|---|
| **役割** | コンサルティング・顧客リレーション | AIプロダクトの開発・運用 |
| **提供するもの** | 顧客基盤・業界知見 | AIツール・技術力 |
| **収益** | コンサルフィー + MRR還元 | プロダクト利用料（MRR） |

---

## 3. ブリッジワン様のメリット

### ① 開発コストゼロ

AIプロダクトの開発費用は全て湊合同会社が負担します。ブリッジワン様に開発投資のリスクはありません。

### ② コンサルの付加価値が飛躍的に向上

従来のコンサルティングに加え、AIによる業務自動化ツールを顧客に提供できるようになります。

- 製造業: 見積業務の自動化（回答3日→30分）
- 建設業: 積算業務の自動化
- 全業界共通: ナレッジ管理・Q&A・業務フロー最適化

「コンサルだけ」の競合と明確に差別化できます。

### ③ 継続的な収益還元

ブリッジワン様経由で導入された顧客のMRR（月額利用料）の**10〜15%を毎月還元**します。コンサル案件が終了した後も、プロダクトの利用が続く限り収益が入り続けます。

### ④ プロダクトの優先利用

ブリッジワン様には、市場価格の**50%割引**での永続利用権と、新機能の**先行アクセス権**（一般公開の3ヶ月前）を付与します。

### ⑤ エクイティによるアップサイド

湊合同会社が株式会社化した際に、ブリッジワン様に**新株予約権（5%）**を発行します。プロダクトが成長すれば、株式としての価値も享受いただけます。

## 4. プロダクト概要: シャチョツー

### 製造業向け（例）

| モジュール | 課題 | AI化の内容 |
|---|---|---|
| 見積AI | 回答に3日かかる | 図面→工程推定→自動見積（30分） |
| 生産計画AI | 工場長の頭の中 | 受注→山積み→ガントチャート |
| 品質管理 | 紙の検査記録 | SPC自動化+不良予兆検知 |
| 在庫最適化 | 多すぎ or 足りない | 需要予測→安全在庫→発注点アラート |

### ROI実績（金属加工・年商5億の場合）

```
年間効果: 約¥1,460万（工数削減+受注増+赤字受注減）
年間費用: 約¥336万
ROI: 4.3倍
```

### 多業界対応

ブリッジワン様が進出されている全ての業界に対応可能です。業界ごとのAI設定（ゲノム）を追加するだけで、同一のプラットフォームで複数業界をカバーできます。

## 5. 契約条件

### 知的財産権

- プロダクトの知的財産権は湊合同会社に帰属します
- ブリッジワン様には永続的なライセンス利用権を付与します

### エクイティ

- 湊合同会社の株式会社化後、新株予約権（5%）を発行します
- 行使条件: MRR 300万円達成 / シリーズA調達完了 / 提携開始から2年経過のいずれか
- 議決権は行使するまで発生しません

### 報酬・費用

| 項目 | 内容 |
|---|---|
| ブリッジワン様の初期費用 | なし |
| MRR還元 | ブリッジワン様経由の顧客MRRの10〜15%を毎月還元 |
| ブリッジワン様のプロダクト利用料 | 市場価格の50%割引 |

### データの取扱い

- 顧客データの利用はAI学習目的に限定し、必ず匿名化します
- 個社を特定可能な形でのデータ利用・第三者提供は行いません
- 契約終了後、提供を受けた生データは削除します

### 販売権

- 湊合同会社はブリッジワン様以外にもプロダクトを販売できます（非独占）
- ただしブリッジワン様には最恵待遇（50%割引・先行アクセス・MRR還元）を保証します

### 契約期間

- 1年間（自動更新）
- 解除は3ヶ月前の書面通知

## 6. 杉本の関わり方

現在、杉本はブリッジワン様に正社員（経営企画）として在籍しております。
本提携においては、以下の通り業務を分離します。

| | 正社員としての業務 | 湊合同会社としての業務 |
|---|---|---|
| 内容 | 経営企画 | AI BPOプロダクト開発・運用 |
| 時間帯 | 業務時間内 | 業務時間外・休日 |
| 成果物の帰属 | ブリッジワン様 | 湊合同会社 |

就業規則第25条に基づき、兼業許可の申請を別途提出いたします。

### 段階的な移行

プロダクト事業の成長に合わせて、以下の段階で移行を予定しています。

```
Phase 1（今〜6ヶ月）: 正社員を継続しながらプロダクト開発
Phase 2（6ヶ月〜1年）: 勤務形態の調整（時短等）
Phase 3（1年〜）: 業務委託契約のみに移行。ブリッジワン様は最大顧客として継続
```

## 7. 想定されるご質問

**Q. 顧客を持っていかれることはないか？**

A. ありません。ブリッジワン様の顧客はブリッジワン様のものです。湊合同会社はAIツールを提供する立場であり、顧客との直接取引は行いません。永続ライセンスにより、ブリッジワン様は提携終了後もプロダクトを継続利用できます。

**Q. 事業がうまくいかなかった場合は？**

A. ブリッジワン様の損失はゼロです。初期費用・開発費用は全て湊合同会社が負担しています。AIツールを利用いただいた期間中の業務改善効果だけが残ります。

**Q. なぜ社内で作らないのか？**

A. AIプロダクトを社内開発する場合、エンジニア採用・開発投資・技術的リスクの全てをブリッジワン様が負うことになります。外部の専門法人に委託することで、ブリッジワン様はリスクゼロで最新のAIツールを活用でき、コンサル事業に集中できます。

**Q. 他社にも売るのか？**

A. はい。他社への販売によりプロダクトの開発資金が増え、機能改善のスピードが上がります。結果として、ブリッジワン様が使うツールの品質も向上します。ブリッジワン様には50%割引・先行アクセス・MRR還元の最恵待遇を保証いたします。
