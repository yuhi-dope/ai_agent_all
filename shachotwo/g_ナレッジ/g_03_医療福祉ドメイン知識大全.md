## 1. 医療・福祉とは何か

### 1.1 業種分類（施設種別の全体像）

日本の医療・福祉は約37万施設で構成される。大きく「医療」「介護」「障害福祉」の3領域に分かれ、それぞれ根拠法・報酬体系・監督官庁が異なる。

```
医療・福祉（約37万施設）
  │
  ├─ 医療（約18万施設）
  │    ├─ 病院（約8,000施設。20床以上）
  │    │    ├─ 急性期病院（救急、手術中心。DPC対象）
  │    │    ├─ 回復期リハビリ病院（脳卒中・骨折後のリハビリ）
  │    │    ├─ 慢性期病院（療養病床。長期入院）
  │    │    └─ 精神科病院（約1,600施設。平均在院日数270日）
  │    │
  │    ├─ 一般診療所（約105,000施設）
  │    │    ├─ 内科（約35,000）← 最大ボリューム
  │    │    ├─ 整形外科（約8,000）← リハビリ併設が多い
  │    │    ├─ 小児科（約6,000）
  │    │    ├─ 皮膚科（約5,000）
  │    │    ├─ 眼科（約4,000）
  │    │    ├─ 耳鼻咽喉科（約3,000）
  │    │    ├─ 産婦人科（約3,000）
  │    │    ├─ 泌尿器科（約2,000）
  │    │    └─ その他（精神科、脳神経外科等）
  │    │
  │    └─ 歯科診療所（約68,000施設）→ 歯科は別業種BPO扱い
  │
  ├─ 介護（約14万事業所）
  │    ├─ 居宅サービス
  │    │    ├─ 通所介護（デイサービス）（約44,000）← 最多。地域密着型含む
  │    │    ├─ 訪問介護（約35,000）
  │    │    ├─ 訪問看護（約15,000）← 急増中。医療保険との併用あり
  │    │    ├─ 通所リハビリ（デイケア）（約8,000）← 医療機関併設
  │    │    ├─ 短期入所（ショートステイ）（約11,000）
  │    │    ├─ 居宅介護支援（ケアマネ事業所）（約39,000）← 独立事業所
  │    │    └─ 福祉用具貸与（約7,000）
  │    │
  │    ├─ 施設サービス
  │    │    ├─ 特別養護老人ホーム（特養）（約10,500）← 要介護3以上
  │    │    ├─ 介護老人保健施設（老健）（約4,200）← 在宅復帰が目的
  │    │    ├─ 介護医療院（約800）← 旧・介護療養型医療施設の転換先
  │    │    └─ 介護付有料老人ホーム（約6,000）← 特定施設入居者生活介護
  │    │
  │    └─ 地域密着型サービス
  │         ├─ グループホーム（認知症対応型共同生活介護）（約14,000）
  │         ├─ 小規模多機能型居宅介護（約5,700）← 通い・泊まり・訪問の複合
  │         ├─ 看護小規模多機能型（約1,000）← 上記+訪問看護
  │         └─ 定期巡回・随時対応型（約1,100）← 24時間対応
  │
  └─ 障害福祉（約13万事業所）
       ├─ 就労継続支援A型/B型（約30,000）
       ├─ 放課後等デイサービス（約20,000）← 急増
       ├─ グループホーム（約15,000）
       ├─ 生活介護（約10,000）
       └─ その他（居宅介護、相談支援等）
```

**ポイント**: 「医療・福祉」と一括りにされるが、病院とクリニック、特養とデイサービス、介護と障害福祉では、根拠法も報酬体系も運営文化も全く異なる。「医療・福祉に詳しい」と言うためには、最低でも施設種別ごとの違いを説明できなければならない。

### 1.2 施設種別の詳細

#### 通所介護（デイサービス）

最も事業所数が多い介護サービス。自宅で生活する高齢者が日中に施設へ通い、入浴・食事・機能訓練・レクリエーションを受ける。

```
■ 施設数: 約44,000事業所（地域密着型含む）
■ 1事業所あたり利用者: 10-50名/日
■ 職員数: 5-30名（パート含む）
■ 運営主体: 株式会社（約40%）、社会福祉法人（約30%）、NPO（約15%）
■ 月間介護報酬: 200-800万円（利用者数・加算取得状況による）
■ 利用者の平均要介護度: 2.2

■ 規模区分:
  地域密着型（定員18名以下）: 約24,000
  通常規模型（定員19名以上、月平均750名以下）: 約16,000
  大規模型I（月平均751-900名）: 約3,000
  大規模型II（月平均901名以上）: 約1,000
  → 規模区分によって基本報酬の単位数が異なる（大規模ほど単位数が低い）

■ 利益率: 平均3-5%。赤字事業所が約4割
  収入の95%以上が介護報酬（公定価格）
  コストの60-70%が人件費
  → 利益を出すには「加算を最大限取る」か「人件費比率を下げる」しかない
```

#### 訪問介護

ホームヘルパーが利用者の自宅を訪問し、身体介護（入浴・排泄・食事介助）や生活援助（掃除・洗濯・調理）を行う。

```
■ 施設数: 約35,000事業所
■ 1事業所あたりヘルパー数: 5-30名（登録ヘルパー含む）
■ サービス提供責任者（サ責）: 利用者40名につき1名以上
■ ヘルパーの平均年齢: 54.4歳（介護労働安定センター）
■ 登録ヘルパーの比率: 約60%（非常勤・パート）

■ 報酬構造:
  身体介護（20分以上30分未満）: 250単位
  身体介護（30分以上1時間未満）: 396単位
  生活援助（20分以上45分未満）: 183単位
  生活援助（45分以上）: 225単位
  → 身体介護の方が単位数（=報酬）が高い

■ 課題:
  移動時間が無報酬（利用者宅→次の利用者宅の移動は報酬に含まれない）
  ヘルパーの高齢化が深刻。60歳以上が約3割
  「買い物代行」「調理」はAI化しにくい。記録・請求業務がBPO対象
```

#### 特別養護老人ホーム（特養）

要介護3以上の高齢者が入所する介護施設。終の棲家（ついのすみか）として看取りまで対応する施設が増加。

```
■ 施設数: 約10,500施設
■ 定員: 50-100名（ユニット型: 10名×ユニット数）
■ 職員数: 50-150名（看護・介護・栄養士・相談員・事務）
■ 入所者の平均要介護度: 3.9
■ 平均在所日数: 約1,500日（約4年）
■ 月間介護報酬: 1,000-3,000万円
■ 運営主体: 社会福祉法人がほぼ全て（株式会社は参入不可）

■ 人員配置基準（法定最低基準）:
  介護・看護職員: 利用者3名につき1名以上（3:1配置）
  → 入所者100名なら常勤換算34名以上
  看護職員: 入所者30名超で常勤1名以上
  生活相談員: 100名につき1名以上
  機能訓練指導員: 1名以上
  栄養士: 1名以上（入所定員40名以上）
  介護支援専門員（ケアマネ）: 100名につき1名以上

■ ユニット型とは:
  10名前後の小グループで生活するスタイル
  1ユニットにリビング・キッチン・個室
  職員は固定配置（馴染みの関係を重視）
  基本報酬がユニット型の方が高い
  → 2025年以降、新設は原則ユニット型
```

#### 介護老人保健施設（老健）

病院と自宅の中間施設。在宅復帰を目的としたリハビリを提供。医師が常勤で配置されている点が特養との最大の違い。

```
■ 施設数: 約4,200施設
■ 定員: 50-200名
■ 在宅復帰率: 施設による（50%以上で「超強化型」の加算）
■ 平均在所日数: 約300日（特養より短い。在宅復帰が目標だから）
■ 運営主体: 医療法人が約80%

■ 人員配置基準:
  医師: 常勤1名以上（100名につき1名以上）← 特養との最大の違い
  看護職員: 入所者数の7分の1以上
  介護職員: 入所者数の7分の1以上（看護+介護で3:1以上）
  リハビリ専門職（PT/OT/ST）: 100名につき1名以上
  支援相談員: 100名につき1名以上

■ 在宅復帰・在宅療養支援等指標（2024年改定）:
  超強化型: 90以上 → 基本報酬が最も高い
  在宅強化型: 70以上
  加算型: 50以上
  基本型: 20以上
  その他: 20未満
  → この指標を上げることが経営上の最重要課題
```

#### グループホーム（認知症対応型共同生活介護）

認知症高齢者が5-9名の少人数で共同生活する施設。家庭的な環境で、調理や掃除も利用者が参加する。

```
■ 施設数: 約14,000事業所
■ 定員: 1ユニット5-9名（最大3ユニット=27名）
■ 利用者: 要支援2以上で認知症の診断がある人
■ 職員数: 5-15名（1ユニットあたり日中3名+夜勤1名が目安）
■ 月間介護報酬: 200-500万円

■ 特徴:
  地域密着型サービス → 市区町村が指定・監督
  事務員がいないことが多い → 管理者（施設長）が全部やる
  請求業務も管理者が夜にやっている、というのが珍しくない
  → 小規模ゆえにBPOの恩恵が最も直接的
```

#### 一般診療所（クリニック）

病床19床以下の医療施設。「かかりつけ医」として地域医療の最前線を担う。

```
■ 施設数: 約105,000施設（有床約6,000、無床約99,000）
■ 1施設あたり職員数: 3-20名（院長+看護師2-5名+医事課1-3名+受付1-2名）
■ 外来患者数: 20-100名/日（診療科による）
■ 月間診療報酬: 300-3,000万円

■ 診療科別の特徴:
  内科: 最大ボリューム。慢性疾患の管理が中心。月2回の通院患者が多い
        特定疾患療養管理料（225点×2回/月）が安定収入
  整形外科: リハビリ収入が大きい。運動器リハII 170点/単位
            午前=外来、午後=リハ、のパターンが多い
  皮膚科: 回転が速い。1人5分で次の患者。外用薬の処方が中心
  眼科: 検査が多い（視力、眼圧、眼底、OCT等）。検査収入が大きい
  耳鼻咽喉科: 季節変動が激しい（花粉症シーズンは通常の2-3倍）
  小児科: ワクチン接種が大きな収入源。感染症の波で外来数が乱高下

■ 開設形態:
  個人開業: 約40%。院長の個人資産で開業。廃業=引退
  医療法人: 約50%。法人格で運営。分院展開が可能
  その他（大学病院の分院等）: 約10%
```

### 1.3 収益構造の全体像

医療・福祉の収益は「公定価格」で決まる。製造業やITのように自由に価格を設定できない。国が決めた報酬体系に従い、サービスを提供した対価を請求する。

#### 介護施設の収益構造

```
典型的なP/L構成（通所介護、利用者35名/日、埼玉県の場合）:

  介護報酬収入:     4,800,000円/月（100%）
  ──────────────────────────────────
  人件費:          3,120,000円（65%）
    うち介護職員:   2,000,000円
    うち看護職員:     400,000円
    うち管理者:       350,000円
    うち事務員:       200,000円
    うち調理・送迎:   170,000円
  ──────────────────────────────────
  事業費:            720,000円（15%）
    うち食材費:       300,000円
    うち送迎費（車両）: 150,000円
    うち消耗品・福祉用具: 120,000円
    うち水道光熱費:   150,000円
  ──────────────────────────────────
  事務費:            480,000円（10%）
    うち家賃:         250,000円
    うち保険料:        50,000円
    うちシステム利用料:  80,000円
    うちその他:       100,000円
  ──────────────────────────────────
  減価償却費:        240,000円（ 5%）
  ──────────────────────────────────
  営業利益:          240,000円（ 5%）

  ※ 赤字事業所が約4割。利益率5%は「うまくやっている方」
  ※ 人件費65%は業界平均。70%超えると経営が苦しい
  ※ 処遇改善加算を取ることで人件費を上げつつ報酬を確保する構造
```

### 1.4 組織構造

#### 通所介護（デイサービス）の典型例（職員15名規模）

```
管理者・施設長（1名）
├── 兼: 営業（ケアマネへの営業、見学対応）
├── 兼: 行政対応（実地指導、届出、補助金申請）
├── 兼: 請求業務（事務員がいない場合）
└── 兼: シフト管理、採用

生活相談員（1名）
├── 利用者・家族との相談対応
├── ケアマネとの連絡調整
├── サービス担当者会議への参加
└── 兼: 介護業務（人が足りないから現場にも入る）

看護職員（1名）
├── バイタル測定（血圧・体温・脈拍）
├── 服薬管理・与薬
├── 健康状態の観察・記録
├── 医療的ケア（インスリン注射、経管栄養等）
└── 兼: 機能訓練指導員（PT/OTがいない場合）

介護職員（6-8名）
├── 入浴介助（2名体制が基本）
├── 食事介助
├── 排泄介助
├── レクリエーション
├── 機能訓練（体操、歩行訓練等）
├── 送迎（運転+介助で2名体制）
└── 記録業務（1日1.5時間/人。手書き or タブレット）

事務員（0-1名）
├── 介護報酬の請求
├── 利用者台帳の管理
├── 経理・給与計算
└── 電話対応
    ※ いない場合は管理者が全部やる。これが「施設長が帰れない」原因

調理員（1-2名）
├── 昼食の準備（40-50食/日）
├── おやつの準備
└── 食材の発注・在庫管理

送迎ドライバー（1-2名）← 兼: 介護職員であることが多い
├── 朝の迎え（8:30-9:30）
├── 夕方の送り（16:00-17:00）
└── ルート管理（10-15名を2-3台で送迎）
```

**ポイント**: 通所介護では「1人=2-3役」が当たり前。生活相談員が介護業務に入り、看護職員が機能訓練を担い、管理者が請求業務をやる。「専任」という概念がほとんどない。

#### 内科クリニックの典型例（職員10名規模）

```
院長（1名）
├── 外来診療（午前40名+午後30名=70名/日）
├── カルテ記載
├── 紹介状・主治医意見書の作成
├── レセプト最終チェック
├── 経営判断（全て院長の一存）
└── 院長の離脱 = クリニックの停止

看護師（3名）
├── 問診（患者の症状を聞き取り、院長に伝える）
├── 検査（採血、心電図、レントゲン補助）
├── 処置（注射、点滴、創傷処理の補助）
├── 在宅患者への訪問看護（在宅医療をやっている場合）
└── 兼: 院長の秘書的役割

医事課（2名）
├── 受付・会計
├── レセプトコンピュータ（レセコン）への入力
├── レセプト点検
├── 施設基準の管理（書類の整備）
├── 公費の確認（生活保護、自立支援、乳幼児医療等）
└── 院長が最も困るのは「ベテラン医事課スタッフの退職」

受付（1名）
├── 電話対応、予約管理
├── 保険証の確認
└── 患者案内
```

### 1.5 医療・福祉の文化

#### 「制度ビジネス」という本質

医療・福祉の最大の特徴は、収入が全て国の制度で決まること。飲食店なら「美味しい料理を作れば客が来る」が成立するが、介護施設は「どんなに良いサービスをしても、要介護度と提供時間で報酬が決まる」。値上げができない。利益を出すには加算を取るか、コストを下げるしかない。

3年ごとの報酬改定は、業界全体の事業計画をリセットする。「今年取れていた加算が来年はなくなる」「新しい加算の要件を満たすために設備投資が必要」。制度改正への対応が遅れた施設は淘汰される。

#### 「お世話」の文化

介護の現場では「効率化」という言葉は嫌われる。「利用者は人間であって、製品ではない」という感覚が根強い。タブレットで記録をつけている間に利用者が転倒するかもしれない。「記録より目の前の利用者を見ろ」という空気がある。

一方で、記録業務に追われて利用者と向き合う時間がないのも事実。この矛盾を解消するのがBPOの価値だが、「効率化」ではなく「利用者と向き合う時間を増やす」と言い換えないと受け入れられない。

#### 紙文化の根強さ

```
まだ紙が主流のもの:
  ・バイタルチェック表（手書きで血圧・体温を記入）
  ・介護記録（手書きノート → 後でソフトに転記）
  ・連絡帳（利用者の家族への日々の報告。手書き）
  ・排泄チェック表（トイレの回数・量・性状を記録）
  ・食事摂取量の記録（主食○割、副食○割）
  ・ヒヤリハット報告書（紙で書いてファイリング）
  ・研修記録（参加者名と内容を紙で保管）
  ・避難訓練記録（実地指導で必ず確認される）

IT化が進んでいるもの:
  ・介護報酬の請求（CSVで国保連に伝送。ほぼ100%電子化）
  ・レセプト（レセコンはほぼ全施設に導入済み。電子請求義務化）
  ・給与計算（弥生・freee等）

中間地帯（施設による）:
  ・介護記録ソフト（カイポケ、ほのぼの等。導入率50-60%）
  ・タブレット記録（導入率30%程度。導入しても使いこなせていないことが多い）
  ・電子カルテ（クリニックの導入率50%程度。紙カルテの施設もまだ多い）
```

#### 変化への慎重さ（介護特有の理由）

製造業の「今までこれでやってきた」と同じだが、介護には固有の理由がある。

- **利用者の安全が最優先**: 新しいシステムの導入で現場が混乱すれば、利用者の事故につながる。「試しに入れてみる」ことのリスクが大きい
- **高齢の職員が多い**: 介護職員の平均年齢は47.4歳。60代以上も15%いる。タブレットの操作に時間がかかる
- **現場に余裕がない**: IT研修をやる時間も人もない。「午後2時間、全員でシステム研修」はシフト的に不可能
- **制度が変わりすぎる**: 3年ごとの報酬改定のたびにソフトのバージョンアップが必要。「また変わるの？」という疲労感

#### 「院長教」の世界

クリニックは院長の独裁国家に近い。診療方針、人事、経費、全てが院長の一存。「院長がいいと言えばいい」「院長が嫌がったら導入できない」。逆に、院長を味方につければ即日導入もあり得る。

ただし院長は多忙。外来中は電話に出られない。説明は5分以内。「院長、この製品で月○万円の増収が見込めます」の一言が刺さるかどうかで決まる。

## 2. 介護報酬の仕組み（★最重要セクション）

### 2.1 介護報酬の全体フロー

介護報酬は「単位」で計算される。1単位の金額はサービス種別と地域によって異なる。

```
サービス提供
  → サービスコードの決定
  → 基本単位数の算出
  → 加算の適用
  → 減算の適用
  → 処遇改善加算の適用
  → 金額換算（単位 × 地域単価）
  → 利用者負担の算出（1割/2割/3割）
  → 保険給付額の算出
  → 国保連へ請求（毎月10日まで）
```

### 2.2 サービスコードとは

介護保険のサービスは全て「サービスコード」（6桁）で管理される。サービスコードが決まれば単位数が決まる。

```
サービスコード体系:
  先頭2桁: サービス種類
    11: 訪問介護
    12: 訪問入浴
    13: 訪問看護
    14: 訪問リハ
    15: 通所介護
    16: 通所リハ
    21: 短期入所（福祉型）
    23: 短期入所（医療型）
    32: グループホーム
    51: 特養
    52: 老健
    ...

  3-6桁: サービス内容の詳細（要介護度、時間区分、施設規模等）

例: サービスコード「151131」
  15: 通所介護
  1131: 通常規模型/7時間以上8時間未満/要介護3
  → 単位数: 896単位
```

### 2.3 基本報酬の単位数（通所介護の例）

| 要介護度 | 3-4時間 | 4-5時間 | 5-6時間 | 6-7時間 | 7-8時間 | 8-9時間 |
|---|---|---|---|---|---|---|
| 要介護1 | 368 | 386 | 567 | 581 | 655 | 666 |
| 要介護2 | 421 | 442 | 670 | 686 | 773 | 787 |
| 要介護3 | 477 | 500 | 773 | 792 | 896 | 911 |
| 要介護4 | 530 | 557 | 876 | 897 | 1,018 | 1,036 |
| 要介護5 | 585 | 614 | 979 | 1,003 | 1,142 | 1,162 |

**読み方**: 要介護3の利用者が7時間以上8時間未満の通所介護を利用すると、基本報酬は896単位。これに各種加算を足し、地域単価を掛けて金額にする。

### 2.4 加算の全体像

加算とは、一定の要件を満たした場合に基本報酬に上乗せされる報酬のこと。加算を取れるかどうかで施設の収入は大きく変わる。

#### 処遇改善加算（2024年改定後の一本化版）

旧・処遇改善加算+特定処遇改善加算+ベースアップ等支援加算が一本化された。

```
■ 通所介護の加算率:
  加算I:   9.4%
  加算II:  8.6%
  加算III: 7.0%
  加算IV:  5.5%
  加算V(1): 4.0%
  加算V(2): 2.2%

■ 訪問介護の加算率:
  加算I:   24.5%
  加算II:  22.4%
  加算III: 18.2%
  加算IV:  13.7%
  加算V(1): 10.0%
  加算V(2):  5.5%
  → 訪問介護は人件費比率が高いため加算率も高い

■ 算定要件:
  キャリアパス要件I:   職位・職責に応じた賃金体系の整備
  キャリアパス要件II:  資質向上の計画策定・研修の実施
  キャリアパス要件III: 経験・資格に応じた昇給の仕組み
  職場環境等要件: 6区分24項目から年度ごとに1以上実施
  見える化要件: 賃金改善の取組をHP等で公表

■ 計算フロー:
  月間の対象サービス総単位数 × 加算率 = 処遇改善加算単位数
  例: 総単位数50,000単位 × 9.4% = 4,700単位
  → 4,700単位 × 10.84円（4級地）= 50,948円/月の追加収入
  → 年間約61万円。これを職員の賃金に充てる義務がある

■ 提出書類:
  計画書（年度初め。賃金改善計画+キャリアパス+職場環境の取組）
  実績報告書（年度末。実際の賃金改善額の報告）
  → これを出し忘れると加算が取り消される。年間数百万円の損失
```

#### サービス提供体制強化加算

```
  I:  22単位/日（介護福祉士70%以上 or 勤続10年以上が25%以上）
  II: 18単位/日（介護福祉士50%以上）
  III: 6単位/日（介護福祉士40%以上 or 常勤職員75%以上 or 勤続7年以上30%以上）

  計算例（利用者30名/日、月22日稼働の場合）:
    加算I: 22単位 × 30名 × 22日 = 14,520単位/月
    → 14,520 × 10.84円 = 157,397円/月 ≒ 189万円/年
```

#### その他の主要加算（通所介護の場合）

| 加算名 | 単位数 | 算定要件（概要） | 年間収入目安 |
|---|---|---|---|
| 個別機能訓練加算Iイ | 56単位/日 | 機能訓練指導員1名以上配置+個別計画 | 約44万円 |
| 個別機能訓練加算Iロ | 85単位/日 | 専従の機能訓練指導員1名以上 | 約67万円 |

| ADL維持等加算I | 30単位/月 | ADL利得が1以上+LIFE提出 | 約12万円 |
| ADL維持等加算II | 60単位/月 | ADL利得が2以上+LIFE提出 | 約23万円 |
| 認知症加算 | 60単位/日 | 認知症の利用者が一定割合+専門研修修了職員 | 約47万円 |
| 中重度ケア体制加算 | 45単位/日 | 要介護3以上が30%以上+看護職員配置 | 約36万円 |
| 入浴介助加算I | 40単位/日 | 入浴介助を行った場合 | 約32万円 |

**加算の取り漏れの実態**: 要件を満たしているのに申請していない加算は「未取得加算」と呼ばれ、1施設あたり年間50-200万円の逸失利益がある。特に処遇改善加算の区分判定ミス（IVで算定しているが実はIが取れる）は大きい。

### 2.5 減算ルール

減算は加算の反対で、一定の基準を満たさない場合に報酬が減額されるもの。

```
■ 人員基準欠如減算: 所定単位数の70%に減算
  条件: 人員配置基準を満たしていない場合
  → 通所介護: 利用者15名まで介護職1名+看護職1名以上
             利用者15名超は5名増ごとに介護職1名追加
  → 特養: 利用者3名につき介護・看護職1名以上
  → 1日でも基準を割ると、その月全体が70%減算
  → 影響: 月間報酬400万 × 30% = 120万円の損失

■ 定員超過減算: 所定単位数の70%に減算
  条件: 利用定員の105%を超えた場合
  → 定員30名の施設で32名以上受け入れた日がある場合

■ 身体拘束減算: 所定単位数の90%に減算（10%減算）
  条件: 身体拘束適正化の取組が不十分な場合
  → 委員会の未開催、指針の未整備、研修の未実施

■ 送迎未実施減算: △47単位/片道
  条件: 送迎を行わなかった場合

■ 同一建物減算:
  事業所と同一建物の利用者: △94単位/日
  事業所と同一建物以外の20人以上の利用者: △188単位/日
```

### 2.6 区分支給限度基準額

介護保険には「1ヶ月に使えるサービスの上限」がある。要介護度ごとに決まっている。

| 要介護度 | 限度額（単位/月） | 金額目安（1単位10円換算） |
|---|---|---|
| 要支援1 | 5,032 | 約50,320円 |
| 要支援2 | 10,531 | 約105,310円 |
| 要介護1 | 16,765 | 約167,650円 |
| 要介護2 | 19,705 | 約197,050円 |
| 要介護3 | 27,048 | 約270,480円 |
| 要介護4 | 30,938 | 約309,380円 |
| 要介護5 | 36,217 | 約362,170円 |

```
超過分の扱い:
  限度額内: 1-3割の自己負担（残りは保険給付）
  限度額超過分: 全額自己負担
  → 限度額ギリギリの利用者が多い。1単位でも超えると自己負担が跳ね上がる
  → ケアマネが「限度額管理」に神経を使う最大の理由

例: 要介護3の利用者（限度額27,048単位）
  通所介護: 896単位 × 12回 = 10,752単位
  訪問介護: 396単位 × 12回 = 4,752単位
  訪問看護: 821単位 × 4回 = 3,284単位
  福祉用具貸与: 350単位/月
  合計: 19,138単位 → 限度額内（残り7,910単位）
  → この「残り」をどう使うかがケアプランの腕の見せどころ
```

| 地域区分 | 1単位あたり | 主な地域 |
|---|---|---|
| 1級地 | 11.40円 | 東京23区 |
| 2級地 | 11.12円 | 東京都特別区の一部 |
| 3級地 | 11.05円 | 大阪市、横浜市、名古屋市 |
| 4級地 | 10.84円 | さいたま市、千葉市、京都市 |
| 5級地 | 10.70円 | 政令指定都市の一部 |
| 6級地 | 10.42円 | 中核市等 |
| 7級地 | 10.21円 | 一般市等 |
| その他 | 10.00円 | 上記以外 |

**注意**: 地域単価はサービス種別によっても異なる。上記は「通所介護」の場合。訪問介護は人件費比率が高い分、地域差が大きい（1級地で11.86円、その他で10.00円）。

### 2.8 介護報酬の算出フロー（完全版計算例）

```
■ 前提条件:
  施設: 通所介護（通常規模型）、埼玉県さいたま市（4級地）
  利用者: 山田太郎さん、要介護3
  サービス: 7時間以上8時間未満
  処遇改善加算: I（9.4%）
  自己負担: 1割

■ 計算フロー:

  ① 基本報酬:
     通所介護/通常規模/要介護3/7-8時間 = 896単位

  ② 加算の加算:
     入浴介助加算I:     40単位
     送迎加算（往復）:   47 × 2 = 94単位
     個別機能訓練加算I:  56単位
     サービス提供体制強化加算I: 22単位
     ─────────────────────
     加算合計:           212単位

  ③ 小計（処遇改善加算の計算対象）:
     896 + 212 = 1,108単位

  ④ 処遇改善加算:
     1,108 × 9.4% = 104.15 → 104単位（端数切捨て）

  ⑤ 総単位数:
     1,108 + 104 = 1,212単位

  ⑥ 金額換算:
     1,212 × 10.84円（4級地）= 13,138.08 → 13,138円

  ⑦ 利用者負担（1割）:
     13,138 × 10% = 1,313.8 → 1,314円（端数切上げ）

  ⑧ 保険給付額:
     13,138 - 1,314 = 11,824円 → 国保連に請求

  ■ この利用者が月12回利用した場合:
    月間請求額: 11,824 × 12 = 141,888円（保険給付）
    利用者負担: 1,314 × 12 = 15,768円
    → 施設のこの利用者からの月間収入: 157,656円
```

### 2.9 請求事務の実際

```
【月次サイクル】

■ 月末〜月初（1-5日）: 実績集計
  全利用者のサービス実績を確認
  加算の算定要件を1件ずつチェック
  要介護認定の有効期間を確認（期限切れ=請求不可）
  → この作業に事務員1人で2-5日かかる

■ 月初（3-8日）: 請求データ作成
  介護ソフトで請求データを作成
  利用者ごとのサービス内容・単位数を確認
  加算の算定可否を最終判断
  → 管理者 or 事務員が介護ソフトとにらめっこする日々

■ 10日（法定期限）: 国保連に伝送
  CSV形式で国保連（国民健康保険団体連合会）に電子請求
  → 1分でも遅れると翌月扱いになる。入金が2ヶ月遅れる
  → 毎月10日が近づくと事務員の顔が曇る

■ 翌月中旬: 審査結果通知
  国保連が審査。問題なければ「審査支払済」
  問題があれば「返戻」（差し戻し）

■ 翌々月末: 入金
  審査通過分が口座に入金（サービス提供月の2ヶ月後）
  例: 3月分のサービス → 4月10日に請求 → 5月末に入金
  → 資金繰りの悪い施設にとって、この2ヶ月のタイムラグはきつい

■ 返戻対応:
  返戻された請求の原因を調査
  修正して再請求（翌月の請求に含める）
  → 返戻率3-5%の施設では、毎月数件の返戻対応が発生
  → 原因究明に1件あたり30分-1時間。地味に時間がかかる
```

## 3. 診療報酬の仕組み

### 3.1 診療報酬の基本構造

診療報酬は「点数」で計算される。1点=10円。

```
診療行為
  → 点数表から該当する点数を選択
  → 合計点数を算出
  → 金額換算（点数 × 10円）
  → 患者負担の算出（1-3割）
  → 保険者に請求（毎月10日まで）
```

### 3.2 外来診療の流れと算定

```
■ 初診の場合:
  受付（保険証確認）
  → 問診（看護師 or 医師）
  → 診察（医師）
  → 検査指示（採血、レントゲン等）
  → 処方（院外処方箋）
  → 会計

■ 算定される主な項目:
  初診料: 288点（2,880円）
  → 再診の場合: 再診料 73点 + 外来管理加算 52点 = 125点

  医学管理料:
    特定疾患療養管理料: 225点/回（月2回まで）
    → 対象: 糖尿病、高血圧、脂質異常症、喘息等
    → 200床未満の病院・診療所のみ

    生活習慣病管理料I: 610点/月
    → 対象: 糖尿病、高血圧、脂質異常症
    → 療養計画書の交付が必要

  検査:
    血液学的検査: CBC 21点、CRP 16点
    生化学的検査: HbA1c 49点、血糖 11点、LDL-C 18点
    心電図: 130点（12誘導）
    レントゲン: 85点（胸部正面）+ 診断料68点

  投薬:
    処方箋料: 68点（院外処方の場合）
    薬剤情報提供料: 10点
    → 院外処方が主流（院内処方は調剤の手間がかかる）

  注射:
    皮下・筋肉内注射: 22点 + 薬剤料
    点滴注射: 99点 + 薬剤料
```

### 3.3 主な施設基準と加算（クリニック）

施設基準は「この施設は一定の基準を満たしています」という届出。基準を満たして届け出ると、対応する加算が算定できる。

### 3.4 レセプト点検の実際

```
■ レセプトとは:
  「診療報酬明細書」の通称。患者1人×1ヶ月分の診療内容と請求額をまとめた書類
  例: 3月に5回受診した患者なら、5回分をまとめて1枚のレセプトにする

■ 点検のポイント:
  1. 傷病名の確認
     → レセプトに記載された傷病名と診療行為が対応しているか
     → 例: 高血圧の薬を処方しているのに「高血圧症」の病名がない → 査定
     → 「レセプト病名」: 査定を避けるためにつける病名。疑い病名（○○の疑い）

  2. 算定回数の確認
     → 月に2回までの管理料を3回算定していないか
     → 1日1回の注射を1日2回算定していないか

  3. 併算定の確認
     → 同時に算定できない項目の組み合わせ
     → 例: 特定疾患療養管理料と生活習慣病管理料は同月に併算定不可

  4. 薬剤の適応確認
     → 処方薬の適応症と傷病名が一致しているか
     → 添付文書の用量を超えていないか
     → 禁忌薬（併用してはいけない薬）の処方がないか

■ 点検にかかる時間:
  月1,000件のレセプト → 事務員2名で3-5日
  院長の最終チェック → 5-10時間/月
  → 「レセプト業務は10日が来るたびに憂鬱」という声が多い

■ 返戻・査定:
  返戻: 請求内容に不備があり、差し戻される。修正して再請求
  査定: 請求内容が認められず、減額される。不服があれば再審査請求
  → 返戻率1-3%（クリニック平均）
  → 査定額: 月間請求額の0.5-1.5%が査定で減額される
```

## 4. 人員配置基準と法令

### 4.1 人員配置基準の詳細

人員配置基準は介護保険法に基づく「最低限」の基準。基準を1日でも割ると70%減算の対象になる。

#### 通所介護

```
■ 管理者: 1名（常勤。兼務可能だが管理業務に支障がないこと）
■ 生活相談員: 利用者数にかかわらず1名以上（常勤）
■ 看護職員: 利用定員10名超で1名以上
■ 介護職員:
  利用者15名まで: 1名以上
  利用者16名以上: 15名を超える5名ごとに1名追加
  → 利用者30名なら: 1 + (30-15)/5 = 4名以上
■ 機能訓練指導員: 1名以上（PT/OT/ST/看護師/柔道整復師/あん摩マッサージ師等）
  → 看護師が兼務していることが多い

■ 計算例（利用者35名の場合）:
  管理者: 1名
  生活相談員: 1名
  看護職員: 1名
  介護職員: 1 + (35-15)/5 = 5名以上
  機能訓練指導員: 1名（看護師が兼務可）
  → 最低8名のシフト確保が必要。1人でも欠けると減算リスク
```

#### 特養

```
■ 3:1配置の意味:
  入所者3名につき介護・看護職員1名以上
  → 入所者100名なら常勤換算34名以上
  → 「常勤換算」= パートの勤務時間を常勤の8時間に換算
     例: 週20時間のパート = 常勤換算0.5

■ 看護職員:
  入所者30名超: 常勤1名以上
  入所者50名超: 常勤2名以上
  入所者130名超: 常勤3名以上

■ 夜勤体制:
  入所者25名まで: 1名以上
  入所者26-60名: 2名以上
  入所者61-80名: 3名以上
  入所者81-100名: 4名以上
  → 夜勤は16時間勤務が基本（16:30-翌9:00）
  → 月72時間ルール: 夜勤従事者1人あたり月72時間以下
     → 月72時間 ÷ 16時間 = 月4.5回が上限
```

#### 訪問介護

```
■ サービス提供責任者:
  利用者40名ごとに1名以上（常勤の介護福祉士 or 実務者研修修了者）
  → 利用者80名なら2名以上

■ 訪問介護員:
  常勤換算2.5名以上
  → サービス提供責任者を含む
```

### 4.2 72時間ルール（夜勤体制の要）

特養・老健の夜勤配置は「月平均夜勤時間数」で管理される。

```
■ 計算式:
  月平均夜勤時間数 = 夜勤延べ時間数 ÷ 夜勤職員数

■ 基準:
  1人あたり月72時間以下
  → 超えると人員基準欠如 → 70%減算

■ 計算例（入所者80名の特養、夜勤4名体制の場合）:
  1日あたり夜勤: 16時間 × 4名 = 64時間
  月間（30日）: 64 × 30 = 1,920時間
  夜勤従事者数: 仮に12名がローテーション
  1人あたり: 1,920 ÷ 12 = 160時間 → 72時間の基準を大幅超過

  → 実際は夜勤を月4-5回に調整
  → 夜勤従事者を24名に増やすと: 1,920 ÷ 24 = 80時間
  → ギリギリ。1名でも休むと基準割れのリスク

■ シフト管理の地獄:
  夜勤回数 × 人数 × 休日 × 有給 × 急な欠勤 の組み合わせ最適化
  → 手計算 or Excelで管理しているのが現状
  → 「来月のシフト作りに3日かかる」は普通
```

### 4.3 法令環境

#### 介護保険法（平成9年法律第123号）

```
■ 概要:
  2000年施行。高齢者の介護を社会全体で支える仕組み
  保険者: 市区町村
  被保険者: 第1号（65歳以上）、第2号（40-64歳で特定疾病）

■ 主な条文:
  第27条: 要介護認定（申請→認定調査→一次判定→二次判定→認定）
  第41条: 居宅介護サービス費の支給
  第43条: 区分支給限度基準額
  第73条: 指定居宅サービス事業者の基準
  第77条: 指定の取消（人員基準欠如、不正請求等）

■ 報酬改定サイクル: 3年ごと
  直近: 2024年4月（+1.59%）
  次回: 2027年4月（予定）
  → 改定のたびにルールが変わる。これが最大の事務負担

■ 指定取消の事例（年間50-100件）:
  架空請求（サービスを提供していないのに請求）
  人員基準欠如の放置
  虐待の隠蔽
  → 指定取消 = 廃業。5年間は再指定不可
```

#### 医療法

```
■ 概要:
  医療提供体制の基本法。病院・診療所の開設・管理を規定

■ 主な規定:
  開設許可: 診療所は都道府県知事への届出（病院は許可制）
  管理者: 医師でなければならない
  施設基準: 構造設備、人員配置の基準

■ 医師法第17条:
  「医師でなければ、医業をなしてはならない」
  → AIは「補助ツール」。最終判断は必ず医師が行う
  → 全ての機能にHuman-in-the-Loopを組み込む必要がある
```

#### 3省2ガイドライン（医療情報の安全管理）

```
■ ガイドライン1: 厚労省
  「医療情報システムの安全管理に関するガイドライン」第6.0版（2023年）
  → 医療情報を扱う全てのシステムに適用
  → 主な要件:
    ・アクセス制御（誰が何にアクセスできるか）
    ・監査ログ（誰がいつ何をしたかの記録）
    ・暗号化（通信・保管時のデータ暗号化）
    ・バックアップ（定期的なデータバックアップ）
    ・外部保存（クラウド利用時の日本国内保存要件）

■ ガイドライン2: 経産省・総務省
  「医療情報を取り扱う情報システム・サービスの提供事業者における安全管理ガイドライン」
  → SaaS事業者に求められる要件
  → ISO27001、SOC2準拠が望ましい
  → リスクアセスメントの実施と定期的な見直し

■ 要配慮個人情報（個人情報保護法 第2条第3項）:
  医療・介護で扱うデータの多くは「要配慮個人情報」に該当
  → 要介護度、病名、服薬内容、家族構成、ADL情報、認知症の程度
  → 本人の同意なしに第三者提供不可
  → PII検出・マスキングが必須
```

#### 個人情報保護法の特則

```
■ 医療・介護関係事業者における個人情報の適切な取扱いのためのガイダンス
  → 院内掲示での利用目的の公表
  → 家族への病状説明時の患者同意
  → 介護施設での連絡帳の取り扱い
  → 死亡した利用者の情報の取り扱い
```

## 6. 記録業務の実態

### 6.1 記録の種類

介護施設で必要な記録は膨大。実地指導（行政監査）で確認されるものは全て整備が必要。

```
■ 日常の記録:
  介護記録（日誌）: 利用者ごとの日々のサービス内容、バイタル、特記事項
  食事記録: 食事摂取量、水分量、嗜好の変化
  排泄記録: 回数、量、性状（下痢・便秘等）
  入浴記録: 入浴日、入浴方法（一般浴/機械浴）、皮膚状態
  バイタル記録: 血圧、体温、脈拍、SpO2
  服薬記録: 与薬の確認、副作用の有無

■ ケアプラン関連:
  アセスメント（課題分析）: 入所/利用開始時+更新時
  ケアプラン（施設サービス計画書/居宅サービス計画書）
  個別機能訓練計画: 3ヶ月ごとに更新
  栄養ケア計画: 3ヶ月ごとに更新
  モニタリング記録: 月1回以上

■ 事故・ヒヤリハット:
  事故報告書: 転倒、誤嚥、離設、骨折等が発生した場合
  ヒヤリハット報告: 事故に至らなかったが危険だったケース
  → 事故報告は市区町村への報告義務あり（重大事故の場合）

■ 研修・会議記録:
  身体拘束適正化委員会: 3ヶ月に1回以上
  感染対策委員会: おおむね3ヶ月に1回以上
  事故防止委員会: 月1回
  虐待防止委員会: 年1回以上
  避難訓練記録: 年2回以上
  職員研修記録: テーマ別に年間計画

■ 行政への届出:
  運営規程の変更届
  加算の届出（新規取得・変更時）
  処遇改善加算の計画書・実績報告書
  事故報告書（重大事故）
```

### 6.2 SOAP形式の記録

医療・介護の記録はSOAP形式が標準。

```
S（Subjective: 主観的情報）:
  利用者本人の訴え。
  例: 「今日は膝が痛い」「お腹がすいた」「家に帰りたい」

O（Objective: 客観的情報）:
  スタッフが観察した事実。数値データ。
  例: 「体温36.8℃、血圧142/88、食事主食8割副食5割摂取」
      「右膝に腫脹あり。歩行時に疼痛の訴えあり」

A（Assessment: 評価）:
  S+Oからの判断。
  例: 「右膝の関節症が悪化している可能性。主治医への報告が必要」
      「食欲は安定しているが、水分摂取量が少ない」

P（Plan: 計画）:
  今後の対応方針。
  例: 「明日の通院時に主治医に報告。膝サポーターの使用を検討」
      「水分補給を2時間ごとに声かけで促す」

■ 実際の記録例:
  S: 「最近よく眠れない」と訴えあり
  O: 夜間巡回時、2:00と4:00にベッド上で覚醒確認
     血圧148/92（普段130/80台）、表情はやや疲労感あり
  A: 不眠傾向あり。血圧上昇との関連性あり
  P: 夜間の覚醒パターンを3日間記録。改善なければ主治医に相談
     就寝前のホットミルク提供を試みる

■ 現場の実態:
  SOAP形式で書ける介護職員は全体の50-60%
  「O」だけ書いて「A」「P」が抜ける記録が多い
  → 「入浴しました」「食事8割摂取」だけの記録は、ケアの質の根拠にならない
  → SOAP記録のAI補助は記録の質を劇的に改善する可能性がある
```

### 8.1 主要変更点

### 8.2 LIFE（科学的介護情報システム）

```
■ LIFEとは:
  厚労省が運営するデータベース。介護サービスのアウトカムデータを収集・分析
  → 利用者のADL、栄養状態、口腔機能等のデータを施設が入力
  → 厚労省がフィードバック（同規模施設との比較データ等）を返す
  → このデータ提出が加算の算定要件になっている

■ LIFE関連加算:
  科学的介護推進体制加算: 40単位/月
  ADL維持等加算I/II: 30/60単位/月
  個別機能訓練加算II: 20単位/月
  栄養マネジメント強化加算: 11単位/日
  口腔衛生管理加算II: 110単位/月

■ 現場の声:
  「LIFEへのデータ入力が新たな業務負担」
  「入力項目が多い（Barthel Index、SGA、口腔アセスメント等）」
  「フィードバックの内容が抽象的で、具体的な改善に活かしにくい」
  → LIFE入力の自動化・効率化は大きなBPO需要

■ 提出頻度:
  初回+6ヶ月ごと（ADL維持等加算は3ヶ月ごと）
  → 利用者30名 × 年4-6回 = 年120-180回のデータ入力
  → 1回15分として年30-45時間の業務
```

### 11.1 介護施設の購買行動

```
  自治体の案内・紹介

  2位: 価格（月3万円以下が心理的ハードル）
  3位: 操作の簡単さ（「スタッフが使えるか」が最大の懸念）
```

### 11.2 介護業界の独特な用語・会話

```
■ 会話で使う用語:
  「介護度が上がった」= 状態が悪化した（要介護2→3）。良いことではない
  「更新が切れる」= 要介護認定の有効期間が終わる。更新申請が必要
  「返戻が来た」= 国保連から請求が差し戻された
  「加算を取る/取れる」= 加算の算定要件を満たして届出→請求する
  「人基準」= 人員配置基準
  「定超」= 定員超過
  「実地が来る」= 実地指導（行政監査）が入る
  「ショート」= 短期入所生活介護（ショートステイ）
  「デイ」= 通所介護（デイサービス）
  「サ担」= サービス担当者会議
  「モニタリング」= ケアプランの実施状況の定期確認
  「サ責」= サービス提供責任者
  「包括」= 地域包括支援センター
  「国保連」= 国民健康保険団体連合会（介護報酬の審査・支払い機関）

■ クリニックで使う用語:
  「レセプト」= 診療報酬明細書
  「レセコン」= レセプトコンピュータ
  「査定」= 審査機関に請求が減額されること
  「返戻」= 請求が差し戻されること
  「ORCA」= 日医標準レセプトソフト（オルカ）
  「施設基準」= 加算を算定するための届出
  「主治医意見書」= 要介護認定に必要な医師の意見書
  「紹介状」= 診療情報提供書（他の医療機関への紹介）
  「算定」= 点数を計上すること
  「病名漏れ」= レセプトに必要な傷病名が記載されていない
```

### 11.3 営業時に気をつけること

```
■ やってはいけないこと:
  ・「効率化」を前面に出す → 「利用者は効率化の対象ではない」と反発される
  ・忙しい時間帯に電話する → 午前中の入浴・食事介助の時間は絶対NG
  ・ITの専門用語を使う → 「クラウド」「API」「AIエンジン」は通じない

  ・他施設の成功事例を見せる → 口コミが最強の販促
  ・操作デモを実際のスタッフにやってもらう → 「これなら使える」の体験
  ・無料お試し期間を用意する → リスクなしで試せることが重要
```

```
  音声入力による記録作成（ハナスト等。話した内容をテキスト化）
  → まだ実証段階が多い。本格普及はこれから

■ オンライン診療:
  2020年のコロナ禍で規制緩和
  初診からのオンライン診療が恒久化（2022年）
  → 慢性疾患の定期フォローに適する
  → 地方・過疎地での医療アクセス改善
  → 処方箋の電子化と連動

■ 医師の働き方改革（2024年4月〜）:
  勤務医の時間外労働上限: 年960時間（A水準）
  → タスクシフト（医師→看護師→介護職）の推進
  → 書類業務（紹介状、主治医意見書、診断書）の効率化が急務
  → AI補助による書類作成は「働き方改革」の文脈で訴求できる
```

## 13. 資金繰り・財務の深掘り

### 13.1 介護施設の資金サイクル

```
■ 入金タイミング:
  サービス提供月の2ヶ月後（国保連からの入金）
  例: 3月のサービス → 4/10に請求 → 5月末に入金
  → 開業時は最初の3ヶ月間、介護報酬の入金がない
  → 3ヶ月分の運転資金（人件費+家賃+食材費）が必要

■ 返戻の影響:
  返戻率3%の場合: 月間報酬500万 × 3% = 15万円が翌月以降に持ち越し
  修正・再請求にさらに1ヶ月 → 入金が3ヶ月遅延
  → 年間の一時的な入金遅延: 15万 × 12ヶ月 = 180万円

■ 処遇改善加算の立て替え:
  処遇改善加算は「職員に支払った額」を報告する義務がある
  → 先に職員に賃金UP → 後から加算として入金
  → 立て替え期間のキャッシュフロー負担

■ 設備投資:
  開業費: 3,000万-1.5億円（物件・内装・設備・車両）
  → 社会福祉法人: 施設整備補助金（建設費の3/4まで）
  → 株式会社: 自己資金+銀行融資
  → 日本政策金融公庫の融資（新規開業向け）
```

### 13.2 クリニックの開業費と返済

```
■ 開業費の目安:
  テナント開業（内科）: 5,000万-8,000万円
    内装工事: 1,500-3,000万
    医療機器: 2,000-4,000万（レントゲン、エコー、内視鏡等）
    電子カルテ+レセコン: 200-500万
    その他（家具、消耗品、広告費等）: 500-1,000万

  戸建て開業: 1-1.5億円
    土地+建物: 5,000万-1億
    + テナント開業と同様の設備費

■ 返済:
  借入額5,000万の場合:
    金利1.5%、15年返済 → 月額約31万円
  借入額1億の場合:
    金利1.5%、20年返済 → 月額約48万円

  → 開業から3-5年は返済負担が大きい
  → 月間診療報酬が1,000万を超えると経営が安定する目安
```
