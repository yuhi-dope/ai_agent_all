## 1. 運送業とは何か

### 1.1 業種分類（事業区分と許認可体系）

日本の貨物運送事業は、貨物自動車運送事業法を根拠法として、以下の事業区分に分かれる。

#### 一般貨物自動車運送事業（約5.7万社）

不特定多数の荷主の貨物を、有償でトラック輸送する事業。**許可制**。
「トラック運送会社」と聞いて普通に想像するのがこれ。

| 要件 | 内容 |
|---|---|
| 車両台数 | 営業所ごとに最低5台 |
| 運行管理者 | 29台まで1名、30台ごとに1名追加 |
| 整備管理者 | 営業所ごとに1名 |
| 法令試験 | 経営者が法令試験に合格する必要あり |
| 資金要件 | 概ね1,500万円以上の自己資金 |
| 車庫 | 営業所から直線10km以内（都市部は20km以内） |
| 休憩・睡眠施設 | 乗務員1人あたり2.5m2以上の休憩スペース |

許可取得まで3-6ヶ月。書類不備で何度も差し戻される。行政書士に依頼すると30-50万円。許可後も毎年の事業報告書・実績報告書の提出義務がある（提出期限: 7月10日）。

#### 特定貨物自動車運送事業（約300社）

特定の1社の荷主のみを輸送する事業。大手メーカーの物流子会社に多い。トヨタ輸送、日立物流の子会社など。許可要件は一般貨物と同じだが、荷主が1社に限定される。荷主の物流戦略に完全に従属するため、経営の自由度は低い。

#### 軽貨物自動車運送事業（約20万人）

軽トラック・軽バンで運送する個人事業主。**届出制**（許可不要）。
Amazon Flex、PickGo、ハコベル等のマッチングプラットフォームで稼働する人が急増。

- 車両: 軽トラック（スズキ キャリイ、ダイハツ ハイゼット）or 軽バン（N-VAN、エブリイ）
- 初期投資: 車両代100-200万円+黒ナンバー取得（約1週間）
- 平均年収: 300-450万円。稼ぐ人で600万円。ガソリン代・保険・車両維持費は自腹
- 課題: 労働者ではなく個人事業主。労災保険の対象外（特別加入は可能）。事故リスクが高い

#### 貨物利用運送事業（約2.3万社）

自社でトラックを持たず、他の運送会社を利用して輸送を手配する事業。いわゆる「水屋」「ブローカー」。

- **第一種**: トラック運送の手配のみ（登録制）
- **第二種**: 集荷→幹線輸送→配達を一貫して手配（許可制）

実態は「配車だけして中間マージンを取る」ビジネス。中間マージンは10-20%。荷主からすると「運送会社を探す手間が省ける」利点がある。運送会社からすると「安い仕事を持ってくる」という不満がある。物流の重層下請構造の原因の一つ。

#### 3PL（サードパーティ・ロジスティクス）

荷主の物流業務を包括的に受託する事業形態。単なる「運ぶ」だけでなく、倉庫管理・在庫管理・流通加工・情報システムまで一括で受ける。

- 大手3PL: 日本通運、ヤマトロジスティクス、SGHグローバル・ジャパン、センコー
- 中小の3PL化: 運送会社が倉庫を持ち、保管+配送のセットで提案するケースが増加
- 契約形態: 3-5年の長期契約。年間数千万-数億円。解約しにくい

### 1.2 重層下請構造

運送業の構造的な問題がここにある。

```
荷主（メーカー・小売・商社）
  │ 運賃: 100
  ↓
元請運送会社（大手 or 利用運送）
  │ 運賃: 85-90（中間マージン10-15%）
  ↓
一次下請運送会社
  │ 運賃: 70-80
  ↓
二次下請運送会社（ここが中小の主力）
  │ 運賃: 60-70
  ↓
三次下請運送会社 or 軽貨物個人事業主
    運賃: 50-60
```

荷主が100払っても、実際にトラックを走らせるドライバーの会社には50-70しか届かない。この構造がドライバーの低賃金と長時間労働の根本原因。2024年のトラック運送業の書面化義務（荷主→元請→下請の運賃・条件の書面化）は、この構造を可視化する第一歩。

### 1.3 中小運送会社の財務構造

#### 売上規模

```
車両台数と売上の目安:
  10台:  年商1-2億円
  30台:  年商3-6億円
  50台:  年商5-10億円
  100台: 年商10-20億円
```

#### 利益構造

```
典型的なP/L構成（一般貨物、車両30台の場合）:

  売上高:           500,000千円（100%）
  ──────────────────────────────
  運送原価:
    ドライバー人件費:  200,000千円（40%）★最大のコスト
    燃料費:            75,000千円（15%）
    車両リース料:      50,000千円（10%）
    修繕費:            25,000千円（ 5%）
    保険料:            15,000千円（ 3%）
    高速道路料:        20,000千円（ 4%）
    傭車費:            40,000千円（ 8%）
    その他（タイヤ等）: 15,000千円（ 3%）
  ──────────────────────────────
  運送原価計:        440,000千円（88%）
  粗利益:             60,000千円（12%）
  ──────────────────────────────
  販管費:             55,000千円（11%）
    うち事務員人件費:  20,000千円
    うち配車マン人件費: 10,000千円
    うち営業所費用:    15,000千円
    うちその他:        10,000千円
  ──────────────────────────────
  営業利益:            5,000千円（1%）
```

営業利益率は全体平均で▲0.3%。**半数以上が赤字経営**。黒字の会社でも営業利益率1-3%がやっと。製造業の5%、卸売業の2-3%と比べても極端に低い。

#### なぜ利益率が極端に低いのか

- **労働集約型**: 売上の40%がドライバー人件費。機械化・自動化の余地が小さい
- **燃料費の変動**: 軽油価格が10円/L上がると、30台で月50万円のインパクト。しかし荷主への転嫁交渉が難しい
- **重層下請による運賃の目減り**: 元請から下請に流れるたびに10-15%抜かれる
- **片荷（空車率）**: 行きは荷物を積んで走るが、帰りは空で走る。全国平均39.7%が空車
- **荷主との力関係**: 「嫌なら他の運送会社に頼む」と言われる恐怖。運賃交渉ができない

### 1.4 組織構造

```
■ 車両30台・ドライバー35名の一般貨物運送会社（典型例）:

  社長（1名）
  ├── 営業（大口荷主は社長が直接対応）
  ├── 経営判断（車両購入、荷主との運賃交渉）
  └── 労務問題の最終対応（事故、違反、トラブル）

  配車担当（1-2名）★この会社の心臓
  ├── 全配車の決定権を握る
  ├── 荷主からの依頼受付（電話/FAX/メール）
  ├── ドライバーの稼働状況が全部頭の中にある
  ├── 傭車の手配（知り合いの運送会社に電話）
  └── 帰り荷の確保（水屋やWebマッチングサービスに連絡）

  運行管理者（1名）
  ├── 法令上の選任義務。国家資格「運行管理者試験」合格者
  ├── 出庫前・帰庫後の点呼実施
  ├── 乗務員の健康状態・アルコールチェック確認
  ├── 運行指示書の作成・交付
  └── 事故発生時の報告・対応

  整備管理者（1名）
  ├── 日常点検の確認
  ├── 3ヶ月点検・12ヶ月点検の管理
  ├── 車検の手配（ディーラー or 指定整備工場）
  └── タイヤ・オイル・ブレーキの交換管理

  ドライバー（35名）
  ├── 大型免許保有: 20名（長距離/大型車）
  ├── 中型免許保有: 10名（4t車/地場配送）
  ├── 準中型/普通: 5名（2t車/軽量物）
  ├── フォークリフト資格: 15名
  ├── 危険物取扱者: 3名
  └── 平均年齢: 48歳。最年長62歳、最年少24歳

  事務員（1-2名）
  ├── 請求書の作成（荷主別に運賃計算）
  ├── 入金確認・消込
  ├── 給与計算
  └── 行政への届出書類（事業報告書等）
```

**ポイント**: 配車担当1-2名にこの会社の全ての配送ロジックが集約されている。この人が辞める or 倒れると、文字通り翌日からトラックが走れなくなる。

### 1.5 運送業の文化

#### 「走ってナンボ」の精神

ドライバーは走った分だけ稼ぐ。待機時間は「無駄な時間」。荷積み・荷卸しの手待ち時間が1-3時間あっても、それは「仕事のうち」として黙って待つ文化がある。2024年問題でこの「待機時間」が拘束時間としてカウントされるようになり、荷主に待機時間の削減を要求する必要が出てきた。しかし「荷主に文句を言う」のはタブーだった。

#### 夜型の長距離文化

長距離ドライバーの典型的な1日:

```
PM 6:00  営業所を出発。幹線道路へ
PM 7:00  高速道路に乗る（深夜割引のため0時以降にIC通過が理想）
PM 8:00  途中のSA/PAで夕食
PM 10:00-AM 0:00  サービスエリアで仮眠（大型車エリアは争奪戦）
AM 0:00  深夜割引適用で高速走行再開
AM 5:00  納品先の近くに到着。荷受けの開始を待つ
AM 7:00  荷卸し開始。フォークリフトで1-2時間
AM 9:00  荷卸し完了。帰り荷がなければ空で帰る
         帰り荷があれば積込先に移動
PM 12:00 帰り荷を積んで帰路へ
PM 6:00  営業所に帰庫。点呼
```

24時間の拘束。これが改善基準告示で「最大13時間（例外15時間）」に制限される衝撃の大きさがわかる。

#### 口約束とアナログ

運賃は口約束で決まっていたケースが多い。「いつも通りでよろしく」で、具体的な金額を確認しないまま走ることもある。請求段階で「え、その金額だったっけ」が発生する。配車指示も電話1本。運行指示書を形式的に作成しても、中身が実態と乖離していることがある。

#### 安全は経験で語る

「俺は30年無事故だ」が最大の信用。安全教育は形式的になりがちで、事故が起きてから「なぜ安全教育をしていなかったのか」と問われる。KYT（危険予知トレーニング）の実施記録が残っていない会社が大半。デジタコ（デジタルタコグラフ）の導入率は大手でほぼ100%だが、中小では50-60%。

## 2. 車両の種類と用途

### 2.1 車格（積載量別）

トラックは積載量で大きく4分類。ドライバーに必要な免許が車格で変わる。

| 車格 | 最大積載量 | 車両総重量 | 必要免許 | 車両価格（新車） | 用途 |
|---|---|---|---|---|---|
| 小型（2tクラス） | 2-3t | 5t未満 | 準中型以上 | 400-700万円 | 市内配送、個配 |
| 中型（4tクラス） | 4-5t | 8t未満 | 中型以上 | 700-1,200万円 | 地場配送の主力 |
| 大型（10tクラス） | 10-13t | 25t未満 | 大型 | 1,200-2,000万円 | 幹線輸送の主力 |
| トレーラー | 20-28t | 36t以下 | 大型+けん引 | ヘッド800-1,500万 + シャーシ200-800万 | 海上コンテナ、大量輸送 |

#### 免許制度の変遷（現場への影響）

```
2007年以前に普通免許取得: 8tまで運転可能（「旧普通免許」）
2007年以降: 中型免許の新設。普通免許では4t車に乗れなくなった
2017年以降: 準中型免許の新設。普通免許では2t車にすら乗れなくなった

影響:
  18歳で普通免許を取っても、2tトラックにすら乗れない
  → 運送会社が「準中型免許の取得費用を会社負担」するケースが増加
  → 免許取得支援が採用の条件になっている
  → 取得費用: 準中型 15-20万円、中型 15-25万円、大型 25-40万円
```

### 2.2 架装（ボディ）の種類

トラックの荷台部分。同じ車格でもボディが異なれば用途が全く変わる。

| 架装 | 特徴 | 用途 | 追加コスト（概算） |
|---|---|---|---|
| **平ボディ** | 荷台がフラット。シートで覆う | 建材、鋼材、重量物 | 基本仕様 |
| **バンボディ** | 箱型。雨風を防ぐ | 一般貨物、雑貨、家電 | +50-100万 |
| **ウイング車** | 側面が跳ね上がる。フォークリフトで横から積める | パレット輸送の主力 | +100-200万 |
| **冷凍車** | -25℃まで冷却可能 | 冷凍食品、アイスクリーム | +200-400万 |
| **冷蔵車** | 0-5℃を維持 | チルド食品、乳製品、生花 | +150-300万 |
| **ユニック車** | クレーン付き。自力で荷物を積卸し | 建材、機械、重量物 | +200-500万 |
| **ダンプ** | 荷台が傾斜して荷物を降ろす | 土砂、砕石、産廃 | +100-200万 |
| **タンクローリー** | 液体を運ぶ | 燃料、化学品、食品原料 | +300-800万 |
| **キャリアカー** | 車を運ぶ | 新車配送、中古車輸送 | +200-500万 |
| **パワーゲート** | 荷台後部にリフト | 重量物の人力積卸し | +80-150万 |

**冷凍車・冷蔵車の温度帯管理**:

```
温度帯の区分（食品物流の基本）:
  ドライ（常温）:    15-25℃。加工食品、飲料、菓子
  チルド（冷蔵）:    0-10℃。乳製品、生鮮、惣菜。「定温」とも
  フローズン（冷凍）: -18℃以下。冷凍食品、アイスクリーム
  超低温:           -40℃以下。マグロ、特殊食品

冷凍車のコスト:
  冷凍機の燃料: 1日あたり軽油10-20L
  冷凍機のメンテナンス: 年間30-50万円
  温度記録: HACCP対応で記録・保管義務あり
  温度逸脱: 荷主への報告義務。最悪、全量廃棄
```

### 2.3 危険物車両

消防法に基づく「危険物」を運ぶ車両。専用の装備とドライバー資格が必要。

```
必要な資格:
  ・危険物取扱者（乙種第4類が最も多い。ガソリン・軽油・灯油）
  ・危険物の運搬に関する講習の受講
  ・タンクローリーの場合: 移動タンク貯蔵所としての消防届出

運行の制約:
  ・トンネルの通行制限（危険物積載車両通行禁止のトンネルがある）
  ・市街地での駐停車制限
  ・積載量の制限（指定数量の倍率で決まる）
  ・イエローカード（緊急連絡カード）の携帯義務
```

## 3. 配車の実務

### 3.1 配車の1日の流れ

これが運送会社の「生産計画」に相当する。製造業の生産管理に匹敵する複雑さがある。

```
■ 地場配送会社（車両30台）の配車担当の1日:

PM 5:00（前日）
  翌日の荷主からの配送依頼を集計
  ・荷主A: 食品スーパー5店舗へ配送。時間指定あり（開店前7:00着）
  ・荷主B: ホームセンター3店舗へ配送。パレット建て
  ・荷主C: 工場への建材搬入。ユニック車指定。現場搬入の段取りあり
  ・荷主D: 冷凍食品倉庫間移動。冷凍車指定
  ・臨時: 荷主E「明日急ぎで2パレット追加できない？」

PM 5:30
  ドライバーの稼働状況を確認
  ・田中: 昨日拘束14時間だった → 今日は13時間まで
  ・佐藤: 今月の拘束時間あと30時間しかない → 短距離配車のみ
  ・山田: 大型免許+フォークリフト+ユニック → 荷主C向き
  ・鈴木: 冷凍車担当 → 荷主D確定
  ・新人の高橋: まだ1人で回れない → 横乗り研修中

PM 6:00
  車両の状態を確認
  ・5号車: 明日オイル交換のため午前中使えない
  ・8号車: 昨日ドライバーが「ブレーキの効きが甘い」と報告 → 整備入り
  ・12号車: 車検が来週 → 今週中に代車手配

PM 6:30
  配車パズルを解く
  ・荷主A（5店舗）→ 2台に分割。1号車で3店舗、3号車で2店舗
  ・荷主B（3店舗）→ 7号車で1台。積載量6t、荷量5.5t。ギリギリ
  ・荷主C（ユニック）→ 山田+10号車（ユニック）
  ・荷主D（冷凍）→ 鈴木+15号車（冷凍車）
  ・荷主E（追加）→ 4号車の帰り道に組み込めるか…

PM 7:00
  改善基準告示のチェック（頭の中で計算）
  ・田中: 出庫5:30→帰庫18:00 = 拘束12.5h → OK
  ・佐藤: 出庫6:00→帰庫14:00 = 拘束8h → OK（月間拘束時間にも余裕）
  ・全員、前日の休息時間は11時間以上 → OK

PM 7:30
  帰り荷の確保
  ・荷主Aの配送後、千葉方面で帰り荷がないか
  ・WebKITで検索 → 15時以降に船橋でパレット4枚の案件あり
  ・しかし時間的に厳しい → 空車で帰すか、30分待って積むか
  ・判断: 30分待てば運賃¥35,000。空車で帰れば¥0。待つ

AM 5:00（当日）
  ・配車確認。天候チェック（雨なら時間余裕を見る）
  ・ドライバーの出勤確認（体調不良の連絡がないか）
  ・運行指示書の最終確認

AM 5:30-6:30
  ・出庫前点呼（対面 or IT点呼）
  ・アルコール検知
  ・運行指示書交付
  ・注意事項の伝達（工事区間、荷主の特記事項）

AM 6:30以降
  ・電話対応。荷主からの追加/変更/キャンセル
  ・トラブル対応（渋滞、車両故障、ドライバーの体調不良）
  ・翌日以降の配送依頼の受付・整理
```

### 3.2 運賃体系の全貌

荷主ごとに運賃の計算方法がバラバラ。**これが請求業務を地獄にしている根本原因。**

#### 主要な運賃体系

| 運賃体系 | 計算方法 | 使われるケース | 典型的な運賃例 |
|---|---|---|---|
| **距離制** | 基本運賃 + km単価 × 走行距離 | 長距離幹線輸送 | 東京→大阪 10t車: ¥120,000-150,000 |
| **重量制** | 重量ランク × 距離ランク | 特積み（路線便）、混載 | 500kg×300km: ¥15,000-25,000 |
| **個建て** | 1個（1ケース）あたり単価 × 個数 | 食品・飲料の配送 | 1ケース ¥150-300 × 100ケース |
| **パレット建て** | 1パレットあたり単価 × 枚数 | 倉庫間輸送、定期便 | 1パレット ¥3,000-8,000 |
| **車建て（チャーター）** | 1台あたりの定額 | 建材搬入、引越し、貸切 | 4t車 都内1日: ¥40,000-60,000 |
| **時間制** | 基本料金 + 時間超過料金 | 市内配送、横持ち | 4時間 ¥25,000 + 超過 ¥4,000/h |

#### 追加料金（荷主ごとに適用ルールが異なる）

```
① 時間指定料:       +5-15%（AM着指定、開店前着指定）
② 休日・祝日割増:   +20-30%
③ 深夜・早朝割増:   +30-50%（22:00-5:00）
④ 冷凍冷蔵割増:     +10-30%（冷凍車・冷蔵車使用時）
⑤ 危険物割増:       +20-50%
⑥ 待機料:           ¥1,500-3,000/30分
    → 実際には請求できないケースが非常に多い
    → 「請求したら次から仕事を回してもらえなくなる」恐怖
    → 2024年の標準的な運賃告示で「待機時間料」が明記された
⑦ 燃料サーチャージ: 基準軽油価格との差額を転嫁
    → 計算式: (実勢軽油価格 - 基準価格) × 燃費係数 × 走行距離
    → 基準価格と係数が荷主ごとに異なる。統一されていない
⑧ 附帯作業料:       荷積み・荷卸し作業（ドライバーが行う場合）
    → フォークリフト作業、手積み・手卸し、検品
    → 1パレット ¥200-500、手積み ¥3,000-5,000/h
⑨ 中継料:           中継輸送時の追加コスト
⑩ 横持ち料:         倉庫内の移動（A倉庫→B倉庫）
```

#### 標準的な運賃告示制度（2020年）

国交省が「標準的な運賃」を告示。荷主との運賃交渉の根拠として使える法的な裏付け。

```
告示運賃の例（距離制・10tウイング車）:
  100km:  ¥50,340
  200km:  ¥68,720
  300km:  ¥87,100
  500km:  ¥123,860
  東京→大阪（約500km）: ¥123,860

実勢運賃:
  告示運賃の70-80%で取引されているのが現実
  東京→大阪 10t: ¥85,000-100,000が相場
  → 告示運賃との乖離を「データで見せて」交渉する会社が増えている
```

### 3.3 改善基準告示の全項目（2024年4月改正）

配車の全ての判断はこの制約条件に支配される。**間違えると事業停止命令。**

#### 改正前後の比較

| 項目 | 改正前（～2024年3月） | 改正後（2024年4月～） | 変化の影響 |
|---|---|---|---|
| 1日の拘束時間 | 原則13h、最大16h | 原則13h、最大15h（14h超は週2回まで） | 最大値が1h縮小。週2回制限が追加 |
| 1ヶ月の拘束時間 | 原則293h、最大320h | 原則284h、最大310h（年6回まで） | 月9h縮小。年6回制限が厳格化 |
| 年間の拘束時間 | 3,516h | 3,300h | 年間216h（約27日分）縮小 |
| 休息期間 | 継続8h以上 | 継続11h以上（最低9h、週3回まで） | +3h。事実上、1日2運行が困難に |
| 連続運転 | 4h→30分以上の休憩 | 4h→概ね30分の合計休憩（1回10分以上で分割可） | 分割が明確化 |
| 時間外労働 | 上限なし（36協定で青天井） | 年960h | 初めて上限が設定された |
| 分割休息の特例 | 1日2分割まで | 1日3分割まで（各3h以上、合計10h以上） | 分割は緩和だが合計時間は延長 |

#### 違反した場合の罰則

```
■ 改善基準告示違反（行政処分）:
  初違反: 文書警告 or 車両停止命令（10-30日車）
  再違反: 事業停止命令（最大30日間）
  悪質: 許可取消（事業廃止）

  「日車」= 1台×1日の停止。30日車=3台×10日 or 1台×30日
  車両が止まる = 売上がゼロになる日があるということ

■ 時間外上限（年960h）超過（労基法違反）:
  6ヶ月以下の懲役 or 30万円以下の罰金
  企業名の公表（「ブラック企業」リスト入り）

■ 点呼未実施:
  初違反: 車両停止命令（20日車）
  繰り返し: 事業停止命令

■ 運転日報の未記載・虚偽記載:
  車両停止命令 + 運行管理者の解任命令
```

#### 現場への具体的影響

```
Before（改正前）:
  長距離ドライバーが東京→大阪を1人で走る（片道6-7時間）
  夜出発→早朝着→荷卸し→帰り荷積み→その日のうちに帰社
  拘束時間: 20時間超もあり得た

After（改正後）:
  拘束13時間（例外15時間、週2回まで）
  東京→大阪の往復を1日で完結するのは事実上不可能
  → 中継輸送: 静岡あたりで別のドライバーに交替
  → 宿泊運行: 大阪で1泊して翌日帰る（休息11時間以上）
  → 1人で回していた仕事が2人必要 = コスト増 = 運賃に転嫁が必要
```

### 3.4 配車の「例外」（日常的に発生する）

| パターン | 頻度 | 対応の難しさ |
|---|---|---|
| 荷主からの当日追加依頼 | 毎日 | 全配車の組み直しが発生する可能性 |
| ドライバーの急な欠勤 | 週1-2回 | 代替ドライバーの改善基準告示チェックが必要 |
| 車両故障 | 月1-2回 | 代車手配+積替え+顧客通知 |
| 帰り荷の突発確保 | 毎日 | 時間×位置×車種のマッチング |
| 荷量の変動 | 毎日 | 「10パレットのはずが15」→ 積載超過チェック |
| 渋滞・天候 | 毎日 | ルート変更+到着時間の再計算+顧客連絡 |
| 待機時間の発生 | 毎日 | 荷受け先での1-3時間待ち。拘束時間に加算される |
| 季節繁忙 | 年4-5回 | 年末、お盆、GW、引越しシーズン |

## 4. ドライバーの労働実態

### 4.1 拘束時間の内訳

「運転」は拘束時間の一部にすぎない。

```
■ 地場配送ドライバーの1日（4tウイング車、5件配送）:

  05:30  出勤。出庫前点呼。アルコールチェック。運行指示書受領
  05:50  積込み（倉庫でフォークリフト作業）
  06:30  出発。1件目へ
  07:00  1件目納品（食品スーパー。開店前。待機15分→荷卸し30分）
  08:00  2件目へ移動
  08:30  2件目納品（ドラッグストア。手卸し。30分）
  09:30  3件目へ移動
  10:00  3件目納品（飲食店。路地裏。トラックが入れず手持ちで運ぶ。45分）
  11:15  昼食・休憩（30分。トラックの中でコンビニ弁当）
  11:45  4件目へ移動
  12:30  4件目納品（工場。フォーク荷卸し。20分）
  13:00  5件目へ移動
  13:30  5件目納品（倉庫。パレット検品+荷卸し。45分）
  14:30  帰路
  15:30  帰庫。帰庫後点呼。日報記入
  16:00  退勤

  拘束時間: 10.5時間
    うち運転: 4時間
    うち荷役作業: 3時間（積込み+荷卸し）
    うち待機: 0.5時間
    うち点呼・日報: 0.5時間
    うち休憩: 0.5時間
    うち移動待ち等: 2時間
```

**問題**: 運転はたった4時間なのに、荷役作業と待機で拘束時間が膨らむ。ドライバーの仕事は「運転」だけではない。

### 4.2 手待ち時間（待機時間）の実態

```
手待ち時間の発生パターン:
  ・到着が早すぎた（時間指定の30分-1時間前に着いてしまう）
  ・荷受け先のバースが空いていない（先着のトラックが荷卸し中）
  ・検品に時間がかかる（食品の温度チェック、数量確認）
  ・荷主の準備が遅い（伝票ができていない、出荷指示が出ていない）

平均手待ち時間:
  全国平均: 1時間45分/回（国交省調査）
  食品配送: 2-3時間（バース待ちが深刻）
  建材搬入: 1-2時間（現場の受入準備待ち）

影響:
  ・年間で1ドライバーあたり300-500時間の手待ち → 実質的にタダ働き
  ・改善基準告示では手待ち時間も「拘束時間」にカウント
  ・手待ち時間が長いと、その分運転できる時間が減る → 配送件数が減る
```

### 4.3 荷役作業

ドライバーが荷物の積み下ろしまでやる「運転手兼荷役作業員」が中小の常識。

```
荷役作業の種類:
  ・フォークリフト作業: パレット積み。免許必要。1パレット1-2分
  ・手積み・手卸し: 段ボール1個ずつ。飲料（1ケース10-12kg）を200ケース = 2t以上
  ・検品: 伝票と現物の照合。食品は温度記録も
  ・附帯作業: 棚入れ、陳列（小売店向け配送で発生）

問題:
  ・荷役作業の対価がドライバーの給与に含まれていない（「サービス」扱い）
  ・腰痛が職業病。休業4日以上の労災の32%が「荷物の取扱い」
  ・2024年4月から荷役作業の実態を記録する「荷待ち・荷役時間の記録義務化」
```

### 4.4 ドライバーの処遇

```
年間所得（2024年 賃金構造基本統計調査）:
  大型ドライバー:   平均477万円（全産業平均497万円より▲20万円）
  中小型ドライバー: 平均431万円（全産業平均より▲66万円）
  → 長時間労働なのに低賃金。「きつい・汚い・危険」の3K

年齢構成:
  29歳以下:  10%
  30-39歳:   18%
  40-49歳:   27%
  50-59歳:   30%
  60歳以上:  15%
  → 50歳以上が45%。10年後に大量退職が始まる

離職の理由（上位3つ）:
  1. 長時間労働（拘束時間が長い）
  2. 低賃金（労働の割に安い）
  3. 体力的にきつい（荷役作業、夜間運転）
```

## 5. 傭車の仕組みと商慣習

### 5.1 傭車とは

自社の車両・ドライバーだけでは対応できない荷量を、他の運送会社に委託すること。「外注」に相当する。

```
傭車が発生するケース:
  ・繁忙期で車両が足りない
  ・特殊車両が必要（冷凍車、ユニック車、トレーラー）
  ・配送エリアが遠い（地元以外の配送を頼まれた）
  ・ドライバーの残業上限に達している

傭車比率（目安）:
  平常時: 10-20%
  繁忙期: 30-50%
  →「売上の半分を外に出している」会社も繁忙期にはある
```

### 5.2 傭車の手配方法

```
①  知り合いの運送会社に電話
    → 最も多い方法。配車マンの「人脈」が会社の資産
    → 「田中運送さん、明日4t空いてない？」で決まる

② 水屋（利用運送事業者）に依頼
    → 中間マージン10-20%取られるが、確実に車が出る
    → 急な案件のときの最後の手段

③ 求荷求車サービス（WebKIT、トラボックス等）
    → Web上で荷物と空車をマッチング
    → 長距離の帰り荷探しでよく使われる
    → マッチング手数料: 月額3,000-10,000円 + 案件ごとの手数料

④ 大手3PLの傭車ネットワーク
    → 大手の下請として登録しておくと、案件が流れてくる
    → 運賃は安いが、安定した仕事量が確保できる
```

### 5.3 傭車の商慣習と問題

```
・運賃は口約束が多い。「いつもの金額で」が常態化
・傭車先の品質が不安定（遅刻、荷扱いが雑、ドライバーのマナー）
・自社便の原価と傭車費の比較ができていない（どちらが得かわからない）
・傭車を使いすぎると粗利が消える。使わなすぎると荷主の仕事を断ることになる
・帰り荷のマッチングが属人的。配車マンが辞めると傭車ネットワークが消滅
```

## 6. 燃料コスト管理

### 6.1 燃料費の構造

```
燃料費 = 軽油単価(円/L) × 走行距離(km) ÷ 燃費(km/L)

車格別の燃費目安:
  2tクラス:  8-12 km/L
  4tクラス:  5-8 km/L
  10tクラス: 3-5 km/L
  トレーラー: 2.5-4 km/L

軽油価格の推移:
  2020年:  90-100円/L
  2022年: 130-170円/L（ウクライナ危機）
  2024年: 140-160円/L（補助金縮小で高止まり）
  2025年: 145-165円/L

車両1台あたりの月間燃料費:
  4t車（月間走行5,000km、燃費6km/L）: 5,000÷6×155=¥129,000
  10t車（月間走行8,000km、燃費4km/L）: 8,000÷4×155=¥310,000
  → 30台の運送会社で月間燃料費400-800万円
```

### 6.2 燃料費の管理と削減

```
管理の現実:
  ・給油は提携スタンドでの掛け売り（月末一括支払い）
  ・車両別の給油量は給油カードで追跡可能
  ・しかし「車両別の燃費」を算出している中小は少数
  ・「あの車は燃費が悪い」という感覚はあるがデータがない

削減の手法:
  ①エコドライブ: 急発進・急加速を減らす。5-10%の燃費改善
    → デジタコでドライバーのエコドライブスコアを可視化
  ②ルート最適化: 距離×時間の最適経路。高速道路の使用判断
    → 高速使えば燃費は良いが通行料がかかる。トータルコストで判断
  ③アイドリングストップ: 荷待ち中のエンジン停止
    → 夏場はエアコンのために止められない問題
  ④タイヤ空気圧管理: 適正空気圧で2-3%の燃費改善
  ⑤車両の更新: 最新の大型車は10年前と比べて20-30%燃費向上
```

## 7. Gマーク制度

### 7.1 概要

全日本トラック協会が認定する「安全性評価事業」。正式名称は「貨物自動車運送事業安全性評価事業」。取得率は全事業者の約30%。

```
Gマーク取得のメリット:
  ・大手荷主の入札条件（Gマークなしでは入札参加不可のケースあり）
  ・保険料の割引（1-3%程度）
  ・IT点呼の対象要件（Gマーク取得事業所はIT点呼が可能）
  ・信頼性の証明（採用活動にもプラス）

デメリット:
  ・取得・維持のための書類整備が大変（安全教育記録、事故記録、点呼記録等）
  ・更新が2年ごと。更新時に再評価
```

### 7.2 評価基準（38項目、100点満点）

```
■ 安全性に対する法令の遵守状況（配点40点）
  ・点呼の実施状況
  ・運転日報の記録状況
  ・運行管理体制
  ・過労運転の防止措置
  ・整備管理体制

■ 事故や違反の状況（配点40点）
  ・過去3年間の事故件数
  ・行政処分の有無
  ・重大事故の有無
  → 事故ゼロが最高点。1件でも重大事故があると大幅減点

■ 安全性に対する取組の積極性（配点20点）
  ・安全教育の実施状況（年間計画+実施記録）
  ・安全ミーティングの開催
  ・デジタコ・ドライブレコーダーの導入
  ・KYT（危険予知トレーニング）の実施
  ・健康管理（健康診断受診率100%、SAS検査等）

合計80点以上で認定。
```

## 8. IT導入の現状

### 8.1 主なシステムとツール

| ツール | 概要 | 普及率（中小） | 価格帯 |
|---|---|---|---|
| **デジタコ** | 運転データ（速度/距離/時間）の自動記録 | 50-60% | 1台 5-15万円+月額1,000-3,000円 |
| **ドライブレコーダー** | 前方・車内カメラの映像記録 | 70-80% | 1台 2-10万円 |
| **GPS動態管理** | 車両のリアルタイム位置追跡 | 30-40% | 1台 月額1,500-5,000円 |
| **配車システム** | 配車計画の作成支援 | 10-15% | 月額 3-30万円 |
| **勤怠管理** | ドライバーの出退勤・拘束時間管理 | 20-30% | 月額 1-5万円 |
| **運送管理システム（TMS）** | 受注→配車→運行→請求の一気通貫 | 5-10% | 月額 10-50万円 |
| **IT点呼** | 遠隔地での点呼をカメラ+アルコール検知器で実施 | 10-15% | 月額 3-10万円 |

**デジタルタコグラフ（デジタコ）の詳細**:

```
記録データ:
  ・走行速度、走行距離、走行時間
  ・エンジン回転数、急加速・急減速の回数
  ・連続運転時間、休憩時間
  ・位置情報（GPS付きモデル）

メーカー:
  矢崎エナジーシステム（DTG7/DTG8）←シェアNo.1
  YAZAC-eye（クラウド連携型）
  デンソー（DTS-D2M）
  富士通（DTS-G1）

義務化の範囲:
  車両総重量7t以上 or 最大積載量4t以上 → タコグラフ装着義務
  → アナログタコグラフ（チャート紙式）も可だが、実質デジタコに移行中
```

### 8.2 なぜIT導入が進まないのか

```
理由1: 初期投資の負担
  30台にGPS+デジタコを導入 → 初期費用300-500万円
  営業利益率1%の会社には重い投資

理由2: ドライバーの抵抗
  「監視されている」という感覚
  特にベテランドライバーは「俺の運転を信用していないのか」

理由3: 導入しても使いこなせない
  配車システムを入れたが、結局ホワイトボードの方が速いと感じる
  → 「入れたけど使っていない」パターンが非常に多い

理由4: データ入力の負担
  日報、点呼記録、車両点検記録…。入力する人がいない
  運行管理者が1人で全部やっている会社では、システム入力の時間がない

理由5: ドライバーの年齢
  平均年齢48歳。スマートフォンの操作に苦手意識がある人が一定数
  「アプリで日報を入力してください」→ 「紙の方が速い」
```

```
1月:  年始で荷動き少ない。正月明けの反動で中旬から回復

3月:  引越しシーズン開始。年度末で企業の物流も増加。繁忙期
4月:  引越しピーク。新年度の納品ラッシュ。傭車比率が最大に
5月:  GWで物流が止まる。GW明けの集中出荷
6月:  梅雨。建材搬入が天候で遅れる。冷凍食品の需要増
7月:  お中元シーズン。飲料の出荷量が年間ピーク
8月:  お盆期間は物流停止。お盆前後に集中出荷
9月:  台風シーズン。災害時の物流寸断リスク
10月: 年末商戦の仕込み。倉庫への入庫が増加
11月: 年末商戦。食品・日用品の出荷量が増加
12月: 最繁忙期。年末の駆け込み出荷。ドライバーの残業が限界に
```

## 11. 運送業の経営者が使う指標

提案・会話で使うと「この人はわかっている」と思ってもらえるKPI。

| 指標 | 計算方法 | 目安 |
|---|---|---|
| 実車率 | 実車走行距離 ÷ 総走行距離 | 60-65%が平均。70%超で優秀 |
| 積載効率 | 実積載量 ÷ 最大積載量 | 50-60%が平均。80%超で優秀 |
| 車両稼働率 | 稼働日数 ÷ 営業日数 | 85-90%が目標 |
| ドライバー1人あたり売上 | 月間売上 ÷ ドライバー数 | 月100-150万円 |
| 燃料費率 | 燃料費 ÷ 売上 | 15-20%。20%超は要改善 |
| 傭車比率 | 傭車費 ÷ 売上 | 10-20%が平常。30%超は自社便不足 |
| 事故率 | 事故件数 ÷ 車両台数 × 100 | 5%以下が目標 |
| 定時率 | 時間指定通りに納品した割合 | 95%以上が荷主の期待 |
| ドライバー離職率 | 年間退職者 ÷ 平均在籍数 | 10-15%。20%超は要注意 |
