### 1.2 建設業の29業種

建設業法では29業種が定められている。1社が複数の業種許可を持つことが一般的。

| 区分 | 代表的な業種 | 許可業者数（概数） | 特徴 |
|---|---|---|---|
| 総合建設（土木） | 土木一式 | 約131,889業者 | 公共工事の標準積算が適用。道路・橋梁・トンネル |
| 総合建設（建築） | 建築一式 | 約143,593業者 | 住宅・ビル・工場。積算項目が多く複雑 |
| 躯体系 | とび・土工・コンクリート | 約183,700業者（最多） | 高所作業が多い。労災リスクが最大 |
| 仕上系 | 内装仕上、塗装、左官、タイル | 約12万業者 | 仕上がりは目視品質が重要。写真記録が命 |
| 設備系 | 電気工事、管工事、消防施設 | 約15万業者 | 資格要件が厳しい。専門性が高い |
| 専門土木系 | 舗装、造園、さく井、しゅんせつ | 約5万業者 | 数量が明確で標準化しやすい |

#### 業種別の日常

**土木一式**: 道路、下水、河川、橋。公共工事が大半。国交省の標準歩掛に基づく積算。設計書PDFとの格闘が日常。

**建築一式**: 住宅から工場まで。木造（W造）、鉄骨造（S造）、RC造（鉄筋コンクリート）、SRC造で施工体制が全く異なる。民間工事は各社独自の積算基準。

**とび・土工**: 足場組立、鉄骨建方、杭打ち。「高いところに登る仕事」の代名詞。安全管理が最優先。

**電気工事**: 電気配線、受変電設備、照明。第一種電気工事士の資格が必須。図面は展開接続図。

**管工事**: 給排水、空調、ガス配管。配管の経路設計が腕の見せ所。冬季の凍結防止が地方の課題。

### 1.3 元請・下請構造

```
発注者（国/自治体/企業）
  │
  ├─ 元請（ゼネコン/地場の総合建設会社）
  │    ├─ 一次下請（専門工事会社: 鉄筋、型枠、電気、配管等）
  │    │    ├─ 二次下請（さらに専門的な業者）
  │    │    └─ 二次下請
  │    ├─ 一次下請
  │    └─ 一次下請
  │
  └─ 設計事務所（図面を描く人たち。施工はしない）
```

重層下請の現実:
- 同じ作業員の名簿を、元請用/1次用/2次用で3回作り直し
- 資格証明書のコピーを何度も提出
- グリーンサイト/ビルディーに登録、かつ紙でも提出

**2024年建設業法改正による下請規制強化**（2025年12月完全施行）:
- 著しく低い材料費等による見積り・見積り変更依頼の**禁止**
- 請負人による不当に低い請負代金設定の**禁止**
- 契約後の請負代金変更に関する**協議義務**（建設業法初の明文化）
- 契約前のリスク情報の**提供義務**化

## 2. 入札と積算

### 2.1 入札の仕組み

```
■ 一般競争入札
  条件を満たす業者は誰でも参加可能。価格競争が激しい
  電子入札の普及率: 国・都道府県・指定都市 ≒ 100% / 市区町村 ≈ 30%未満（令和5年調査）

■ 指名競争入札
  発注者が「この会社に入札させる」と指名。実績のある会社が有利

■ 総合評価方式
  価格だけでなく技術力・実績も評価。「安かろう」では勝てない
  国交省直轄工事の適用率: 約97%（原則全工事適用）
  構造: 標準点100点 + 加算点10〜50点 ÷ 入札価格 で評価値を算出

■ 随意契約
  入札なしで直接契約（小規模工事や緊急時）

入札のキーワード:
  予定価格: 発注者が設定する上限価格
  最低制限価格: これ以下では失格（ダンピング防止）
  低入札価格調査基準: 2022年度改定で「一般管理費等×0.68」に引き上げ（従来×0.55）
  落札率: 落札額 ÷ 予定価格 × 100（通常80-95%）
```

### 2.2 積算とは何か

```
積算 = 工事にいくらかかるかを事前に計算すること

  入力: 設計図面、仕様書、数量計算書
  処理: 数量拾い出し → 単価選定 → 歩掛適用 → 諸経費計算
  出力: 工事費内訳書（見積書）
```

### 2.3 なぜ積算が重要か

1. **赤字受注の防止**: 積算が甘い → 実際のコストが上回る → 数百万〜数千万の損失
2. **入札での勝率**: 高すぎれば競合に負け、安すぎれば赤字 or 最低制限価格割れで失格
3. **実行予算の基準**: 積算結果 = 現場の予算目標。原価管理の出発点
4. **後継者問題**: 積算できる人が55歳以上。10年後に半減

### 2.4 積算の種類

| 種類 | タイミング | 精度 | 用途 |
|---|---|---|---|
| 概算積算 | 入札前の初期段階 | ±20-30% | 入札参加の判断 |
| 詳細積算 | 入札書類作成時 | ±5-10% | 入札金額の決定 |
| 実行予算 | 受注後 | ±3-5% | 現場の原価管理 |

### 2.5 積算ソフト

```
■ ATLUS（アトラス）: 最大手。公共工事向け。年間利用料50-100万円
■ Gaia（ガイア）: 中堅。使いやすさに定評
■ 武蔵: 内田洋行。中小向け
■ RIBC: 公益財団法人が提供。低価格

※ 中小建設会社ではExcel積算も多い（ソフト代が高いため）
```

### 2.6 積算の業務フロー

```
Step 1: 設計図書の受領・読み込み
  ├─ 図面（平面図、断面図、構造図、詳細図）
  ├─ 仕様書（施工条件・基準）
  ├─ 特記仕様書（その工事固有の条件）
  └─ 数量計算書（公共工事で発注者が提供する場合あり）

Step 2: 数量拾い出し ← 最も時間がかかる（全体の50-60%）
  ├─ 分類体系: 工種 → 種別 → 細別 → 規格
  ├─ 数量計算書がある → 転記（最も早い）
  ├─ PDFから読み取る場合
  └─ 図面から手計算で拾う場合

Step 3: 単価の選定
  ├─ 設計労務単価（国交省が毎年2月に公表。都道府県別×職種別51職種）
  ├─ 市場単価（建設物価、積算資料の月刊誌に掲載）
  ├─ 見積単価（専門業者から2-3社取得）
  └─ 自社実績単価（過去工事で実際にかかった単価）

Step 4: 歩掛（ぶがかり）の適用
  ├─ 歩掛 = 作業に必要な人・機械・材料の標準的な量
  ├─ 国交省「土木工事標準積算基準書」が最重要
  └─ 補正係数: 夜間×1.3〜1.5、狭隘×1.2〜1.3、水中×1.5〜2.0

Step 5: 諸経費の計算
  ├─ 直接工事費 = Σ(材料費 + 労務費 + 機械経費)
  ├─ 共通仮設費 = 直接工事費 × 5-15%
  ├─ 現場管理費 = 純工事費 × 20-40%
  ├─ 一般管理費等 = 工事原価 × 10-20%
  └─ 消費税

Step 6: 内訳書の作成・提出
  ├─ 鏡（表紙）、内訳書、明細書、単価表
  └─ 上長チェック → 経営判断で入札金額決定 → 提出
```

### 2.7 工事費の積み上げ構造

```
┌─ 直接工事費（材料費 + 労務費 + 機械経費）
├─ 共通仮設費（率 or 積み上げ）
├─ 現場管理費（率計算）
├─ 一般管理費等（率計算。本社経費+利益）
└─ 消費税

計算の流れ:
  直接工事費
  → 純工事費 = 直接工事費 + 共通仮設費
  → 工事原価 = 純工事費 + 現場管理費
  → 工事価格 = 工事原価 + 一般管理費等
  → 工事費 = 工事価格 + 消費税
```

## 3. 建設業許可・経営事項審査

### 3.1 建設業許可の要件

**許可が不要な軽微な工事**:
- 建築一式: 請負代金1,500万円未満 or 延べ150㎡未満の木造住宅
- その他: 請負代金500万円未満

**一般建設業 vs 特定建設業**

| 区分 | 一般建設業 | 特定建設業 |
|---|---|---|
| 対象 | 下請発注額 < 4,500万円（建築一式7,000万円） | 下請発注額 ≥ 4,500万円（建築一式7,000万円） |
| 財産的要件 | 自己資本500万円以上（または預金残高証明） | 資本金2,000万円以上、自己資本4,000万円以上、欠損額≤資本金の20%、流動比率75%以上（全条件同時充足） |
| 技術者要件 | 主任技術者（2級施工管理技士等）で可 | 監理技術者（1級施工管理技士等）が必須 |

**許可の5要件**:
1. **経営業務管理体制**（2020年改正）: 5年以上の建設業経営経験を有する常勤役員等を置く、または補佐者体制の整備
2. **専任技術者**: 営業所ごとに1級施工管理技士等の資格者を専任配置
3. **財産的基礎**: 上記の財産要件を充足
4. **誠実性**: 請負契約について不正行為をするおそれがない
5. **欠格要件非該当**: 破産・処分歴等なし

**2020年10月改正（経営業務管理体制への転換）**:
- 改正前: 個人（経管）が5年以上の建設業経営経験を有することが必須
- 改正後: 以下のいずれかで足りる
  1. 5年以上の建設業経営経験を有する常勤役員等（従来型・継続）
  2. 5年以上の建設業役員経験 + 2年以上の財務・労務・業務経験の補佐者の組み合わせ
  3. 2年以上の役員経験 + 5年以上の業務執行役員補佐経験の組み合わせ
- 趣旨: 経管要件のみで廃業を余儀なくされる小規模企業への対応。個人能力から組織体制の適切さへ評価軸を移行

許可は**5年更新**。更新を忘れると営業停止。

**出典**: 国土交通省「建設業許可の要件」

### 3.2 経営事項審査（経審）

公共工事を受注するには経審が必須。中小建設会社にとって入札参加資格の維持 = 会社の存続。

**総合評定値(P)の算出式**:

```
P = 0.25×X1 + 0.15×X2 + 0.20×Y + 0.25×Z + 0.15×W
（小数点未満四捨五入）

理論値: 最高2,143点 〜 最低-18点（令和5年8月以降）
全国平均P値: 約700点（制度設計上700点が全体の平均となるよう設計）
```

### 3.3 X1・X2（経営規模）

```
X1（25%）: 完成工事高評点
  ・2期平均 or 3期平均（有利な方を選択可）の工種別年間平均完成工事高
  ・CORINSへの登録が基礎データとなる

X2（15%）: 自己資本額・平均利益額評点
  ・自己資本額（純資産合計）
  ・2期平均の税引前当期純利益（絶対値）
  ・資本性借入金（劣後ローン）を2025年7月以降は自己資本に加算可
    （残存期間5年未満になると1年ごとに20%ずつ減額の逓減ルール）
```

### 3.4 Y点（経営状況分析）

外部の登録経営状況分析機関（CIIC等）が算出。8指標から計算。

| 分類 | 指標 | 計算式 |
|---|---|---|
| 負債抵抗力 | X1 純支払利息比率 | （支払利息－受取利息配当金）÷売上高×100 |
| 負債抵抗力 | X2 負債回転期間 | 負債合計÷（売上高÷12） |
| 収益性・効率性 | X3 総資本売上総利益率 | 売上総利益÷総資本×100 |
| 収益性・効率性 | X4 売上高経常利益率 | 経常利益÷売上高×100 |
| 財務健全性 | X5 自己資本対固定資産比率 | 自己資本÷固定資産×100 |
| 財務健全性 | X6 自己資本比率 | 自己資本÷総資本×100 |
| 絶対的力量 | X7 営業キャッシュフロー | 当期純利益＋減価償却費±その他 |
| 絶対的力量 | X8 利益剰余金 | 貸借対照表上の利益剰余金 |

**Y点算出式（CIIC公式）**:
```
A = -0.4650×X1 - 0.0508×X2 + 0.0264×X3 + 0.0277×X4
    + 0.0011×X5 + 0.0089×X6 + 0.0818×X7 + 0.0172×X8 + 0.1906
Y = 167.3×A + 583
```

各指標の良好値・悪化値の例: X1: -0.3%（良）〜5.1%（悪）/ X2: 0.9ヶ月（良）〜18.0ヶ月（悪）/ X3: 63.6%（良）〜6.5%（悪）

**出典**: 建設業情報管理センター（CIIC）公式

### 3.5 Z点（技術力）

**技術職員1人あたりの配点（審査基準日時点での在籍者）**:

## 4. 安全書類と現場管理

### 4.1 安全書類（全建統一様式）

元請に提出する安全書類。下請が作成する主なもの:

| 様式番号 | 名称 | 内容 | 工数 |
|---|---|---|---|
| 第1号-甲 | 下請負業者編成表 | 下請の構成 | 中 |
| 第1号-乙 | 下請負通知書 | 下請の情報 | 中 |
| 第2号 | 施工体制台帳 | 現場の施工体制 | 大 |
| 第3号 | 施工体系図 | 下請構造の図示 | 中 |
| 第5号 | 作業員名簿 | 現場に入る全作業員 | 最大（40名分を手書き/手打ち） |
| 第6号 | 工事安全衛生計画書 | 安全対策の計画 | 大 |
| 第9号 | 有資格者一覧表 | 作業員の保有資格 | 大 |

**新規現場で20-40時間**、月次更新で5-10時間/現場。人の出入りが激しいため名簿更新は毎週のように発生する。

### 4.2 施工計画書

工事着手前に元請が作成・提出する書類。1件30-50ページ。作成に2-5日。

主な記載事項: 工事概要・施工方法 / 工程表（バーチャート or ネットワーク）/ 安全対策 / 品質管理計画 / 仮設計画 / 環境対策（廃棄物処理計画）

過去の類似工事からのコピペと修正が実態。「去年のあの工事の施工計画書どこ？」が日常。

### 4.3 工事写真

1現場で数千〜数万枚。国交省の「写真管理基準」に基づく分類・整理が必要。
- 整理に1現場あたり20-40時間
- 黒板に工事名・工種・撮影内容を記載して撮影
- 電子納品時はXML形式での分類が必須（CALS/EC）
- 品質管理写真・安全管理写真・記録写真に分類

### 4.4 原価管理

```
赤字工事率: 中小建設会社の工事の15-25%が最終的に赤字

原価の4要素:
  ① 材料費: 資材・部材の購入費
  ② 労務費: 自社社員の人件費（出面表ベース）
  ③ 外注費: 下請・一人親方への支払い（売上の50-70%を占める。最大費目）
  ④ 経費: 重機リース・交通費・仮設費等

コスト上昇の実態（2021年比 2025年）:
  土木部門: +41% / 建築部門: +37% / 全体平均: +38%
  建設工事費デフレーター（2024年12月）: 130.9（2015年度=100）

問題:
  Excelの数式が壊れる（毎月同じことが起きる）
  現場監督が忙しくて実績入力が遅れる
  出来高と原価が別々のファイルで管理されている
```

**出典**: 国土交通省「建設工事費デフレーター」、帝国データバンク

### 4.5 技術者配置制度

```
工事現場には「主任技術者」or「監理技術者」の専任配置が必要。

  主任技術者: 全ての工事現場に配置（一般建設業）
    → 2級施工管理技士 or 実務経験10年以上
  
  監理技術者: 特定建設業の元請が配置（下請総額4,500万円以上）
    → 1級施工管理技士が必須

監理技術者の兼任可能要件（2020年改正・技士補制度）:
  ・1級技士補を「監理技術者補佐」として専任で各現場に配置する場合
    → 監理技術者が2現場を兼任可能
  ・これが技士補制度の最大の実務的意義（人手不足対策）

中小建設会社で1級施工管理技士は2-5名程度。
複数現場の掛け持ちは原則法令違反 → 技術者の「空き状況」を把握していないと受注できない。
```

  ・雇用保険料率（建設業）: 2024年度1.85% → 2025年度1.75%（0.1pt低下）

未加入の問題:

## 7. 建設業の経理

### 7.1 建設業特有の勘定科目

| 一般企業 | 建設業 | 備考 |
|---|---|---|
| 売上高 | 完成工事高 | 引渡し完了で計上（完成工事基準） |
| 売上原価 | 完成工事原価 | 材料費+労務費+外注費+経費 |
| 売掛金 | 完成工事未収入金 | 引渡し後の未入金 |
| 買掛金 | 工事未払金 | 材料・下請への支払 |
| 仕掛品 | 未成工事支出金 | 施工中の原価累計 |
| 前受金 | 未成工事受入金 | 着手金など |

### 7.2 建設業の財務構造

```
コスト構造（中小建設会社の典型）:
  外注費: 売上の50-70%（最大費目。元請大手は特に高比率）
  材料費: 売上の10-20%
  労務費（直接）: 売上の5-10%
  経費: 売上の5-10%
  現場管理費+一般管理費: 売上の10-20%
  利益: 売上の3-8%（赤字工事が15-25%存在）

建設業の平均原価率: 76.5%（中小企業実態基本調査 令和2年度決算）

企業規模別の業況:
  スーパーゼネコン: 資材価格・人件費上昇、受注競争の影響で利益率低下傾向
  準大手: マンション建設等比較的高利益率事業で安定
  中小企業: 外注依存が高く、外注費上昇の影響を直接受ける
```

```
建設業倒産件数の推移（帝国データバンク・東京商工リサーチ）:

  物価高（コスト増）倒産: 250件（構成比13.2%）

```

### 7.4 出来高払いの仕組み

```
工事期間6ヶ月、契約金額3,000万円の場合:

  1月（着工）: 出来高500万 → 2月初に出来高確認書提出 → 2月末請求
               → 元請の支払サイト60日 → 4月末に入金
  2月: 出来高600万 → 5月末に入金

★ 1月に材料費・外注費を払うが、入金は4月末
  → 3ヶ月分の運転資金が必要
  → 年商1億の会社なら常時2,500-3,000万円の運転資金が必要
```

### 7.5 手形・ファクタリング

```
建設業の入金手段は多様:
  現金振込: 最も望ましいが、大口工事では少ない
  約束手形: 期日90-120日。資金繰り圧迫の主因。不渡りは2回で銀行取引停止
  電子記録債権（でんさい）: 手形の電子版。分割譲渡が可能
  ファクタリング: 売掛金の早期現金化。借入ではないので経審スコアに影響しない

1つの工事で複数パターンが混在するのが常態
  例: 着手金=現金、中間金=手形50%+現金50%、完成金=でんさい
```

### 7.6 工事別原価管理

```
工事完成基準（中小の90%以上がこちら）:
  引渡し完了まで売上計上しない。施工中は全て「未成工事支出金」

工事進行基準（収益認識基準）:
  進捗度に応じて売上を段階計上（2021年収益認識基準改正で適用拡大）
  中小では見積精度が低く適用困難
```

## 9. 主要な資格・資格制度

### 9.1 施工管理技士（国家資格）

建設業法における「主任技術者」「監理技術者」の要件となる最重要資格。

**受験者数・合格率（2024年度 第一次検定）**:
| 種別 | 受験者数 | 合格率 | 前年度比（受験者） |
|---|---|---|---|
| 1級土木 | 51,193人 | 44.4% | **+55%**（受験資格緩和で急増） |
| 1級建築 | 37,651人 | 36.2% | **+56%** |
| 1級電気工事 | 23,927人 | 36.7% | +47% |

**2024年度に受験者数が急増した理由**: 2024年4月の受験資格緩和（19歳以上であれば実務経験不問で受験可能に）→ 実務未経験者の大量流入で合格率が大幅低下。

**2021年度制度改正（令和3年度）の主要内容**:
- 「学科試験」→「第一次検定」、「実地試験」→「第二次検定」に名称変更
- **第一次検定合格が生涯有効な国家資格**となった（旧来は第二次検定合格前提）
- 第一次検定合格者に「**施工管理技士補**」の称号が付与（国家資格として新設）

**技士補制度の最大の実務的意義**:
- 1級技士補を「監理技術者補佐」として各現場に専任配置 → 監理技術者の2現場兼任が可能
- 技術者不足対策として2020年改正建設業法と連動

| 資格 | 対応業種 | 経審Z点 | 特徴 |

## 10. 工事情報システム・インフラ

### 10.1 主要なクラウド・システム

```
■ グリーンサイト（建設サイト・ジャパン）
  安全書類（全建統一様式）のクラウド管理
  大手ゼネコンが元請の場合にほぼ必須
  → 「グリーンサイトに登録してから現場に来てくれ」が定番

■ CORINS（工事実績情報システム）
  公共工事の実績情報を登録・管理するシステム（建設業情報管理センター JACIC 運営）
  経審の完成工事高の根拠となる
  → 登録漏れがあると経審X1点数が下がる

■ CCUS（建設キャリアアップシステム）
  技能者カード + 就業履歴管理
  → 経審Z点・W点に影響するため登録必須化が進む

■ 電子納品システム（CALS/EC）
  国交省発注工事では電子納品が原則
  成果物（設計図書・工事完成図書）をXML・PDF形式で納品
  → 写真管理ソフトとの連携が必要

■ ANDPAD（アンドパッド）
  施工管理・現場管理クラウド
  利用社数23.3万社・ユーザー数68.4万人（建設業マネジメントクラウド8年連続シェアNo.1）
```

### 10.2 電子入札システム

```
■ CALS/EC（公共工事電子入札）
  普及率（令和5年7月1日時点）:
    国土交通省直轄工事: 原則100%
    都道府県・指定都市: 実質100%
    市区町村: 電子入札対応は約30%未満（大半が未対応）
  → 市区町村レベルでは窓口持参・郵送入札が依然として多い

■ ICカード認証（建設業電子証明書）
  電子入札に必要。1〜3年更新で数万円
  ICカードリーダー + 専用ソフトが必要
  → 「ICカードの期限が切れていて入札できなかった」が定番のミス
```

### 10.3 公共工事前払金保証

```
仕組み:
  公共工事では工事着工前に代金の一部（通常40〜60%）が前払いされる
  保証会社（東日本建設業保証・西日本建設業保証等）が保証
  → 中小建設会社の資金繰りを支える重要な仕組み

毎月15日に東日本・西日本の保証3社が統計を公表
  → 内閣府も公共工事の発注動向を把握する先行指標として参照
```

## 11. 建設業の地域特性

### 11.1 地方と都市部の違い

```
地方部での公共工事依存度:
  ・地域建設業の経営基盤の多くを公共投資が支える構造
  ・公共発注者の内訳: 市区町村50.1%・都道府県17.2%が主体（受注動態統計）

地方建設業の課題:
  ・許可業者数のCランク業者（中小規模）の減少が顕著
  ・公共施設の維持管理費: 2013年度時点で約3.6兆円/年
    → 10年後: 4.3〜5.1兆円/年、20年後: 4.6〜5.5兆円/年に増加見通し
  ・維持管理を担うべき地域建設業者の減少が同時進行
    → 「担い手の空洞化」が深刻化

技能者の地域格差:
  ・都市部（東京・大阪・愛知等）に技能者が集中
  ・地方の建設会社は大型工事受注時に技能者を確保できない場合も
```

### 11.2 施工時期の平準化（国の対策）

```
問題: 公共工事が年度末（2〜3月）に集中 → 建設会社の繁閑差が激しい

対策（担い手3法に明記）:
  ・債務負担行為・繰越明許費の積極活用
  ・ゼロ国債（ゼロ債）工事による翌年度着手の促進
  ・中長期的な発注見通しの作成・公表義務化

効果: 年間工事量の平準化により、建設会社の人材・設備の安定的活用が可能になる
```
