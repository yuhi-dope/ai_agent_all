# g_09 参入障壁マトリクス

> **用途**: クライアント企業の現在の能力（設備/認証/実績）から「どのTier・どの産業に参入できるか」を判定する
> **最終更新**: 2026-03-30
> **前提**: BPO提案時に「次にどの市場を狙うべきか」をAIが自動提案するためのナレッジベース

## 1. 産業×Tier別の参入要件マトリクス

### 1-1. 自動車産業

| Tier | 必要認証 | 必要設備 | 最低実績 | 参入難易度 | 年間売上目安(1社) | 備考 |
|---|---|---|---|---:|---:|---|
| **OEM** | IATF16949 + 個別認定 | フルライン(鋳造/鍛造/切削/組立) | 10年+大手取引実績 | ★★★★★ | 100億円+ | 実質参入不可。M&Aのみ |
| **Tier1** | IATF16949必須 | 5軸MC/CNC旋盤/CMM/専用ライン | 5年+Tier2実績 + PPAP承認 | ★★★★ | 10-50億円 | コアツール(APQP/PPAP/FMEA/MSA/SPC)完全運用必須 |
| **Tier2** | IATF16949推奨(ISO9001最低) | CNC旋盤/MC/CMM | 3年+量産実績 + 品質保証体制 | ★★★ | 1-10億円 | IATF未取得でもISO9001+実力で参入可能 |
| **Tier3** | ISO9001推奨(なしでも可) | 汎用旋盤/フライス/MC | 1年+加工実績 | ★★ | 3,000万-3億円 | 単工程の部品加工。参入しやすいが利益率低い |

**IATF16949の取得要件**
- 前提: ISO9001認証取得済み
- 取得期間: 12-18ヶ月（コンサル含む）
- 取得コスト: 500-1,500万円（企業規模による）
- コアツール習熟: APQP/PPAP/FMEA/MSA/SPC（各ツール2-3ヶ月の研修+実践）
- 審査機関: JQA/SGS/PJR等の認定機関
- 現状: 中小製造業の38.4%が取得済み、21.7%が取得要求されているが未取得

### 1-2. 航空宇宙産業

| Tier | 必要認証 | 必要設備 | 最低実績 | 参入難易度 | 年間売上目安(1社) | 備考 |
|---|---|---|---|---:|---:|---|
| **OEM(機体/エンジン)** | JIS Q 9100 + Nadcap + 顧客個別認定 | 大型5軸MC/特殊合金加工設備/クリーンルーム | 10年+航空実績 | ★★★★★ | 50億円+ | 三菱重工/IHI/川崎重工クラス |
| **Tier1(モジュール)** | JIS Q 9100 + Nadcap(特殊工程) | 5軸MC/CMM/非破壊検査装置 | 5年+航空部品実績 | ★★★★ | 5-30億円 | OEM直接認定が必要 |
| **Tier2(部品加工)** | JIS Q 9100必須 | 5軸MC/CNC旋盤/CMM | 2年+精密加工実績 | ★★★ | 5,000万-5億円 | **最も現実的な参入ポイント** |

**JIS Q 9100の取得要件**
- 前提: ISO9001認証取得済み
- 取得期間: 8-12ヶ月（最短3ヶ月の急ぎ対応あり）
- 取得コスト: 200-500万円
- 審査機関: JQA/ペリージョンソン等
- 注意: Nadcapは JIS Q 9100 取得後でないと申請不可

**Nadcap認証（特殊工程）**
- 対象工程: 溶接/熱処理/表面処理/非破壊検査/化学処理
- 取得期間: 12-24ヶ月
- 取得コスト: 300-800万円
- 更新: 18ヶ月ごと

### 3-3. 参入可否の早見チェックリスト

**自動車Tier2に参入できるか？**
- [ ] ISO9001取得済み
- [ ] CNC旋盤 or MC保有
- [ ] CMM保有（外注でも可）
- [ ] 量産実績あり（月産1,000個以上）
- [ ] SPC管理の基本知識あり
→ 3つ以上チェック = 参入検討可能

**航空Tier2に参入できるか？**
- [ ] ISO9001取得済み（JIS Q 9100は未取得でも可）
- [ ] 5軸MC保有（または導入予定）
- [ ] CMM保有
- [ ] 難削材(チタン/インコネル)の加工経験
- [ ] トレーサビリティ体制あり
→ 3つ以上チェック = JIS Q 9100取得から始めるべき

**半導体装置部品に参入できるか？**
- [ ] ISO9001取得済み
- [ ] 精密MC/旋盤保有（±10μm以下の実績）
- [ ] クリーン加工環境（または整備可能）
- [ ] SUS316L/アルミ精密加工の経験
- [ ] 洗浄・梱包のクリーン化が可能
→ 3つ以上チェック = すぐに営業開始可能

## 5. 売上予測ロジック

### 5-1. 基本計算式

```
年間売上予測 = ターゲット産業の部品調達額
             × 中小企業の調達シェア
             × クライアントが取れるシェア（キャパ/認証/地理）
             × 受注確率
```

### 5-2. 計算例1: 半導体装置向け精密切削部品

```
■ 前提
  対象企業: 従業員30名、MC4台、CNC旋盤3台、ISO9001取得済み
  現在の年商: 3億円（自動車部品中心）
  目標: 半導体装置向け精密部品への参入

■ 市場サイズ算出
  半導体装置の日本製販売高(2024): 4兆4,371億円
  うち部品調達比率: 約40% = 1兆7,748億円
  うち精密切削部品: 約15% = 2,662億円
  うち中小企業への発注比率: 約30% = 799億円

■ シェア算出
  クリーン加工対応投資: 実施予定(500万円)
  地理的優位性: 関東圏(装置メーカー多数) = 係数1.2
  設備キャパ(MC4台+旋盤3台): 年間加工能力 約1.5億円分
  認証: ISO9001あり = 参入可能

■ 売上予測
  アドレス可能市場: 799億円
  地理係数: × 1.2 = 959億円
  シェア(キャパベース): 1.5億 / 959億 = 0.016%
  受注確率(初年度): 30%
  → 初年度売上: 1.5億 × 30% = 4,500万円
  → 3年目売上(受注率60%): 1.5億 × 60% = 9,000万円
  → キャパ増強後(MC+2台): 2.5億 × 60% = 1.5億円

■ 利益率改善効果
  現在(自動車Tier3): 営業利益率 3%
  半導体装置部品: 営業利益率 8-12%（高精度=高単価）
  → 売上1.5億円 × 利益率10% = 営業利益1,500万円
     (現在の自動車3億 × 3% = 900万円より高い)
```

### 5-3. 計算例2: 航空宇宙Tier2への参入

```
■ 前提
  対象企業: 従業員50名、5軸MC2台、CMM1台、ISO9001取得済み
  現在の年商: 5億円（自動車+一般機械）
  目標: 航空宇宙Tier2（JIS Q 9100取得）

■ 市場サイズ算出
  日本の航空機関連出荷額(2024): 1.8兆円
  うち部品加工の外注比率: 約30% = 5,400億円
  うちTier2向け: 約40% = 2,160億円
  うち切削加工: 約50% = 1,080億円

■ シェア算出
  JIS Q 9100: 取得予定(12ヶ月後)
  5軸MC: 2台保有 → 加工能力 年間1億円分
  地理的優位性: 中部圏(航空産業集積) = 係数1.3
  難削材経験: チタン加工経験あり = 係数1.5

■ 売上予測
  アドレス可能市場: 1,080億円
  地理+技術係数: × 1.3 × 1.5 = 2,106億円
  シェア(キャパベース): 1億 / 2,106億 = 0.005%
  受注確率(JIS Q 9100取得後初年度): 20%
  → 初年度売上: 1億 × 20% = 2,000万円
  → 3年目売上(実績蓄積、受注率50%): 1億 × 50% = 5,000万円
  → 5軸MC増設+Nadcap取得後: 2億 × 60% = 1.2億円

■ 投資回収
  認証取得コスト: 500万円(JIS Q 9100) + 500万円(Nadcap) = 1,000万円
  → 3年目の年間売上5,000万 × 利益率12% = 600万円
  → 投資回収: 約2年（3年目利益 + 4年目利益で回収）
```

### 5-4. 計算例3: 医療機器部品への参入（体外機器向け）

```
■ 前提

  目標: 医療機器の体外部品（筐体・治具・検査装置部品）

■ 市場サイズ算出

  うち体外機器の部品加工外注: 約15% = 6,750億円
  うち中小企業への発注: 約25% = 1,688億円

■ シェア算出

  設備キャパ: 年間8,000万円分
  地理: 関東・東海圏 = 係数1.1

■ 売上予測
  アドレス可能市場: 675億円 × 1.1 = 743億円
  シェア: 8,000万 / 743億 = 0.011%
  受注確率(ISO13485取得後初年度): 25%

  → 3年目売上(受注率50%): 8,000万 × 50% = 4,000万円

■ 投資回収
  ISO13485取得コスト: 200-300万円
  → 初年度売上2,000万 × 利益率10% = 200万円で1年回収
  → 医療機器は単価が高く、一般機械比で利益率+3-5%向上
```
