## 1. 不動産業とは何か

### 1.1 4つの業態

不動産業と一口に言っても中身は全く違う。大きく4業態に分かれ、それぞれ収益構造もスキルも顧客も異なる。

#### 売買仲介

不動産の売主と買主をつなぐ。宅建業免許が必要。

- **収益構造**: 仲介手数料（売買価格 × 3% + 6万円 + 消費税が上限。400万円超の場合）。両手（売主・買主の双方から受領）なら2倍。1件の成約で数十万〜数百万円の手数料
- **取引の頻度**: 社員1人あたり月1-3件の成約が平均。繁忙期（1-3月）に偏る
- **利益の変動**: 在庫を持たないため固定費は低いが、成約ゼロの月は売上ゼロ。波が極めて大きい
- **必要スキル**: 物件査定力、媒介営業力（「売らせてください」と言える営業力）、住宅ローンの知識、登記・税務の基礎
- **典型的な会社**: 社長+営業1-3名+事務1名。社長が査定と大口案件を担当

#### 賃貸仲介

部屋を探している人に物件を紹介する。宅建業免許が必要。

- **収益構造**: 仲介手数料（家賃1ヶ月分が上限）+ AD（オーナーからの広告費、家賃0.5-3ヶ月分）。ADが実質的な利益の源泉。仲介手数料だけでは店舗の家賃も払えない
- **取引の頻度**: 繁忙期（1-3月）は月30-50件、閑散期は月5-15件。季節変動が売買より激しい
- **粗利**: 1件あたり10-20万円（仲介手数料+AD）。薄利多売
- **必要スキル**: 物件紹介力（SUUMOの反響対応→内見→成約のスピード）、管理会社との関係構築、保証会社の審査知識
- **典型的な会社**: 駅前の1-2店舗。FC（アパマンショップ、ミニミニ、エイブル等）加盟が多い。社長+営業2-5名

#### 賃貸管理（PM: プロパティマネジメント）

オーナーに代わって物件を管理する。2021年施行の賃貸住宅管理業法により、管理戸数200戸以上は国交省への登録が義務化された。

- **収益構造**: 管理手数料（家賃の3-5%）が安定収入のベース。加えて更新手数料、修繕マージン（10-30%）、鍵交換・ハウスクリーニングの手配手数料
- **収入の安定性**: ストック型ビジネス。管理戸数が増えれば増えるほど安定する。1,000戸×平均家賃8万×5%=月400万円の管理手数料
- **利益率**: 粗利率30-40%。ただし人件費比率が高い（PM担当者1人が200-500戸を担当）
- **必要スキル**: 入金消込、滞納督促、オーナー対応（収支報告・修繕提案・空室対策）、退去立会い、修繕手配、法令遵守
- **典型的な会社**: 社長+PM担当2-5名+事務1-2名。仲介も兼業していることが多い

#### 開発・分譲

土地を仕入れて建物を建てて売る。または大規模マンションを企画・販売する。

- **収益構造**: 販売価格 - （土地取得費+建築費+販売費）= 利益。粗利率10-20%
- **資本の必要性**: 土地取得に億単位の資金が必要。借入金が大きい。金利上昇で一気に利益が飛ぶ
- **リスク**: 売れ残りリスク。建築費の上昇リスク。市況変動リスク。景気後退期には倒産する会社が出る
- **典型的なプレイヤー**: 大手デベロッパー（三井不動産、住友不動産等）、パワービルダー（飯田グループ等）。中小でやっている会社もあるが資金力が必要

#### 4業態の関係性

実際には1社が複数の業態を兼業していることが多い。特に中小は「賃貸仲介+賃貸管理」「売買仲介+賃貸管理」のセットが定番。仲介で入居者を見つけ、管理で継続収入を得る。管理物件が増えるほど空室が出たとき自社で埋められるので、仲介手数料+ADも入る。この「仲介×管理の相乗効果」が中小不動産会社の経営の基本形。

```
収益安定度:  開発・分譲 < 売買仲介 < 賃貸仲介 < 賃貸管理
1件あたり利益: 賃貸仲介 < 賃貸管理 < 売買仲介 < 開発・分譲
参入障壁:    賃貸仲介 < 売買仲介 < 賃貸管理 < 開発・分譲
```

```
  ├── 入居者クレーム対応（騒音、設備故障、ゴミ問題）
  ├── 修繕手配（業者への連絡、見積取得、オーナー承認）
  └── オーナー対応（月次報告書の説明、空室対策の相談）

  事務（2名）
  ├── 経理担当: 入金消込、オーナー送金、請求書発行
  ├── 契約事務: 契約書作成、更新手続き、保険手続き
  ├── 電話受付: 入居者・オーナー・業者からの電話が1日30-50件

```

**ポイント**: 宅建士が社長1人の会社では、重説のたびに社長が出てくる必要がある。社長不在の日は契約ができない。

### 1.5 不動産会社の文化

#### 「物件を回す」ことが全て

不動産会社にとって最大の敵は「空室」。空室=売上ゼロ。管理会社は空室が続くとオーナーから「管理会社を変える」と言われる。仲介会社は空室がなければ紹介する物件がない。売買は売り物件がなければ始まらない。「物件を動かす」「客付けする」が業界共通の最大命題。

#### 人間関係がビジネスの根幹

オーナーとの関係、仲介会社とのネットワーク、修繕業者との付き合い。全てが人間関係。「あそこの管理会社に頼めば空室もすぐ埋まる」「あの修繕業者は腕がいい」「あの仲介会社は客付けが早い」。口コミと紹介でビジネスが回る。Webマーケティングの効果はあるが、最終的に「この人に任せたい」で決まる。

#### 紙・FAX・電話の世界

```
実態:
  管理ソフト導入済み:    30-40%（いえらぶ、リドックス、ESいい物件One等）
  Excel+手動管理:       40-50%
  紙+電話+FAX:          10-20%（小規模ほど多い）

  管理ソフトを入れていても:
    - 入金消込は手動Excel
    - オーナー報告書はExcel→PDF→メール添付
    - 修繕記録はノートに手書き
    - 更新管理はExcelの日付フィルタ
    → ソフトに入力するのが面倒で、一部機能しか使っていない
```

管理ソフトの月額費用は¥1-15万。「高い」と感じる会社が多い。費用対効果の説明が足りないのではなく、ITに対する心理的ハードルが高い。「前のやり方で回っているのに、なぜ変えるのか」が本音。

#### 業法の縛りと資格

不動産業は法律に縛られた業界。宅建業法、借地借家法、建築基準法、都市計画法、消費者契約法、賃貸住宅管理業法。法令違反=免許取消=廃業。だから「法律がどうなっているか」を常に気にする。宅建士の試験は年1回。合格率17%。「宅建に受からない営業マン」は肩身が狭い。

#### 土日出勤・夜間対応

不動産業は土日が書き入れ時。内見は土日に集中する。賃貸仲介は水曜定休が業界慣例。管理会社は入居者からの「水漏れ」「鍵をなくした」の電話が夜中でもかかってくる。24時間対応のコールセンターを外注している会社もあるが、中小は社長の携帯がそのまま緊急連絡先。

## 2. 賃貸管理の実務

### 2.1 月次サイクル（管理会社の1ヶ月）

賃貸管理の業務は月次サイクルで回る。毎月同じことの繰り返し。だが「同じこと」に例外が毎月山ほど混じるから大変。

```
■ 月初（1-5日）: 入金確認・消込

  口座振替の結果データを取得
  → 振替OK → 自動消込
  → 振替NG（残高不足等） → 督促リストへ

  銀行振込の明細をダウンロード
  → 請求一覧と突き合わせ（名義+金額）
  → 一致 → 消込
  → 不一致 → 手動確認

  クレジット決済の結果を取得
  → ほぼ自動消込

  この作業が月の最初の3-5日を支配する。300戸で月40-60時間。

■ 月初〜中旬（5-15日）: 滞納督促

  未入金リストから督促開始
  Day 1-3:  電話（「行き違いでしたらすみません」）→ 70%解決
  Day 4-7:  SMS/メール → さらに15%解決
  Day 8-14: 督促状（書面郵送）→ さらに5%解決
  Day 15+:  保証会社への代位弁済請求 or 内容証明

  保証会社加入率は新規契約の80-90%。保証会社が入っていれば、
  管理会社は代位弁済請求→保証会社が立替→管理会社は回収完了。
  問題は古い契約で保証会社なし・連帯保証人のみのケース。

■ 月中（10-15日）: オーナー送金

  送金額 = 入金額 - 管理手数料 - 修繕費立替 - その他経費
  物件ごと・オーナーごとに仕分けて計算
  送金明細書を作成 → 振込実行

  オーナーが50人いれば、50通りの計算。
  管理費率がオーナーによって違う（3%〜5%）。
  修繕費の立替があるオーナー、ないオーナー。
  ローン返済を差し引いて送金するケース。
  → ミスると信頼を一気に失う。管理委託の解約に直結。

■ 月中〜下旬（15-25日）: オーナー収支報告書

  月次報告書の作成:
    ① 収支報告（入金・支出・送金額の明細）
    ② 滞納状況（誰が・いくら・何ヶ月）
    ③ 修繕報告（何を・いくらで直したか）
    ④ 空室状況（何号室が空き・募集状況・反響数）

  40オーナー × 報告書作成30分 = 月20時間。
  オーナーの要望で書式が違うケースも。Excelテンプレの管理が地獄。

■ 随時: 入退去対応

  入居:
    申込受付 → 保証会社審査 → 審査通過 → 契約書作成 → 重説 → 契約締結 → 鍵渡し
    1件あたり3-5時間。重説は宅建士が必須。

  退去:
    解約通知受付（通常1-2ヶ月前）→ 退去日確定 → 退去立会い → 原状回復範囲確定
    → 修繕見積取得 → 敷金精算 → 入居者へ精算書送付 → 次の入居者募集開始
    1件あたり5-10時間。原状回復でもめると長期化。

  繁忙期（1-3月）: 300戸で月20-30件の入退去。通常月の3-5倍。

■ 随時: 修繕・クレーム対応（後述の「5. 修繕管理の実務」で詳述）

■ 随時: 更新管理

  更新時期（通常2年ごと）の6ヶ月前から準備
  → 更新案内送付 → 入居者の意思確認 → 条件交渉（賃料改定等）
  → 更新契約書作成 → 締結 → 保証会社更新 → 火災保険更新
  → 更新料徴収（家賃1-2ヶ月分。首都圏の慣例、関西は少ない）

  更新案内の送り忘れ = 法定更新（自動更新）→ 更新料を徴収する機会を失う
  → 期限管理が極めて重要。
```

### 2.2 入金消込の地獄

入金消込は賃貸管理会社の最大の工数ポイント。何がそんなに大変なのかを掘り下げる。

#### 入金手段の混在

```
口座振替:   60-70%  収納代行会社（SMBCファイナンスサービス等）経由
銀行振込:   20-30%  入居者が自分で管理会社の口座に振り込む
クレジット:  1-5%   導入は増加中だが、決済手数料2-3%を管理会社が負担
現金持参:    1-5%   高齢の入居者。受領書の発行・現金管理が必要
```

口座振替は振替結果データ（CSV）に「OK/NG」が返ってくるから消込は楽。問題は銀行振込。

#### 名義不一致のパターン（実例）

```
■ カタカナ変換系
  契約者「山田太郎」→ 振込名義「ヤマダ タロウ」         → 金額一致でほぼ特定可能
  契約者「山田太郎」→ 振込名義「ﾔﾏﾀﾞ ﾀﾛｳ」            → 半角カナ。古い銀行のATM

■ 家族・代理振込系
  契約者「山田太郎」→ 振込名義「ヤマダ ハナコ」         → 配偶者。要確認
  契約者「山田太郎」→ 振込名義「ヤマダ ケンジ」         → 息子? 親? 要電話確認
  契約者「田中花子」→ 振込名義「サトウ ハナコ」         → 旧姓（結婚前）

■ 法人系
  契約者「田中花子」→ 振込名義「カ）タナカショウジ」     → 社宅。会社が家賃負担
  契約者「鈴木一郎」→ 振込名義「ユ）ABCコーポレーション」→ 有限会社の略号
  契約者「佐藤次郎」→ 振込名義「シャ）エヌピーオーホウジン △△」→ NPO法人の社宅

■ 金額不一致系
  請求¥80,000 → 入金¥75,000   → 共益費を引いて振り込んだ?
  請求¥80,000 → 入金¥160,000  → 2ヶ月分? 先月の滞納分?
  請求¥80,000 → 入金¥79,000   → 振込手数料を差し引いた（あるある）
  請求¥80,000 → 入金¥82,000   → 更新料の一部? 水道代の追加?

■ 部屋番号付き系
  振込名義「101 ヤマダ」「ヤマダ 101ゴウ」→ 部屋番号を入れてくれる人
  → ありがたいが、名義のマッチングルールが増える

■ 不明入金
  振込名義「タカハシ」→ 該当者が3人いる
  振込名義「フリコミ」→ 金額で推定するしかない
  振込名義なし → ATMの仕様で名義が省略されることがある
```

これらを毎月、毎入金、1件ずつ確認している。300戸あれば銀行振込が60-90件。うち20-30%（12-27件）が何らかの名義不一致。1件あたり3-10分かかるから、消込だけで月8-12時間。

#### 口座振替への移行が最善だが

「全員口座振替にすればいいじゃないか」は正しい。しかし:

- 古い入居者は「長年振込でやってきた」と変えたがらない
- 外国人入居者は口座振替の手続きが煩雑
- 法人契約は会社の経理の都合で振込にしか対応できないケースがある
- 口座振替の振替不能（残高不足）は結局手動対応が必要

結局、口座振替率を上げつつ、銀行振込の消込をAIで効率化する、の両面が必要。

### 2.3 滞納対応の実務

#### エスカレーションフロー

```
[Day 1-3] 電話連絡
  「お家賃のお支払いが確認できておりません。行き違いでしたら申し訳ございません」
  → うっかり忘れが8割。ここで70%が解決
  → 連絡がつかない場合は時間帯を変えて3回まで

[Day 4-7] SMS/メール
  「○月分のお家賃¥XX,XXXの入金が確認できておりません。
   ○月○日までにお支払いをお願いいたします」
  → さらに15%が解決

[Day 8-14] 督促状（書面郵送）
  「度重なるご連絡にもかかわらずお支払いが確認できておりません。
   ○月○日までにお支払いがない場合は保証会社へ連絡いたします」
  → 配達証明付きで送ると証拠になる。さらに5%が解決

[Day 15-30] 保証会社への代位弁済請求（保証会社加入の場合）
  代位弁済請求書を提出 → 保証会社が管理会社に家賃を立替払い
  → 以後は保証会社が入居者に直接請求（管理会社の手を離れる）
  保証会社未加入の場合 → 連帯保証人へ連絡

[Month 2] 内容証明郵便
  弁護士名義で送る場合もある
  「○日以内に全額お支払いがない場合は法的措置を講じます」
  → 法的証拠力がある

[Month 3+] 明渡請求訴訟
  弁護士費用: 30-50万円。訴訟から退去まで3-6ヶ月
  強制執行: 裁判所の執行官が行う。費用: 20-40万円
  → ここまで来ると完全に赤字。保証会社の存在がいかに重要か分かる
```

#### 保証会社の仕組み

```
入居時:
  入居者が保証会社と保証委託契約を締結
  保証料: 家賃の0.5-1ヶ月分（初回）+ 年1-2万円（更新）

滞納時:
  管理会社 → 保証会社に「代位弁済請求」
  保証会社 → 管理会社に家賃を立替払い（通常1-2週間で入金）
  保証会社 → 入居者に求償（取り立て）

  管理会社にとってのメリット:
    滞納リスクから解放される
    督促業務の一部を保証会社に移管できる
    代位弁済の範囲: 家賃+共益費+更新料+原状回復費+残置物撤去費（保証会社による）

  主な保証会社:
    全保連、日本セーフティー、Casa、日本賃貸保証（JID）、
    フォーシーズ、Casaなど約50社

  保証会社の審査基準:
    信販系: CIC等の信用情報を照会。審査厳しめ
    LICC系: 賃貸保証の情報共有。中程度
    独立系: 自社基準。緩め。外国人も通りやすい
```

現在、新規契約の80-90%が保証会社加入。連帯保証人のみの契約は減少傾向（2020年民法改正で保証人の極度額設定が義務化された影響）。

### 2.4 オーナー対応の実態

管理会社にとってオーナーは「お客様であり上司」。オーナーが管理委託を解約すれば、管理戸数が減り、売上が直撃する。

#### 月次報告書

```
記載事項:
  ① 収支明細（入金額・管理手数料・修繕費・送金額）
  ② 滞納状況（誰が何ヶ月分滞納しているか）
  ③ 修繕実施報告（内容・費用・業者名）
  ④ 空室状況（空室の募集状況・反響数・内見数）

送付方法:
  郵送:     40%（高齢オーナー。紙じゃないと読まない）
  メール:    40%（PDF添付）
  ポータル:  20%（オーナー向けWebポータルを用意している管理会社）

問題:
  ・Excelで手作業で作成。物件ごとにフォーマットが違う
  ・オーナーが読んでいない（送っても電話で聞いてくる）
  ・「先月の入金が少ないのはなぜ？」に即答できない
  ・送付の遅延が信頼低下に直結
```

#### オーナーが気にすること

```
1. 空室率 — 「空いてる部屋はいつ埋まるの?」
2. 家賃収入 — 「今月いくら入った?」
3. 修繕費 — 「また修繕? いくらかかるの?」
4. 賃料改定 — 「家賃上げられない? 下げないとダメ?」
5. 管理会社の対応 — 「ちゃんとやってくれてる?」

→ オーナーの満足度 = 空室率の低さ × 報告の丁寧さ × 修繕提案の的確さ
→ 管理委託の継続/解約はオーナーの「体感」で決まる
```

#### 空室対策の相談

空室が続くとオーナーから「どうすればいい？」と聞かれる。管理会社の対応:

```
① 賃料の見直し
  周辺相場を調べて提案。根拠がないと「下げたくない」のオーナーを説得できない
  → SUUMO/HOME'Sで類似物件を検索して比較表を作る（30分-1時間）

② AD（広告料）の増額
  仲介会社への報酬を増やして優先的に紹介してもらう
  AD 0 → AD 1ヶ月にすると反響が2-3倍

③ 設備のバリューアップ
  追い焚き、独立洗面台、宅配ボックス、無料Wi-Fi、モニター付きインターホン
  投資対効果の高い設備提案ができるかが管理会社の腕

④ リフォーム・リノベーション
  築古物件の全面改装。費用100-500万円
  「¥200万の投資で家賃¥1万アップ × 10年 = ¥120万の回収」の収支計算

⑤ 募集条件の緩和
  ペット可、楽器可、事務所利用可、外国人可
  → 競合との差別化になるがリスクも増える
```

### 3.1 売買取引の全体フロー

```
■ 売主側のフロー（媒介獲得→販売活動→成約）

  Step 2: 物件査定（後述の「4. 物件査定の実務」で詳述）
  Step 3: 媒介契約の締結（売主と仲介会社の契約。3種類ある）
  Step 4: 販売活動
    レインズ登録 → ポータルサイト掲載 → チラシ配布 → オープンハウス
  Step 5: 買主からの購入申込（買付証明書）
  Step 6: 条件交渉（価格、引渡日、ローン特約、手付金額）
  Step 7: 重要事項説明（宅建士が実施。後述の「6. 重要事項説明」で詳述）
  Step 8: 売買契約の締結（37条書面）
  Step 9: ローン本審査（買主側。2-4週間）
  Step 10: 決済・引渡し
    残代金の支払い → 所有権移転登記 → 鍵の引渡し
    司法書士が立ち会う。銀行の応接室で行うことが多い

  全工程: 2-6ヶ月（査定から引渡しまで）

■ 買主側のフロー

  Step 1: 物件探し（ポータルサイト/仲介会社に相談）
  Step 2: 内見（3-10件見るのが平均）
  Step 3: 購入申込（買付証明書の提出）
  Step 4: ローン事前審査（1-2週間）
  Step 5: 重要事項説明
  Step 6: 売買契約
  Step 7: ローン本審査（2-4週間）
  Step 8: 決済・引渡し
```

### 3.2 媒介契約の3種類

```
■ 一般媒介
  複数の仲介会社に依頼できる
  レインズ登録義務なし
  報告義務なし
  → 売主には自由度が高いが、仲介会社はやる気を出しにくい
  → 「どうせ他社で決まるかも」と思うと広告費をかけにくい

■ 専任媒介
  1社のみに依頼。ただし自己発見取引（売主自身が買主を見つけること）はOK
  レインズ登録義務: 7営業日以内
  報告義務: 2週間に1回以上
  有効期間: 3ヶ月以内（自動更新不可）
  → 最も一般的。仲介会社が集中的に販売活動できる

■ 専属専任媒介
  1社のみに依頼。自己発見取引も不可
  レインズ登録義務: 5営業日以内
  報告義務: 1週間に1回以上
  有効期間: 3ヶ月以内
  → 仲介会社に最も有利。だが売主の自由度が低いので嫌がられることも
```

### 3.3 一括査定サイトの圧力

```
HOME4U、イエウール、すまいValue、リビンマッチ等から査定依頼が流れてくる。

仕組み:
  売主がサイトに物件情報を入力
  → 3-6社の仲介会社に査定依頼が一斉配信
  → 各社が査定書を作成して売主に提示
  → 売主が「一番良さそうな会社」に媒介を依頼

問題:
  ・回答スピードが最大の差別化要因
    「最初に電話をかけた会社に依頼する」が40%
  ・高値査定の誘惑
    「他社より高い査定を出せば媒介が取れる」→ 高すぎて売れない→長期滞留
  ・コスト
    1件の査定依頼に¥5,000-15,000の課金（サイトによる）
    → 成約しなくても費用はかかる。成約率10-30%

  → 「正確な査定を、早く出す」ことが媒介獲得の鍵
```

## 4. 物件査定の実務

### 4.1 査定の3手法

不動産鑑定では「取引事例比較法」「収益還元法」「原価法」の3手法を使う。不動産鑑定士は3手法を全て適用して鑑定評価書を作成するが、仲介会社の実務査定では物件タイプによって主たる手法を使い分ける。

#### 取引事例比較法

周辺の類似物件の取引事例を集め、補正して価格を求める。居住用不動産（マンション・戸建て）で最も多用される手法。

```
手順:
  ① 事例の収集
    レインズの成約事例を検索（直近2-3年、同一エリア、類似条件）
    → 5-10件は欲しい。3件未満だと信頼性が低い

  ② 事情補正
    売り急ぎ、買い進み等の特殊事情を排除
    例: 相続で急いで売った事例 → 10%低い可能性

  ③ 時点修正
    取引時点と現在の市況変動を補正
    例: 2年前の事例で、この地域は年3%上昇 → +6%

  ④ 地域要因の比較
    駅距離、商業施設、学区、騒音、日照等
    例: 駅徒歩5分→10分で-10%〜-15%

  ⑤ 個別要因の比較
    階数（マンション）、方角、築年数、間取り、管理状態
    例: 南向き vs 北向きで+5%、1階 vs 5階で+8%

  → 最終的に「坪単価 × 専有面積」で算出
```

#### 収益還元法

物件が生み出す収益から価格を逆算する。投資用物件（アパート・マンション1棟、区分投資用）で使う。

```
■ 直接還元法（簡易版）
  不動産価格 = NOI（年間純収益）÷ Cap Rate（還元利回り）

  例:
    年間家賃収入: ¥960万（8万×10戸×12ヶ月）
    空室損失:    ¥48万（空室率5%）
    運営費用:    ¥144万（管理費5%+修繕積立+保険+固定資産税）
    NOI:         ¥768万

    Cap Rate:    6%（このエリアの投資用マンションの標準利回り）

    不動産価格 = ¥768万 ÷ 6% = ¥1億2,800万

■ DCF法（精密版）
  将来N年間のキャッシュフロー + N年後の売却価格を現在価値に割り引く
  → 不動産鑑定評価では必須だが、仲介の実務では直接還元法で十分

■ Cap Rate（還元利回り）の目安
  東京23区 RC造:    3.5-5.0%
  首都圏郊外 木造:   6.0-8.0%
  地方中核都市 RC造: 5.5-7.0%
  地方 木造:        8.0-12.0%

  → Cap Rateが低い = 高い価格がつく = 需要が強いエリア
  → Cap Rateが高い = 安い = リスクが高い or 需要が弱い
```

#### 原価法

建物の再調達原価から減価修正して価格を求める。戸建ての建物部分の評価に使う。

```
手順:
  ① 再調達原価の算出
    「今この建物を新築したらいくらかかるか」
    構造別の㎡単価:
      木造:   ¥15-22万/㎡
      鉄骨造: ¥20-30万/㎡
      RC造:   ¥25-40万/㎡

  ② 減価修正
    築年数による減価（経済的耐用年数で按分）
    木造住宅の経済的耐用年数: 22-25年（税法上は22年だが実際は30年以上使う）

    例: 木造戸建て、延床120㎡、築15年
    再調達原価: 120㎡ × ¥18万 = ¥2,160万
    減価修正:  ¥2,160万 × (1 - 15/25) = ¥864万
    → 建物評価額: ¥864万

  ③ 土地価格を加算
    路線価 × 地積（路線価は公示地価の約80%）
    or 公示地価 × 地積
    → 土地評価額: 別途算出

  → 土地+建物 = 査定価格

■ 原価法の限界
  築20年超の木造は「建物価値ゼロ」と評価されがち
  → 実際にはリフォーム済みなら価値がある
  → 原価法だけでは実態と乖離する
  → 取引事例比較法と併用するのが実務
```

### 4.2 3手法の使い分け

| 物件タイプ | 主たる手法 | 補助的な手法 | 理由 |
|---|---|---|---|
| 区分マンション（居住用） | 取引事例比較法 | — | 事例が豊富。同一マンション内の成約実績が最強の根拠 |
| 戸建て | 取引事例比較法 + 原価法 | — | 土地は事例比較、建物は原価法で分離評価 |
| 投資用マンション（1棟） | 収益還元法 | 取引事例比較法 | 投資家は利回りで判断する |
| 投資用区分（ワンルーム） | 収益還元法 | 取引事例比較法 | 同上。ただし事例も参照 |
| 土地（更地） | 取引事例比較法 | 公示地価・路線価参照 | 事例が最も信頼できる |
| 土地（建付地） | 取引事例比較法 | 原価法（解体費控除） | 古家付きは解体費を差し引く |

### 4.3 査定でベテランが見ているポイント

数字に出ない要素が査定精度を左右する。ベテランは現地を見て5分で「この物件は強い/弱い」を判断する。

```
■ プラス要因
  ・日当たり良好（南向き、前面に建物なし）
  ・角部屋/角地
  ・管理状態が良い（マンションのエントランス、共用部の清掃状態）
  ・リフォーム済み（水回り、壁紙。築15年でもリフォーム済みは強い）
  ・学区の評判（「○○小学校区は人気」）
  ・再開発予定（駅前再開発、大型商業施設建設）

■ マイナス要因
  ・幹線道路沿い（騒音）
  ・墓地・工場・嫌悪施設が近い
  ・旧耐震（1981年6月以前の建築確認）→ ローン審査が通りにくい
  ・管理費・修繕積立金が高い（月3万超は敬遠される）
  ・管理組合の財政状況が悪い（修繕積立金の不足、滞納率が高い）
  ・心理的瑕疵（事故物件。告知義務あり）
  ・接道義務を満たさない（再建築不可）→ 評価大幅減
```

## 5. 修繕管理の実務

### 5.1 修繕の3レイヤー

#### 緊急修繕（当日-翌日対応）

```
対象:
  ・水漏れ（上階からの漏水、給湯器故障、排水管詰まり）
  ・ガス漏れ（ガス会社に即連絡）
  ・鍵のトラブル（鍵紛失、オートロック故障）
  ・エアコン故障（真夏のエアコン故障は命に関わる）
  ・停電（ブレーカー落ち、電気設備故障）
  ・トイレ故障（使えないと生活できない）

対応フロー:
  入居者から連絡 → 状況確認（写真送ってもらう）→ 緊急度判定
  → 緊急 → 提携業者に即電話 → 当日訪問手配 → 入居者に連絡
  → 夜間・休日 → 24時間対応の業者 or 社長の携帯

費用目安:
  水漏れ修理:     ¥1-5万（パッキン交換〜配管補修）
  給湯器交換:     ¥15-30万（機種による。エコジョーズは高い）
  鍵交換:        ¥1-3万（ディンプルキーは¥2万〜）
  エアコン修理:   ¥1-5万。交換なら¥8-15万
  トイレ修理:     ¥0.5-3万（部品交換。便器交換は¥10-20万）
```

#### 計画修繕（1-4週間）

```
対象:
  ・設備交換（エアコン、給湯器、コンロ、換気扇 → 耐用年数到達）
  ・内装修繕（クロス張替、フローリング補修、畳表替え）
  ・外部修繕（共用廊下の照明、郵便ポスト、フェンス）
  ・共用設備（エレベーター保守、受水槽清掃、消防設備点検）

対応フロー:
  見積取得（2-3社で相見積が理想。実際は付き合いのある1社のみが多い）
  → オーナー承認（¥5万以上は事前承認が慣例。管理委託契約で金額基準を決める）
  → 工事実施 → 完了確認 → 費用精算（オーナーの送金から差し引き or 請求書発行）
```

#### 大規模修繕（数ヶ月〜1年）

```
対象:
  ・外壁塗装（築10-15年で1回目。¥300-800万/棟）
  ・屋上防水（築10-15年。¥100-300万/棟）
  ・共用配管更新（築25-30年。¥500-2,000万/棟）
  ・エレベーター更新（築25-30年。¥1,000-2,000万/基）

流れ:
  長期修繕計画の策定（築年数×設備×コスト×時期を一覧化）
  → 劣化診断（専門業者に依頼。¥20-50万）
  → 工事仕様書の作成
  → 施工業者の選定（相見積3社以上）
  → オーナー承認（修繕積立金で賄えるか、追加出資が必要か）
  → 工事実施（3-6ヶ月）

問題:
  ・長期修繕計画がそもそもない（中小管理会社の多くが未策定）
  ・修繕積立金が足りない（初期設定が低すぎて、必要な時に金がない）
  ・オーナーが修繕を渋る（「まだ大丈夫だろう」→放置→劣化加速→費用増大）
```

### 5.2 原状回復の実務

#### 国交省ガイドラインの基本原則

```
■ 大原則
  通常損耗・経年変化 → オーナー（貸主）負担
  故意・過失・善管注意義務違反による損耗 → 入居者（借主）負担

■ 通常損耗の具体例（オーナー負担）
  ・日焼けによるクロスの変色
  ・家具の設置跡（冷蔵庫の裏の壁の黒ずみ）
  ・画鋲の穴（ポスター等を貼った跡）
  ・フローリングのワックス劣化
  ・網戸の張替え（経年劣化の場合）
  ・鍵の交換（入居者の過失がない場合）
  ・設備の経年劣化（エアコン、給湯器等）

■ 入居者負担の具体例
  ・タバコのヤニ汚れ（通常の使用を超える）
  ・ペットによる傷・汚れ・臭い
  ・釘穴・ネジ穴（画鋲より大きい穴）
  ・引越し時の引っかき傷
  ・結露を放置してカビが発生
  ・飲み物をこぼして放置したシミ
  ・掃除を怠ったことによる水回りのカビ・水垢

■ 経過年数による減価
  クロス:          6年で残存価値1円（入居6年超→オーナー全額負担）
  カーペット:      6年で残存価値1円
  フローリング:    建物の耐用年数で按分（木造22年、RC47年）
  クッションフロア: 6年で残存価値1円
  畳表:           消耗品扱い
  設備:           耐用年数による（エアコン6年、給湯器6-10年）

■ 退去立会いの実務
  1. 入居者立会いのもと、部屋の状態を確認
  2. 写真撮影（入居時の写真と比較）
  3. 損耗箇所のリスト作成
  4. 入居者負担/オーナー負担の仕分け
  5. 修繕見積の取得（退去後1-2週間）
  6. 敷金精算書の作成・送付（退去後1ヶ月以内が目安）

■ もめるパターン
  ・入居時の写真がない → 入居前からあった傷かどうか証明できない
  ・喫煙を「通常の使用」と主張する入居者
  ・「6年住んだからクロス全額オーナー負担だろ」と言う入居者
    → 正しい。ただし、タバコのヤニは別
  ・ペット禁止物件でペットを飼っていた → 契約違反。全額入居者負担
  ・特約で「退去時のハウスクリーニング費用は入居者負担」→ 有効（判例あり）
```

## 6. 重要事項説明（35条書面）の実務

### 6.1 重説とは何か

宅建業法35条に基づき、契約の相手方に対して宅地建物取引士が行う説明義務。買主（借主）が「この物件のリスクを知らなかった」とならないための消費者保護制度。

重説を行えるのは宅建士のみ。宅建士証を提示して説明し、書面に記名する。2022年5月からIT重説（オンライン）が解禁されたが、書面作成の工数は変わらない。

### 6.2 売買の重要事項説明（主要項目）

```
■ 物件に関する事項
  1. 登記簿に記録された事項
     ・所有権（誰の名義か。共有の場合は持分）
     ・抵当権（銀行のローンが残っているか。残高はいくらか）
     ・地上権、賃借権（他人の権利が付いていないか）
     ・差押え、仮差押え（競売リスク）

  2. 法令上の制限
     ・都市計画区域（市街化区域/市街化調整区域）
     ・用途地域（第一種低層住居専用、商業地域等13種類）
     ・建蔽率・容積率
     ・防火地域・準防火地域
     ・高度制限、日影規制
     ・開発許可が必要かどうか
     ・建築基準法上の道路（接道義務。幅員4m以上の道路に2m以上接していること）
     → 接道義務を満たさない場合「再建築不可」→ 評価額大幅減

  3. インフラ
     ・飲用水（上水道/井戸/受水槽）
     ・排水（公共下水/浄化槽/汲取り）
     ・電気（引込済み/要引込）
     ・ガス（都市ガス/プロパン）

  4. 災害リスク
     ・造成宅地防災区域
     ・土砂災害警戒区域（イエロー/レッド）
     ・津波災害警戒区域
     ・洪水浸水想定区域（ハザードマップ）
     → 2020年改正で水害ハザードマップの説明が義務化

  5. マンション固有の事項
     ・管理規約の内容（ペット可否、楽器演奏、民泊禁止等）
     ・修繕積立金の額と滞納状況
     ・大規模修繕の予定
     ・管理組合の財政状況
     ・管理会社の名称

■ 取引条件に関する事項
  6. 代金以外に必要な金銭
     ・手付金、仲介手数料、登記費用、固定資産税精算金、管理費精算金

  7. 契約の解除に関する事項
     ・手付解除（手付金を放棄して解除）
     ・ローン特約（融資が降りなかったら白紙撤回）
     ・契約違反解除（違約金の額。売買価格の10-20%が相場）

  8. その他
     ・瑕疵担保責任（契約不適合責任）の内容
     ・住宅性能評価の有無
     ・アスベスト調査の有無
     ・耐震診断の有無
```

### 6.3 重説作成で何を調べるか

```
調査先と取得する情報:

  法務局:      登記簿謄本（権利関係の確認）
  市区町村:    都市計画証明書（用途地域、建蔽率/容積率）
               道路台帳（接道状況）
               埋設管図面（上下水道の配管位置）
  ハザードマップ: 洪水、土砂災害、津波、高潮のリスク
  管理組合:    管理規約、修繕積立金、滞納情報（マンションの場合）
  ライフライン: 電力会社、ガス会社、水道局に確認

  → 売買1件で4-6ヶ所に問い合わせる。役所は平日のみ。
  → 1件あたり1-3時間。「調べ忘れ」は重説義務違反=損害賠償リスク
```

## 7. 契約書（37条書面）の実務

### 7.1 37条書面とは

宅建業法37条に基づく契約書面。重説が「契約前の説明」なのに対し、37条書面は「契約内容を確定する書面」。売買契約書・賃貸借契約書がこれにあたる。

### 7.2 売買契約書の実務的ポイント

```
■ 手付金
  売買価格の5-10%が相場（2,000万円の物件なら100-200万円）
  手付解除期限: 「相手方が履行に着手するまで」（民法557条）
  → 実務では「契約後○日以内」と期限を明記

■ ローン特約
  「金融機関の融資承認が得られなかった場合、白紙撤回」
  期限: 契約から3-4週間
  → これがないと、ローンが降りなくても違約金が発生する
  → 買主保護のため必ず入れるが、売主にとっては不安要素

■ 契約不適合責任（旧: 瑕疵担保責任）
  2020年民法改正で「隠れた瑕疵」→「契約不適合」に変更
  売主が宅建業者の場合: 引渡しから2年以上の担保責任が義務
  売主が個人の場合: 「引渡しから3ヶ月」「現状有姿」等の特約が可能
  → 特約の内容が契約書の最重要ポイント

■ 固定資産税・管理費の精算
  引渡し日を基準に日割り精算
  起算日が1月1日（関東）か4月1日（関西）かで変わる

■ 特約事項
  契約書のテンプレには書かれていない個別条件をここに書く
  例: 「残置物の撤去は売主負担」「境界確定は売主負担で行う」
  「心理的瑕疵の告知事項: ○年○月に本物件内で○○が発生」
```

### 7.3 賃貸借契約書のポイント

```
■ 契約の種類
  普通借家契約: 正当事由がない限り更新される（借主保護が強い）
  定期借家契約: 期間満了で確定的に終了（転勤社宅、建替予定等）

■ 主な記載事項
  ・賃料、共益費、駐車場代
  ・敷金の額と返還条件
  ・禁止事項（ペット、楽器、又貸し、事務所使用等）
  ・修繕に関する取り決め（小修繕は借主負担の特約が多い）
  ・解約予告期間（通常1-2ヶ月前）
  ・原状回復の範囲（ハウスクリーニング費用の負担者等）

■ 特約の有効性（判例で確立されている範囲）
  有効:
    ・退去時のハウスクリーニング費用は借主負担
    ・退去時のエアコン内部洗浄は借主負担
    ・鍵交換費用は借主負担
  無効になりうる:
    ・通常損耗の修繕費を借主に全額負担させる特約
    ・敷金を一切返還しない特約（敷引特約。一部は有効だが高額は無効）
```

## 8. 法令環境

### 8.1 宅地建物取引業法（宅建業法）

```
■ 概要
  不動産の売買・交換・賃貸の仲介を業として行う者を規制する法律
  免許制: 都道府県知事免許（1つの都道府県に事務所）or 国交大臣免許（2以上の都道府県）
  5年ごとの更新

■ 主要な義務
  ・事務所ごとに専任の宅建士を置く（5人に1人以上）
  ・営業保証金の供託（本店¥1,000万、支店¥500万）or 保証協会への加入
  ・重要事項説明（35条書面）
  ・契約書面の交付（37条書面）
  ・帳簿・従業者名簿の備付け
  ・手付金等の保全措置（未完成物件の場合）
  ・クーリング・オフ制度の告知

■ 禁止行為
  ・誇大広告（「日当たり良好」の物件が実は北向き）
  ・未完成物件の広告制限（建築確認前の広告は不可）
  ・両手取引の囲い込み（レインズに載せておいて「商談中」と嘘をつく）
  ・断定的判断の提供（「必ず値上がりします」）

■ 違反の罰則
  免許取消 → 5年間再取得不可
  業務停止 → 最大1年間
  指示処分 → 改善命令
  → 免許取消は実質的に廃業
```

### 8.2 借地借家法

```
■ 概要
  建物の賃貸借に関する特別法。民法の賃貸借よりも借主保護が強い

■ 普通借家契約の特徴
  ・期間の定めがあっても、貸主からの更新拒絶には「正当事由」が必要
  ・正当事由 = 貸主自身が使用する必要性、建物の老朽化、立退料の提供等
  ・借主からの解約は自由（1-2ヶ月前予告）
  ・借主は弱者保護されている

■ 定期借家契約の特徴（2000年施行）
  ・期間満了で確定終了（更新なし）
  ・ただし:
    公正証書等の書面で契約すること
    事前に「期間満了で終了する」旨を書面で説明すること
    期間満了の1年前〜6ヶ月前に終了の通知をすること
  → 要件を満たさないと普通借家とみなされる

■ 家賃の増減請求
  借地借家法32条: 当事者は賃料の増額・減額を請求できる
  → 話し合いで解決しない場合は調停→裁判
  → 「家賃相場が上がったから上げたい」はオーナーの正当な権利
  → 「家賃相場が下がったから下げたい」は入居者の正当な権利
```

### 8.3 賃貸住宅管理業法（2021年施行）

```
■ 背景
  サブリースのトラブル（かぼちゃの馬車事件等）が社会問題化
  賃貸住宅管理業の規制がなかった → 2021年に法整備

■ 登録義務
  管理戸数200戸以上の事業者は国交省への登録が義務
  登録の有効期間: 5年。更新あり

■ 主要な義務
  ① 業務管理者の設置
    事務所ごとに「賃貸住宅管理業務管理者」を配置
    資格: 管理業務主任者 or 賃貸不動産経営管理士 + 指定講習

  ② 管理受託契約の重要事項説明
    オーナーに対して、管理業務の内容・報酬を書面で説明
    → 宅建業法の重説と同じ構造

  ③ 財産の分別管理
    家賃等の金銭を自社の固有財産と分別して管理

  ④ 定期報告
    管理業務の実施状況をオーナーに報告（少なくとも年1回。実務では月次）

  ⑤ 国交省への業務報告
    年1回。管理戸数、従業者数、苦情件数等

■ 実務への影響
  登録していない業者は「200戸以上の管理」ができない
  分別管理の証拠書類作成が新たな事務負担
  重説・定期報告の書面作成義務がコンプライアンスコストを押し上げ
  → 中小管理会社は「法令遵守しながら日常業務を回す」だけで精一杯
```

### 9.2 不動産テックの動向

#### 不動産ID（2025年〜段階導入）

```
国交省が推進する「全不動産に一意のIDを付与する」制度。
登記情報（不動産番号）をベースに、住所の表記ゆれを解消し、
物件情報の名寄せ・履歴管理を劇的に改善する。

影響:
  ・物件データの統合が容易になる（A社の「港区赤坂1-1-1」とB社の「港区赤坂一丁目1番1号」が同一物件と確定）
  ・修繕履歴、事故情報、賃貸履歴が物件IDに紐づく
  ・不動産取引の透明性が向上
  → 対応システムの導入コストが発生するが、長期的にはデータ活用の基盤
```

#### 電子契約（2022年5月解禁）

```
宅建業法の書面交付義務がデジタル化
  ・35条書面（重説）の電子交付が可能に
  ・37条書面（契約書）の電子交付が可能に
  ・IT重説（オンラインでの重要事項説明）が本格解禁

現状:
  電子契約の導入率: 10-15%（2024年時点）
  → 大手は進んでいるが中小はまだ紙
  → 「電子化したい」が、オーナーや入居者が紙を求めるケースがまだ多い
  → 電子署名サービス（クラウドサイン、docusign、GMOサイン）の月額費用も障壁

主なサービス:
  いえらぶサイン、電子契約くん、IMAoS（イマオス）等の不動産特化サービスも登場
```

#### AI査定

```
SRE AI査定（SREホールディングス）:
  ソニーグループの不動産AI。マンション・戸建ての自動査定
  公開されている成約データ+独自データで価格を推定
  不動産会社向け+一般消費者向けの両方を展開

HowMa:
  AIによるマンション自動査定。所有者が物件を登録すると定期的に査定更新
  オーナー向けサービス

LIFULL HOME'S PRICE MAP:
  地図上で不動産の参考価格を表示
  機械学習ベース

→ AI査定の精度: ±10-20%（データが豊富なエリア・物件タイプなら±5-10%）
→ 「AIの数字だけで媒介は取れない」が現場の声。最終的に営業力
→ ただし「根拠のある数字を早く出せる」ことで差がつく
```

#### その他の不動産テック

```
VR内見:    NURVE、Spacely。コロナ禍で普及が進んだ。遠方の入居者に有効
IoT:       スマートロック（NinjaLock、Qrio）、スマートメーター
RPA:       ポータルサイトへの物件登録の自動化。いえらぶCLOUDが連携
チャットボット: 入居者からの問い合わせ一次対応
BaaS:      銀行API連携。入金データの自動取得（住信SBI、GMOあおぞら）
```

### 9.3 なぜDXが進まないか

```
1. 業務の多様性
   同じ「賃貸管理」でも、会社ごとにフローが微妙に違う
   「うちの場合は○○が違う」が必ず出る
   → 標準化が難しい。カスタマイズ要望が膨らむ

2. データの分散
   入金データは銀行、物件データは管理ソフト、修繕データはExcel、
   オーナー情報はノート、督促記録は電話メモ
   → 一元化の工数が膨大

3. 人材不足
   ITに詳しい人が社内にいない
   システム導入の旗振り役がいない
   → ベンダーに丸投げ → 使いこなせない → 「システムは高いだけ」

4. ROIの不透明さ
   「月5万のシステムで何時間削減できるか」が事前に見えない
   → 「やってみないと分からない」では社長は決裁しない

5. 切替コスト
   既存のExcelフォーマット、管理ソフト、業務フローの変更
   → 移行期間（2-3ヶ月）は二重管理になる
   → 「面倒」が最大の敵
```

## 11. 地域による違い

不動産は「同じ日本」でもエリアによって全く事情が違う。

```
■ 首都圏（東京23区・横浜・川崎・さいたま）
  空室率:       8-12%
  平均賃料:     ¥8-15万（1K〜1LDK）
  売買相場:     マンション坪単価¥200-600万（エリアによる幅が巨大）
  特徴:
    ・供給過剰だが需要も強い
    ・築古マンションの建替え・リノベが活発
    ・一括査定サイトからの査定依頼が非常に多い（月10-30件の仲介会社も）
    ・外国人入居者の増加（対応スキルが必要）

■ 関西（大阪・京都・神戸）
  空室率:       10-15%
  平均賃料:     ¥5-10万
  売買相場:     東京の6-7割
  特徴:
    ・敷引（保証金の一部を返還しない慣習）が残るエリアあり
    ・更新料の慣習が弱い（関東ほど取らない）
    ・コスト意識が強い。「安くて良いもの」を求める
    ・民泊（Airbnb）の影響が大きい（京都・大阪）

■ 地方中核都市（札幌・仙台・広島・福岡）
  空室率:       12-20%
  平均賃料:     ¥4-7万
  特徴:
    ・新築アパートの供給過剰（サブリース業者の建築提案が原因）
    ・オーナーの高齢化。相続→売却の相談が増加
    ・管理会社の競争が激しい。管理手数料のダンピング

■ 地方・過疎地域
  空室率:       20-40%
  平均賃料:     ¥3-5万
  特徴:
    ・賃料が低すぎてシステム投資が回収できない
    ・管理会社自体が存続危機（管理戸数の減少）
    ・空き家問題（取り壊し費用が物件価値を上回る）
    ・DX以前に「事業継続できるか」が問題
```

## 12. 不動産会社との会話で使える知識

### 12.1 こう言えば「分かってる」と思われる

```
■ 賃貸管理の会話
  ×「家賃の入金管理って大変ですよね」
  ○「月初の消込、名義不一致が多い銀行振込分が一番時間食いますよね。
     口座振替に移行できない入居者がどのくらいいますか?」

  ×「滞納は困りますね」
  ○「保証会社加入率はどのくらいですか? 旧契約で保証会社なしの入居者が
     残ってると、代位弁済できなくて回収に時間かかりますよね」

  ×「オーナーへの報告は毎月やってるんですか?」
  ○「月次の収支報告書、物件ごとにフォーマットが違ったりしますか?
     空室が出てるオーナーへの賃料改定提案って、相場データはどこから取ってます?」

■ 売買仲介の会話
  ×「査定って難しいですか?」
  ○「一括査定サイトからの依頼って月何件くらい来ます?
     回答スピードと査定価格、どっちが媒介獲得に効いてます?」

  ×「契約書を作るのが大変ですよね」
  ○「重説の法令制限の確認、役所回りで1件何時間かかってますか?
     IT重説って使ってますか?」

■ 修繕の会話
  ×「修繕対応も管理会社の仕事ですか?」
  ○「夏場のエアコン故障、当日対応できる業者って何社確保してますか?
     オーナーの修繕承認、金額基準はいくらで切ってます?
     長期修繕計画って策定してますか?」
```

### 12.2 聞いてはいけないこと・注意点

```
・「レインズのデータ見せてもらえますか?」
  → レインズは宅建業者専用。部外者に見せることは規約で禁止
  → 「レインズの成約事例をCSVで取り込めると精度が上がります」とだけ言う

・「両手取引って囲い込みじゃないですか?」
  → 触れてはいけないタブー。両手は合法だが、囲い込みは違法（灰色含む）
  → 正面から聞くと警戒される

・「AIで宅建士いらなくなりますよね?」
  → 法律で宅建士の対面説明が義務。AI化はあくまで「補助」
  → 「宅建士の先生の時間を、もっと大事な業務に使えるようにしたい」が正しい言い方

・オーナーとの関係について根掘り葉掘り聞かない
  → 管理会社にとってオーナーは「最大の機密」
  → 「管理委託を解約されたら終わり」なので、情報を安易に外部に出したくない
```
