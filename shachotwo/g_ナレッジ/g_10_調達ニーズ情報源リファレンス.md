# g_10 調達ニーズ情報源リファレンス

> **目的**: 「どのOEM/Tier1が新規サプライヤーを探しているか」を把握するための公開情報源をまとめ、BPO営業に活用する。
> **更新頻度**: 展示会日程は年1回更新。ポータル情報は半年ごとに確認。

## 1. 調達ポータル / マッチングサイト

### NCネットワーク（エヌシーネットワーク）

- **URL**: https://www.nc-net.or.jp/
- **概要**: 日本最大級の製造業マッチングサイト。発注者が案件を掲載し、加工会社が応募する形式。
- **登録企業数**: 約18,000社
- **費用**: 受注側の基本登録無料。プレミアム会員は月額制（案件の早期閲覧・優先表示）
- **特徴**: 案件ごとに加工種別・材質・数量・納期が明示される。図面付き案件も多い。
- **BPO営業での活用法**: クライアントの保有設備に合致する案件の出現頻度を定期モニタリングし、「御社の設備なら月○件の引合いが来ている」と定量提案に使う。案件傾向から市場の温度感も把握できる。

### Aperza（アペルザ）

- **URL**: https://www.aperza.com/
- **概要**: 製造業向けのカタログ・技術情報プラットフォーム。BtoB購買のリード獲得に使われる。
- **登録企業数**: 約5,000社（出展側）
- **費用**: 出展側は有料（月額制）。閲覧・資料請求は無料。
- **特徴**: 技術カタログのダウンロード数でリードの温度がわかる。購買担当者のアクセスデータを提供。
- **BPO営業での活用法**: クライアントの技術カタログを掲載し、どの業界の購買担当者がダウンロードしているかを分析。「半導体装置メーカーからのカタログDLが増えている→攻めどき」という示唆を提供。

### イプロスものづくり

- **URL**: https://www.ipros.jp/
- **概要**: 製造業のBtoBデータベースサイト。技術・製品・サービスの検索に使われる。
- **登録企業数**: 約60,000社（ものづくり分野）
- **費用**: 基本掲載無料。リード獲得オプションは有料。
- **特徴**: 国内最大級の閲覧数。カテゴリ別のアクセスランキングで「今、何が求められているか」がわかる。
- **BPO営業での活用法**: クライアントの加工技術ページへのアクセス分析。「5軸加工」「チタン加工」等のキーワードトレンドから需要動向を把握し、営業提案に反映。

### ミスミ meviy

- **URL**: https://meviy.misumi-ec.com/
- **概要**: 3D CADデータをアップロードすると即時見積・発注できる自動加工プラットフォーム。
- **特徴**: 短納期・小ロットに強い。価格の透明性が高い。
- **BPO営業での活用法**: meviyの自動見積価格をベンチマークとして利用。「meviy価格より○%安く、かつ品質保証付き」というポジショニングで差別化提案。meviyでは対応できない複雑形状・特殊材料・大ロットが攻め所。

### CADDi（キャディ）

- **URL**: https://caddi.com/
- **概要**: 図面データから最適な加工会社をAIマッチングする受発注プラットフォーム。
- **登録サプライヤー**: 約600社（厳選）
- **費用**: サプライヤー側は無料。発注側もプラットフォーム手数料込み。
- **特徴**: AIによる自動見積。品質管理体制の審査あり。大手企業の利用が増加。
- **BPO営業での活用法**: CADDiのサプライヤー登録支援をBPOサービスに含める。登録審査を通過できる品質体制の構築支援が付加価値になる。CADDi経由の受注データから「どの加工が高単価で出ているか」も分析可能。

## 2. 展示会・商談会

### JIMTOF（日本国際工作機械見本市）

- **規模**: 出展社数 約800社 / 来場者数 約14万人
- **開催**: 2年に1回（偶数年）、東京ビッグサイト
- **次回**: 2026年11月（予定）
- **特徴**: 工作機械の世界4大展示会の1つ。最新設備のトレンドを一望できる。
- **BPO営業での活用法**: クライアントの来場支援（見るべきブースリスト作成）。自社ブースを持つOEM/Tier1の調達担当者との商談セッティング。「新型5軸MCの導入で参入可能になる市場」を具体的に提案。

### インターモールド（金型・金属プレス加工技術展）

- **規模**: 出展社数 約400社 / 来場者数 約4万人
- **開催**: 年1回（4月 大阪 / 6月 東京 交互開催）
- **特徴**: 金型に特化。プレス・鍛造・ダイカストの最新技術が集まる。
- **BPO営業での活用法**: 金型メーカーのクライアント向け。発注元企業（自動車・家電）の調達ニーズを現場で収集。「来場企業リスト×クライアント設備マッチング」で具体的な営業リストを作成。

### MF-TOKYO（プレス・板金・フォーミング展）

- **規模**: 出展社数 約300社 / 来場者数 約6万人
- **開催**: 2年に1回（奇数年）、東京ビッグサイト
- **次回**: 2027年7月（予定）
- **特徴**: 板金加工に特化した国内最大の展示会。レーザー・ベンダー・パンチの最新機が揃う。
- **BPO営業での活用法**: 板金加工クライアントの新規顧客開拓支援。来場するOEMの調達担当者と直接コンタクトする場として活用。

### FOOMA JAPAN（国際食品工業展）

- **規模**: 出展社数 約800社 / 来場者数 約10万人
- **開催**: 年1回（6月）、東京ビッグサイト
- **特徴**: 食品製造・加工機械の総合展示会。食品製造業のDXニーズも展示。
- **BPO営業での活用法**: 食品製造業のクライアント向け。設備メーカーの新製品情報を収集し、生産性向上提案に組み込む。食品工場の自動化トレンドをキャッチ。

### MEDTEC Japan（医療機器の設計・製造に関する展示会）

- **規模**: 出展社数 約500社 / 来場者数 約2万人
- **開催**: 年1回（4月）、東京ビッグサイト
- **特徴**: 医療機器の部品・材料・製造受託（CMO/CDMO）に特化。
- **BPO営業での活用法**: 医療機器分野への参入を検討するクライアント向け。ISO13485取得済みの加工会社は高い需要がある。「医療機器メーカーが求めるサプライヤー要件」を収集し、ギャップ分析に使う。

### 国際航空宇宙展（Japan Aerospace）

- **規模**: 出展社数 約800社 / 来場者数 約5万人
- **開催**: 2年に1回、東京ビッグサイト
- **次回**: 2026年10月（予定）
- **特徴**: 防衛・宇宙・航空の総合展示会。サプライチェーン国内回帰の流れで中小加工会社への期待大。
- **BPO営業での活用法**: JIS Q 9100（AS9100）取得済みのクライアント向け。Tier1/Tier2の調達担当者との関係構築。未取得クライアントには認証取得ロードマップを提案。

### 地域開催の商談会（随時）

- **中小機構主催のビジネスマッチング**: 年間100回以上、全国各地で開催。無料。
- **都道府県の産業振興財団主催**: 「○○ものづくりマッチング」等。地元企業同士のマッチング。
- **金融機関主催**: メガバンク・地銀が取引先同士をつなぐビジネスマッチング。
- **BPO営業での活用法**: 参加コストがゼロ〜低いため、クライアントに積極参加を推奨。事前に「クライアントの強み1枚シート」をBPOで作成し、商談の質を上げる。

### 活用のポイント

- **会員企業リスト**: 多くの工業会がWebサイトで会員一覧を公開。営業ターゲットリストの基礎資料になる。
- **受注統計**: 金型工業会・工作機械工業会の受注統計は、3-6ヶ月先の生産動向を予測する先行指標。受注が増えている工程→その工程の外注ニーズが増える。
- **技術委員会**: 業界団体の技術委員会に参加すると、大手企業の調達責任者と直接つながれる。クライアントの入会を支援するのもBPOの付加価値。

### 新規サプライヤー登録の一般的な手順

1. **Webエントリー**: 企業情報・保有設備・認証・実績を登録
2. **書類審査**: 会社案内・品質管理体制・財務情報の提出
3. **工場監査**: 調達担当者+品質担当者が工場を訪問（通常1-2日）
4. **試作発注**: 少量の試作で品質・納期・対応力を評価
5. **量産承認**: 試作合格後、正式サプライヤーとして登録

**BPO営業での活用法**: 上記の登録プロセス全体を支援する「OEM参入支援パッケージ」としてBPOに組み込む。特にStep 2の書類作成（会社案内の英訳、品質マニュアルの体裁整備）と、Step 3の監査対策（5S・見える化の事前整備）で大きな付加価値を出せる。

## 5. 政府系支援機関

### 中小機構（中小企業基盤整備機構）

- **URL**: https://www.smrj.go.jp/
- **ビジネスマッチング**: J-GoodTech（https://jgoodtech.smrj.go.jp/）で企業プロフィールを公開し、国内外からマッチングを受ける。
- **費用**: 無料
- **登録企業数**: 約15,000社
- **BPO営業での活用法**: クライアントのJ-GoodTechプロフィール作成を支援。英語対応のプロフィールを作ると海外からの引合いも来る。「登録したけど更新していない」企業が多いため、プロフィール最適化だけでも効果が出る。

### JETRO（日本貿易振興機構）

- **URL**: https://www.jetro.go.jp/
- **活用サービス**: 海外バイヤーとのマッチング、海外展示会への出展支援、輸出相談
- **費用**: 多くのサービスが無料〜低額
- **BPO営業での活用法**: 海外取引を志向するクライアント向け。JETROのバイヤーマッチングに参加するための技術資料（英語）作成をBPOで支援。特に東南アジア・インドの日系工場からの調達ニーズは増加傾向。

### よろず支援拠点（都道府県ごとに設置）

- **URL**: https://yorozu.smrj.go.jp/
- **概要**: 中小企業の経営相談をワンストップで受ける国の機関。コーディネーターが常駐。
- **費用**: 完全無料（何度でも相談可能）
- **BPO営業での活用法**: コーディネーターは地域の企業ネットワークを持っている。クライアントを紹介してもらうルートとして有効。BPOサービスの紹介を兼ねてコーディネーターとの関係構築を行う。

### ものづくり補助金・事業再構築補助金

- **概要**: 設備投資に対して最大1,000万〜1億円の補助。認証取得費用も対象になる場合がある。
- **BPO営業での活用法**: 補助金申請書の作成支援をBPOで行う。特に「事業計画書」の部分は、売上予測モデル（d_11）の出力をそのまま使える。「補助金で設備導入→新市場参入→売上増」のストーリーを一気通貫で提案。

## 6. LinkedIn / 求人情報からの調達ニーズ推定

### LinkedIn活用

大手企業の調達部門の動きから、サプライヤー探索の温度感を推定する。

| シグナル | 推定される調達ニーズ | BPO営業アクション |
|---|---|---|
| 「購買担当者」「Procurement」を増員募集 | 調達量が増加している / 新規サプライヤー開拓を強化 | クライアントの技術PRを当該企業に送付 |
| 「品質管理」「SQE」を募集 | サプライヤー品質に課題あり / 新規認定を増やしたい | 品質体制が整っているクライアントを優先的にマッチング |
| 「SCM」「サプライチェーン」人材を募集 | サプライチェーンの再構築中 / 新規ソース探索 | 「国内回帰」「BCP対応」の文脈でアプローチ |
| 工場新設・ライン増設のニュース | 部品の調達量が大幅増加 | 新工場の立地近隣のクライアントをマッチング |

### 求人サイト活用

- **Indeed / リクナビNEXT / doda**: 製造業の求人で「購買」「調達」「品質管理」のキーワードでアラート設定
- **求人数の推移**: 特定企業の購買職求人が増えている→調達活動が活発化している証拠
- **職務内容の記述**: 「新規サプライヤー開拓」「コスト削減」「国内回帰」等のキーワードから具体的なニーズを推定

### 活用の自動化（将来構想）

`workers/micro/signal_detector.py` で以下を自動化する（Phase 2以降）:

1. LinkedIn Sales Navigatorで調達担当者の異動・増員をウォッチ
2. 求人サイトのAPIで購買職の求人数推移をトラッキング
3. 企業のプレスリリース（工場新設・設備投資）を自動収集
4. 上記シグナルをスコアリングし、「今アプローチすべき企業」をクライアントに通知
