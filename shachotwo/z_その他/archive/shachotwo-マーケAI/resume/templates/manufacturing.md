# 製造業向け AI社員テンプレート

## 得意業務（トップ3）
1. **見積AI** — 過去の見積データから自動算出。見積作成時間を80%削減
2. **生産計画** — 受注・在庫・リードタイムから最適計画を自動作成
3. **在庫最適化** — 在庫回転率20%改善。欠品・過剰在庫を防止

## 対応可能業務
4. FAX受発注の電子化 — FAX→OCR→自動入力で手入力ゼロ
5. 品質管理 — 検査記録の自動集計・異常値アラート
6. 出荷管理 — 出荷指示書・納品書の自動生成
7. 原価計算 — 材料費・加工費の自動集計
8. 設備保全 — 定期点検スケジュール・故障予兆検知

## 訴求テンプレート
- 零細（1-10名）: 「社長が受注から出荷まで全部見ている → 社長の右腕に」
- 小規模（11-50名）: 「FAX受発注の手入力に毎日2時間 → ゼロに」
- 中規模（51-300名）: 「ベテランしか見積が出せない → 属人化リスク解消」

## 業界キーワード
多品種少量 / FAX文化 / 熟練工不足 / IoT / スマートファクトリー
